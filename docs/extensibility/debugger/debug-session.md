# 调试会话
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

根据调试器体系结构， **会话**:  
  
-   是设置所有进程，相关或不相关，由或生成的附加到从 IDE 的一个实例。  
  
-   由会话实现调试管理器 \(SDM\)。  有关更多信息，请参见 [会话调试管理器](../../extensibility/debugger/session-debug-manager.md)。
