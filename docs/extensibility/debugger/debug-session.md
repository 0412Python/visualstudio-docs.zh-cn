# <a name="debug-session"></a>调试会话
在调试器体系结构，方面**会话**:  
  
-   为组的所有处理，相关或多个不相关、，已启动或附加到 IDE 的单个实例。  
  
-   由会话调试管理器 (SDM) 实现。 有关详细信息，请参阅[调试会话的管理器](../../extensibility/debugger/session-debug-manager.md)。
