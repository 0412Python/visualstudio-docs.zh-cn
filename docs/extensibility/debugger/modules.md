# 模块
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

根据调试器体系结构， **模块**:  
  
-   是实际容器代码，如可执行文件或 DLL。  
  
-   可以重新加载其符号和自我描述。  模块声明在 IDE 的模块 " 窗口中显示。  
  
-   由 [IDebugModule2](../../extensibility/debugger/reference/idebugmodule2.md) 接口表示，创建的调试引擎描述模块。
