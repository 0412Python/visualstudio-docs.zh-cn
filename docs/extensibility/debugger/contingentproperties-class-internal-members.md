# ContingentProperties 类-内部成员
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

包含的其他属性 <xref:System.Threading.Tasks.Task> 对象。  
  
 **命名空间︰** <xref:System.Threading.Tasks?displayProperty=fullName>  
  
 **程序集︰** mscorlib （在 mscorlib.dll\)  
  
 由于无法在.NET Framework 中访问这些内部成员，下面的语法提供通用中间语言 \(CIL\)。  
  
## 语法  
  
```  
.class auto ansi nested assembly beforefieldinit ContingentProperties  
       extends System.Object  
```  
  
## 成员  
  
### 字段  
  
|名称|描述|  
|--------|--------|  
|[m\_children](../../extensibility/debugger/m-children-field.md)|与此任务注册的子任务的列表。|  
  
## 备注  
 仅在需要时，.NET Framework 初始化此类的字段。
