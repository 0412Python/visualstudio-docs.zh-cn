# 参考 (Visual Studio 调试 Api)
[!INCLUDE[vs2017banner](../../../code-quality/includes/vs2017banner.md)]

debugging SDK 的 Visual Studio 引用轮廓调试器及其元素的运行时体系结构和函数的基本设计。  它描述该通过接口定义的文档进行尽可能少的假设。有关组件和接口，而不是行为除外。  
  
 参考部分的目标是提供支持对同一组不同语言、运行时和操作系统尽可能。
