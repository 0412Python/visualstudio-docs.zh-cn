# <a name="adprocessid"></a>AD_PROCESS_ID
指定的进程 ID，这可能是系统 ID 或 GUID。  
  
## <a name="syntax"></a>语法  
  
```cpp  
typedef struct _AD_PROCESS_ID {  
   AD_PROCESS_ID_TYPE ProcessIdType;  
   union {  
      DWORD dwProcessId;   
      GUID  guidProcessId;   
      DWORD dwUnused;   
   } ProcessId;  
} AD_PROCESS_ID;  
```  
  
```csharp  
public struct AD_PROCESS_ID {  
   AD_PROCESS_ID_TYPE ProcessIdType;  
   DWORD              dwProcessId;   
   GUID               guidProcessId;   
   DWORD              dwUnused;   
};  
```  
  
## <a name="members"></a>成员  
 `ProcessIdType`  
 取值范围为[AD_PROCESS_ID_TYPE](../../../extensibility/debugger/reference/ad-process-id-type.md)枚举指定如何解释`ProcessId`联合 （或对于托管代码，该结构的成员才能访问）。  
  
 dwProcessId  
 从系统值形式的进程 ID。  
  
 guidProcessId  
 作为 GUID 进程 ID。  
  
 dwUnused  
 填充。
