# <a name="stepunit"></a>STEPUNIT
单步执行指定单步执行单元。  
  
## <a name="syntax"></a>语法  
  
```cpp  
enum enum_STEPUNIT {   
   STEP_STATEMENT   = 0,  
   STEP_LINE        = 1,  
   STEP_INSTRUCTION = 2  
};  
typedef DWORD STEPUNIT;  
```  
  
```csharp  
enum enum_STEPUNIT {   
   STEP_STATEMENT   = 0,  
   STEP_LINE        = 1,  
   STEP_INSTRUCTION = 2  
};  
```  
  
## <a name="members"></a>成员  
 STEP_STATEMENT  
 语句的步骤。  
  
 STEP_LINE  
 行的步骤。  
  
 STEP_INSTRUCTION  
 通过指令的步骤。  
  
## <a name="remarks"></a>备注  
 作为自变量传递[步骤](../../../extensibility/debugger/reference/idebugprocess3-step.md)方法。
