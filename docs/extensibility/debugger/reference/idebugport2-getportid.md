# <a name="idebugport2getportid"></a>IDebugPort2::GetPortId
获取端口标识符。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetPortId(   
   GUID* pguidPort  
);  
```  
  
```csharp  
int GetPortId(   
   out Guid pguidPort  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pguidPort`  
 [out]返回标识端口的 GUID。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`; 否则为返回错误代码。
