# IDebugThread2::SetThreadName
[!INCLUDE[vs2017banner](../../../code-quality/includes/vs2017banner.md)]

设置线程的名称。  
  
## 语法  
  
```cpp#  
HRESULT SetThreadName (   
   LPCOLESTR pszName  
);  
```  
  
```c#  
int SetThreadName (   
   string pszName  
);  
```  
  
#### 参数  
 `pszName`  
 \[in\] 线程的名称。  
  
## 返回值  
 如果成功，则返回; `S_OK`否则，返回错误代码。  
  
## 备注  
 获取线程名称，调用 [GetName](../../../extensibility/debugger/reference/idebugthread2-getname.md) 方法。
