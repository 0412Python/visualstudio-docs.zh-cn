# <a name="idebugmemorybytes2getsize"></a>IDebugMemoryBytes2::GetSize
检索的大小，以字节为单位，表示此内存的[IDebugMemoryBytes2](../../../extensibility/debugger/reference/idebugmemorybytes2.md)对象。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetSize(   
   UINT64* pqwSize  
);  
```  
  
```csharp  
int GetSize(  
   out ulong pqwSize  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pqwSize`  
 [out]返回的大小，以字节的内存空间。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`; 否则为返回错误代码。
