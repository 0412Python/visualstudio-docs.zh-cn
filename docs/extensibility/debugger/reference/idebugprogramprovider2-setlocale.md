# <a name="idebugprogramprovider2setlocale"></a>IDebugProgramProvider2::SetLocale
建立的区域设置要用于任何特定于区域设置的资源。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT SetLocale(  
   WORD wLangID  
);  
```  
  
```csharp  
int SetLocale(  
   ushort wLangID  
);  
```  
  
#### <a name="parameters"></a>参数  
 `wLangID`  
 [in]若要建立的语言 ID。 例如，1033 为英语的。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`; 否则为返回错误代码。
