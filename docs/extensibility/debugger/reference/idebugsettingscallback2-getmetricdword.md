# <a name="idebugsettingscallback2getmetricdword"></a>IDebugSettingsCallback2::GetMetricDword
检索给定名称的情况的指标的值。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetMetricDword(  
   LPCWSTR pszType,  
   REFGUID guidSection,  
   LPCWSTR pszMetric,  
   DWORD*  pdwValue  
);  
```  
  
```csharp  
private int GetMetricDword(  
   string   pszType,  
   ref Guid guidSection,  
   string   pszMetric,  
   out uint pdwValue  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pszType`  
 [in]度量值的类型。  
  
 `guidSection`  
 [in]部分中的唯一标识符。  
  
 `pszMetric`  
 [in]度量值名称。  
  
 `pdwValue`  
 [out]返回的度量值的值。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`; 否则为返回错误代码。
