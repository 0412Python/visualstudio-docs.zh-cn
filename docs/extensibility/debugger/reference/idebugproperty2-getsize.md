# <a name="idebugproperty2getsize"></a>IDebugProperty2::GetSize
获取的大小，以字节为单位的属性值。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetSize (   
   DWORD* pdwSize  
);  
```  
  
```csharp  
int GetSize (   
   out uint pdwSize  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pdwSize`  
 [out]返回的大小，以字节为单位的属性值。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`; 否则返回错误代码。 返回`S_GETSIZE_NO_SIZE`如果属性具有没有大小。
