# <a name="stepkind"></a>STEPKIND
指定单步执行的步骤类型。  
  
## <a name="syntax"></a>语法  
  
```cpp  
enum enum_STEPKIND {   
   STEP_INTO      = 0,  
   STEP_OVER      = 1,  
   STEP_OUT       = 2,  
   STEP_BACKWARDS = 3  
};  
typedef DWORD STEPKIND;  
```  
  
```csharp  
public enum enum_STEPKIND {   
   STEP_INTO      = 0,  
   STEP_OVER      = 1,  
   STEP_OUT       = 2,  
   STEP_BACKWARDS = 3  
};  
```  
  
## <a name="members"></a>成员  
 STEP_INTO  
 单步执行函数。  
  
 跨度  
 过程执行函数的步骤。  
  
 STEP_OUT  
 从函数的步骤。  
  
 STEP_BACKWARDS  
 到函数向后的步骤。  
  
## <a name="remarks"></a>备注  
 作为自变量传递[步骤](../../../extensibility/debugger/reference/idebugprocess3-step.md)方法。
