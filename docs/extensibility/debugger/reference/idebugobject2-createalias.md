# <a name="idebugobject2createalias"></a>IDebugObject2::CreateAlias
创建唯一的 ID 或别名此对象或返回现有别名。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT CreateAlias(  
   IDebugAlias** ppAlias  
);  
```  
  
```csharp  
int CreateAlias(  
   out IDebugAlias ppAlias  
);  
```  
  
#### <a name="parameters"></a>参数  
 `ppAlias`  
 [out]新的 （或现有的） 的别名。  
  
## <a name="return-value"></a>返回值  
 如果成功，返回，则为 S_OK;否则，返回错误代码。  
  
## <a name="remarks"></a>备注  
 别名是表示特定对象，虽然该对象是在内存中的标签。
