# <a name="process-debug-manager"></a>进程调试管理器
进程调试管理器 (PDM) 是管理程序的 Visual Studio 的一个组件，使它们可用于该会话的进程调试管理器和调试引擎。  
  
 PDM 管理可调试的所有进程。 若要进行调试，必须向 PDM 注册程序。 在启动程序时，通过一个端口或调试引擎时完成此注册。
