# <a name="process-debug-manager"></a>进程调试管理器
过程调试管理器 (PDM) 是一个组件的 Visual Studio 的管理程序和进程，使其可用于会话调试管理器和调试引擎。  
  
 PDM 管理可调试的所有进程。 要调试，必须向 PDM 注册程序。 这可在启动程序时，通过一个端口或调试引擎的时间。
