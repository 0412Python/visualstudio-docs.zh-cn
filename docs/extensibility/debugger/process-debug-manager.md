# 进程调试管理器
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

进程内调试管理器 \(PDM\)是管理程序并处理 Visual Studio 的元素，将其提供给会话调试管理器和调试引擎。  
  
 PDM 管理可能正在调试的所有进程。  要调试程序，必须向 PDM 注册。  ，在过程由端口或调试引擎时，启动，这样做。
