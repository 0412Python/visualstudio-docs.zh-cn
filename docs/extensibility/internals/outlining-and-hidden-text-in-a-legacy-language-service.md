# <a name="outlining-and-hidden-text-in-a-legacy-language-service"></a>在早期语言服务中的大纲显示和隐藏文本
以大纲方式显示使可以折叠成单个行的文本行的序列。 例如，在 c + + 中的所有方法可以都折叠到单个行中，显示仅方法签名。 隐藏的文本是可以显示或隐藏的文本行的序列。
