# <a name="web-projects"></a>Web 项目
Web 项目系统是创建 Web 项目的项目系统。 Web 项目又会创建 Web 应用程序。 您可以编写使用传统编程语言中，如 Web 页面[!INCLUDE[vbprvb](../../code-quality/includes/vbprvb_md.md)]或[!INCLUDE[csprcs](../../data-tools/includes/csprcs_md.md)]，以收集和处理来自用户的信息，将其存储在数据库中，依次类推。 使用关联的代码的网页称为智能 Web 页。
