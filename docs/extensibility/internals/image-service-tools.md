# <a name="image-service-tools"></a>映像服务工具
VS SDK 包含多种工具，帮助扩展生成器使用 Visual Studio 映像服务。  
  
-   [从资源清单](../../extensibility/internals/manifest-from-resources.md)工具使用的图像资源 （.png 或.xaml 文件） 的列表，并生成图像清单文件。  
  
-   [清单复制到代码](../../extensibility/internals/manifest-to-code.md)工具获取图像清单文件，并生成要在 c + +、 C#、 VB 或.vsct 文件中使用的包装器文件。  
  
-   [图像库查看器](../../extensibility/internals/image-library-viewer.md)可以加载、 编辑和搜索图像的清单。
