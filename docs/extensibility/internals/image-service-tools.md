# <a name="image-service-tools"></a>图像服务工具
VS SDK 包含多种工具，帮助使用 Visual Studio 映像服务的扩展生成器。

-   [清单资源的](../../extensibility/internals/manifest-from-resources.md)工具提取列表的图像资源 （.png 或.xaml 文件），并生成一个映像清单文件。

-   [清单复制到代码](../../extensibility/internals/manifest-to-code.md)工具采用映像清单文件并生成要在 c + +、 C#、 VB 或.vsct 文件中使用的包装器文件。

-   [图像库查看器](../../extensibility/internals/image-library-viewer.md)可以加载、 编辑和搜索图像的清单。
