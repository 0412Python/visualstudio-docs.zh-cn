# 映像服务工具
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

VS SDK 包括多种工具以帮助使用 Visual Studio 映像服务的扩展生成器。  
  
-   [从资源清单](../../extensibility/internals/manifest-from-resources.md) 工具获取图像资源 \(.png 或.xaml 文件\) 的列表，并生成一个映像清单文件。  
  
-   [代码清单](../../extensibility/internals/manifest-to-code.md) 工具采用图像清单文件并生成要在 c \+ \+、 C\#、 VB 或.vsct 文件中使用的包装器文件。  
  
-   [图像库查看器](../../extensibility/internals/image-library-viewer.md) 可以加载、 编辑和搜索图像清单。
