# <a name="extensibility-in-visual-studio"></a>Visual Studio 的扩展性
Visual Studio 提供了大量的扩展选项。 可以创建您自己的 Sdk、 Windows、 Windows Phone 和 Azure Sdk （它作为 Visual Studio 的一部分进行安装），使用和扩展 Visual Studio 本身。  
  
## <a name="extend-visual-studio"></a>扩展 Visual Studio  
 可以使用 Visual Studio SDK 扩展 Visual Studio 的几乎每个部分︰ 命令、 菜单、 windows、 编辑器和项目。 若要获得详细信息，请参阅[Visual Studio SDK](../extensibility/visual-studio-sdk.md)。  
  
## <a name="create-your-own-sdks"></a>创建您自己的 Sdk  
 了解如何创建、 打包和部署您自己的平台和扩展 Sdk:[创建软件开发工具包](../extensibility/creating-a-software-development-kit.md)。  
  
## <a name="roslyn-extensibility"></a>Roslyn 可扩展性  
 了解如何使用 Roslyn 的可扩展性︰ [.NET 编译器平台 ("Roslyn") 可扩展性](../extensibility/dotnet-compiler-platform-roslyn-extensibility.md)。
