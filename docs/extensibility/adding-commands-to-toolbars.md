# <a name="add-commands-to-toolbars"></a>将命令添加到工具栏
工具栏是水平或垂直条包含绑定到命令的按钮。 可以将自己的命令添加到 Visual Studio 中的许多不同的 windows 上的工具栏： Visual Studio 主窗口、 解决方案资源管理器和你自己的自定义工具窗口。  
  
 本部分介绍大量可以使用工具栏上的命令执行的操作。
