# 将命令添加到工具栏
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

工具栏是一个包含绑定到命令的按钮的水平或垂直条。 可以将您自己的命令添加到 Visual Studio 中的许多不同的 windows 上的工具栏: Visual Studio 主窗口、 解决方案资源管理器和您自己的自定义工具窗口。  
  
 本部分说明了大量可以使用工具栏上的命令执行的操作。
