# <a name="extending-properties-and-the-property-window"></a>扩展属性和属性窗口
**属性**窗口用于查看和编辑的设计时属性和所选对象的事件。  
  
 当选择中的某个对象[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)]，该对象的公共属性将显示在**属性**窗口。 有关详细信息，请参阅[属性窗口概述](../extensibility/internals/properties-window-overview.md)。
