# 如何：添加或移除 SharePoint 连接
  利用服务器资源管理器，您可以浏览 SharePoint 网站和数据连接。  但您必须先将 SharePoint 网站添加到**“SharePoint 连接”**节点中，然后才能浏览 SharePoint 网站的内容。  
  
### 将 SharePoint 网站添加到“SharePoint 连接”节点  
  
1.  在菜单栏上，依次选择**“视图”**、**“服务器资源管理器”**  
  
2.  在 **服务器资源管理器**中，选择 **SharePoint 连接** 节点，然后，菜单栏上，选择 **工具**，选择 **添加 SharePoint 连接**。  
  
3.  在 **添加 SharePoint 连接** 框中，输入 SharePoint 网站的 [!INCLUDE[TLA2#tla_url](../sharepoint/includes/tla2sharptla-url-md.md)] \(例如，http:\/\/testserver\/sites\/unittests\)。  
  
### 从“SharePoint 连接”节点中删除 SharePoint 网站  
  
1.  在菜单栏上，选择**“视图”**，**“服务器资源管理器”**来打开**服务器资源管理器**。  
  
2.  展开**“SharePoint 连接”**节点以显示要从**“服务器资源管理器”**中删除的 SharePoint 网站。  
  
3.  选择网站，然后在菜单栏上，选择 **编辑**，**删除**。  
  
    > [!NOTE]  
    >  这一步不会删除基础网站，而只是从**“服务器资源管理器”**中删除连接。
