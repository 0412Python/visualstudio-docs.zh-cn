# <a name="how-to-add-a-feature-to-sharepoint-solutions"></a>如何：向 SharePoint 解决方案添加功能
  创建 SharePoint 解决方案时，一个功能和一个包被添加到解决方案中。 你可以添加更多的功能项，来组织不同类型的 SharePoint 文件。  
  
### <a name="to-add-a-feature-through-solution-explorer"></a>若要添加到解决方案资源管理器功能  
  
1.  在**解决方案资源管理器**，打开快捷菜单**功能**，然后选择**添加功能**。
