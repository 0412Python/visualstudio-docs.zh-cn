# 如何：向 SharePoint 解决方案添加功能
  在创建 SharePoint 解决方案时，会向该解决方案添加一个功能和一个包。  您可以添加多个功能项以组织不同类型的 SharePoint 文件。  
  
### 通过解决方案资源管理器添加功能  
  
1.  在“解决方案资源管理器”中，打开“功能”的快捷菜单，然后选择“添加功能”。
