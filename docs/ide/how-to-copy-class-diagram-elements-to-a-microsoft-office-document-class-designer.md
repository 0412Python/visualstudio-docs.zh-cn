# How to: Copy Class Diagram Elements to a Microsoft Office Document (Class Designer)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

您可以将形状从 .NET 类关系图 \(.cd 文件）复制到其他文档中。  您可以基于将要粘贴到的文档的类型获取形状或及其底层代码的拷贝。  若要复制建模项目中的 UML 类关系图中的形状，请参阅[将关系图导出为图像](../modeling/export-diagrams-as-images.md)。  
  
### 若要复制的单个元素  
  
-   右键单击该形状，然后选择**复制图像**。  
  
### 若要复制多个元素  
  
1.  在关系图上选择要复制的形状。  
  
2.  右键单击您的选择，然后选择**复制图像**。  
  
### 若要在类图中复制所有元素  
  
1.  右键单击关系图面并选择**全选**。  （键盘：**Ctrl \+ A**）  
  
2.  在**编辑**菜单上，选择**复制图像**。  
  
 您还可以选择**复制**而不是**复制图像**。  **复制**将复制为普通的位图图像。  **复制图像**将图像复制为大多数 Office 应用程序的显示效果会更好的基于矢量的图像。
