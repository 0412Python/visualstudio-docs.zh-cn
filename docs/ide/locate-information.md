# 找到信息
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

“帮助查看器”提供了多种不同的方式查找帮助信息。  根据所使用的信息要查找，某些方案中的其他可能有效。  
  
|有关以下内容的信息|使用|  
|---------------|--------|  
|用户界面的元素。|F1 键|  
|在目录相同区域中作为当前主题的其他主题|工具栏上**“在内容中显示主题”**的按钮|  
|功能或技术|索引|  
|语言元素如方法和事件|搜索|
