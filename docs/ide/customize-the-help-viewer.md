# 自定义帮助查看器
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

您可以通过使用**“查看器选项”**对话框配置“帮助查看器”的多个功能。  
  
## 设置查看器选项  
 您通过选择工具栏上的**“查看器选项”**按钮打开了**“查看器选项”**对话框。  
  
|若要执行此任务：|执行此步骤：|  
|--------------|------------|  
|更改文本的字体大小|在 **“字号”** 的列表中选择大小。|  
|更改在**“搜索”**选项卡中显示的搜索结果的最大数。|在 **“最大搜索结果”** 列表中选择值。|  
|更改在**“收藏夹”**窗口中显示的历史记录项的最大数。|在 **“保存的最大历史记录项”** 列表中选择值。|  
|在查看产品非英语版本的内容时，目录中包含或排除英文内容。|选中或清除**“在任何导航选项卡和 F1 请求中包含英文内容”**复选框。 **Caution:**  此功能还控制是否可以下载位于**“管理内容”**选项卡的英文内容。|
