# <a name="internet-resources-used-by-visual-studio"></a>Visual Studio 使用的 Internet 资源

Visual Studio 是一个连接的环境：

- [登录](../ide/signing-in-to-visual-studio.md) Visual Studio 后，可以访问与用户帐户以及链接到该帐户的所有帐户关联的所有服务。 无论你在何处使用 Visual Studio，你的帐户设置都随你一起漫游。  
- 当安装的任何组件（例如 SDK）有可用更新时，Visual Studio 都会[通知你](../ide/visual-studio-notifications.md)。  
- 可以下载 [NuGet](/nuget/) 包来获取库。  
- 可以下载 Visual Studio [扩展](../ide/finding-and-using-visual-studio-extensions.md)。  
- 可以连接到 Azure 服务。  
- 开始页会更新来自不同网站的最新开发人员新闻。  
- 等等  

本部分的主题介绍 Visual Studio 对 internet 资源的不同使用方式。
