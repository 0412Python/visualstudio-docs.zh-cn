# <a name="connected-environment"></a>连接的环境
Visual Studio 是一个连接的环境：  

- 登录 Visual Studio 后，可以访问与用户帐户以及链接到该帐户的所有帐户关联的所有服务。 无论你在何处使用 Visual Studio，你的帐户设置都随你一起漫游。  
- 当安装的任何组件（例如 SDK）有可用更新时，Visual Studio 都会通知你。  
- 可以下载 NuGet 包和扩展。
