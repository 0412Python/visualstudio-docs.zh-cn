# 转到行
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

此对话框允许移动到活动文档中的特定行。  若要访问此对话框，请打开要编辑的文档，然后在**“编辑”**菜单上选择**“转到”**。  
  
 行号 \(1 \- `n`\)  
 允许您输入要移动到的活动文档中的行号。  输入的数字必须在 1 和 `n` 之间，后者表示当前文档中的行数。
