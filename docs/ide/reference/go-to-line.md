# <a name="go-to-line-dialog-box"></a>“转到行”对话框

通过此对话框可以移动到活动文档中的特定行。 要访问此对话框，请打开要编辑的文档，然后在“编辑”菜单上选择“转到”。

## <a name="line-number-1---n"></a>行号 (1 - `n`)

允许输入想要移动到的活动文档中的行号。 输入的数量必须在 1 到 `n` 之间（n 为当前文档中的行号）。
