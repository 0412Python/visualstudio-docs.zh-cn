# “设置当前堆栈帧”命令
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

使您可以设置特定的堆栈帧。  
  
## 语法  
  
```  
Debug.SetCurrentStackFrame index  
```  
  
## 参数  
 `index`  
 必选。  按堆栈帧的索引选择堆栈帧。  
  
## 示例  
  
```  
>Debug.SetCurrentStackFrame 1  
```
