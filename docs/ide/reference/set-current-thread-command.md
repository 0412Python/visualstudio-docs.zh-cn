# “设置当前线程”命令
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

将指定的线程设置为当前线程。  
  
## 语法  
  
```  
Debug.SetCurrentThread index  
```  
  
## 参数  
 `index`  
 必选。  按线程的索引选择线程。  
  
## 示例  
  
```  
>Debug.SetCurrentThread 1  
```
