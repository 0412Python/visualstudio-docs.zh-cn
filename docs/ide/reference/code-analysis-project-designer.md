# “项目设计器”-&gt;“代码分析”
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

包含代码分析工具，可以选择它在代码上运行。  该工具报告有关程序集的信息，如在 Microsoft .NET Framework 设计准则中提出的编程冲突和设计规则。  
  
## UIElement 列表  
 启用代码分析  
 为项目启用或禁用代码分析。  
  
 设计规则  
 启用或禁用设计规则。  也可以展开此项以启用或禁用个别规则。  
  
 全球化规则  
 启用或禁用全球化规则。  也可以展开此项以启用或禁用个别规则。  
  
 互操作性规则  
 启用或禁用互操作性规则。  也可以展开此项以启用或禁用个别规则。  
  
 可维护性规则  
 启用或禁用可维护性规则。  也可以展开此项以启用或禁用个别规则。  
  
 移动性规则  
 启用或禁用移动性规则。  也可以展开此项以启用或禁用个别规则。  
  
 命名规则  
 启用或禁用命名规则。  也可以展开此项以启用或禁用个别规则。  
  
 性能规则  
 启用或禁用性能规则。  也可以展开此项以启用或禁用个别规则。  
  
 可移植性规则  
 启用或禁用可移植性规则。  也可以展开此项以启用或禁用个别规则。  
  
 可靠性规则  
 启用或禁用可靠性规则。  也可以展开此项以启用或禁用个别规则。  
  
 安全规则
