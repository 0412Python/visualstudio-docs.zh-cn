# F1 帮助

登陆到此似乎是误操作所致。 若要按 F1 以寻求关键字或 API 名称方面的帮助，请务必先将光标放在相应单词内部或右侧（紧挨着它），再按 F1。 如果这样仍不起作用，请尝试访问 https://docs.microsoft.com，并搜索帮助信息。

若要搜索文本编辑器本身方面的帮助信息，请参阅[在编辑器中编写代码](../../ide/writing-code-in-the-code-and-text-editor.md)。

## 帮助我们改进 F1 功能

请针对你按 F1 时要查找的内容，向我们提供反馈，进而帮助我们改善 F1 帮助体验。 例如，你的光标置于哪个符号或窗口上？ 请使用下面的“提供文档反馈”按钮。 谢谢！
