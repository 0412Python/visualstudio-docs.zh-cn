# Visual Basic 中经过筛选的完成列表
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

在 Visual Basic 中，IntelliSense 完成列表的底部附近有两个选项卡控件。  **“通用”**选项卡默认情况下处于选中状态，显示最常用于完成正在编写的语句的项。  **“全部”**选项卡显示可用于自动完成的所有项，包括同时位于**“通用”**选项卡上的那些项。
