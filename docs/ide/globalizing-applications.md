# <a name="globalize-applications"></a>全球化应用程序

全球化是指设计和开发针对多个区域性的软件产品的过程。 以下文章适用于 Windows 窗体和 Web 窗体页面。

 [全局 Windows 窗体和 Web 窗体的区域性特定的类](../ide/culture-specific-classes-for-global-windows-forms-and-web-forms.md)：介绍根据区域性设置，为日期、时间、数字、货币和其他信息设置格式的类。
