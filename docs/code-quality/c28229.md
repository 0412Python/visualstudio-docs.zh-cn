# C28229
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28229：批注不支持参数  
  
 此警告意味着注解使用了参数，当注解声明没有参数时。  如果标准宏用于注释时应该是不可能的；此警告在典型使用中不期望可见。
