# C28285
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28285：对于函数，参数中有语法错误  
  
 当在模型文件中遇到一个可能的错误时，代码分析工具报告此警告。  某些源文件错误也会引起此类错误。
