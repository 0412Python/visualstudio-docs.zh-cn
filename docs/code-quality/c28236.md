# <a name="c28236"></a>C28236
警告 C28236: 批注不能在表达式中使用  
  
 此警告指示尝试使用声明要在表达式的上下文中批注的函数。 如果正对批注使用标准宏，则这应不是可能实现的；此警告在正常使用时应不显示。
