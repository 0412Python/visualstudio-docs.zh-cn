# C28236
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28236：不能在表达式中使用该批注  
  
 此警告意味着尝试在表达式上下文一个声明使用的函数是一个注释。  如果标准宏用于注释时应该是不可能的；此警告在典型使用中不期望可见。
