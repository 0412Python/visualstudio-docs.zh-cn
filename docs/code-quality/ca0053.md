# <a name="ca0053"></a>CA0053
无法加载规则程序集。  
  
 这表示无法通过反射加载规则程序集。 这可以发生如果用户具有不正确，生成的自定义规则或分析工具无法加载程序集出于其他原因，例如文件系统访问被拒绝。
