# <a name="c26477-usenullptrnotconstant"></a>C26477 USE_NULLPTR_NOT_CONSTANT

`nullptr` 具有一种特殊类型`nullptr_t`允许具有特殊的 null 处理的重载。 使用`0`或`NULL`来代替`nullptr`绕过类型安全和推断的`nullptr`提供。
