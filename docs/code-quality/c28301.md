# C28301
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28301：\<function\> 的第一个声明没有批注。\<note1\> 请参见 \<filename\>\(\<行\>\)。\<note2\>  
  
 注释当未找到在特定函数中的第一个声明中，报告此警告。
