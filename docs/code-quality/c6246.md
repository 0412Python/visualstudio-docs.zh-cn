# <a name="c6246"></a>C6246
警告 C6246： 局部声明\<变量 > 隐藏外层作用域中的相同名称的声明。 其他信息： 请参阅前面的声明在\<位置 >。  
  
 此警告指示两个声明在局部范围内具有相同的名称。 在内部范围内声明隐藏处外层作用域的名称。 如果打算的使用的外部范围的声明将导致本地声明的使用。  
  
## <a name="example"></a>示例  
 下面的代码将生成此警告：  
  
```  
#include <stdlib.h>  
#define UPPER_LIMIT 256  
int DoSomething( );  
  
int f( )  
{  
  int i = DoSomething( );  
  if (i > UPPER_LIMIT)  
  {  
    int i;  
    i = rand( );  
  }  
  return i;  
}  
```  
  
 若要更正此警告，请使用另一个变量名称，如下面的代码中所示：  
  
```  
#include <stdlib.h>  
#define UPPER_LIMIT 256  
int DoSomething( );  
  
int f ( )  
{  
  int i = DoSomething( );  
  if (i > UPPER_LIMIT)  
  {  
    int j = rand( );  
    return j;  
  }  
  else  
  {  
    return i;  
  }  
}  
```  
  
 此警告仅标识范围重叠。
