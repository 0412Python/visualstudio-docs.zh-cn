# <a name="ca0502"></a>CA0502
不支持的语言

 当用户尝试分析包含一个或多个页面或除 C# 或 Visual Basic 语言编写的类的 ASP.NET 项目时，将发生此错误。 若要解决此错误，请从项目中删除该文件或禁用代码分析。
