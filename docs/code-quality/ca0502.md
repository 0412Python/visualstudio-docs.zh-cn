# <a name="ca0502"></a>CA0502
不支持的语言  
  
 当用户尝试分析 ASP.NET 项目包含一个或多个页或除 C# 或 Visual Basic 语言编写的类时，将出现此错误。 若要修复此错误，请从项目中删除文件或禁用代码分析。
