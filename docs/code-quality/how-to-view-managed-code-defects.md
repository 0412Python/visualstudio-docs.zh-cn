# 如何：查看托管代码缺陷
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

在代码分析中，在代码分析窗口显示错误、警告和消息。  代码分析窗口自动显示当您运行项目或解决方案的代码分析时。  
  
 在分析代码窗口，您可以：  
  
-   **选择警告标题。**  
  
     有关代码分析窗口出现的警告的附加信息。  源文件中引发警告的代码行会在源代码编辑器中显示并突出显示。  
  
-   **选择 警告 ID .**警告 ID 以 **CA**为前缀 并且包含一个四位数字。  例如，CA1234。  
  
     显示警告的 MSDN Library 帮助主题。  帮助主题包含有关警告产生原因的详细信息并且包括的示例来演示警告问题如何产生和解决。
