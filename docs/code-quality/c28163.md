# C28163
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28163：该函数不应从 try\/except 块中调用  
  
 此警告报告，一个函数如果是 `try/except` 块中发现的在 `try/except`  块中不应封闭的类型时此警告被报告。  代码分析工具找到在 `try/except` 块中调用的函数的至少一个路径。
