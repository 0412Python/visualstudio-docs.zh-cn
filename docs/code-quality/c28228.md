# C28228
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28228：批注参数：不能在批注中使用类型  
  
 此警告意味着参数的类型不支持。  注释只能使用有限类型用作参数。  如果标准宏用于注释时应该是不可能的；此警告在典型使用中不期望可见。
