# <a name="c6293"></a>C6293
警告 C6293: for 循环： 进行倒计时从最小值  
  
 此警告意味着 for 循环可能不会按预期。 当循环进行倒计时从最小值，但具有更高版本的终止条件，则它时发生。  
  
 有符号 — 或无符号-以及负增量索引变量将导致循环的计数，直到，会发生溢出。 这将终止循环。  
  
## <a name="example"></a>示例  
 下面的示例代码将生成此警告：  
  
```  
void f( )  
{  
   signed char i;  
  
   for (i = 0; i < 100; i--)  
   {  
      // code ...  
   }  
}  
  
```  
  
 若要更正此警告，请使用下面的代码：  
  
```  
void f( )  
{  
   signed char i;  
  
   for (i = 0; i < 100; i++)  
   {  
      // code ...  
   }  
}  
```
