# <a name="c26481-nopointerarithmetic"></a>C26481 NO_POINTER_ARITHMETIC
此检查支持规则*I.13： 不要将数组传递作为单个指针*。 无论何时中它们应使用更安全类型的缓冲区进行替换的算术运算使用原始指针都喜欢`span<T>`或`vector<T>`。
