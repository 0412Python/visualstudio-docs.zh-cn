# <a name="ca0503"></a>CA0503
属性已弃用。 使用取代属性  
  
 当不推荐使用的代码分析属性包含生成或项目配置文件中的值，则会生成警告 CA0503。 若要解决此警告，使用指定的属性，用于取代它。
