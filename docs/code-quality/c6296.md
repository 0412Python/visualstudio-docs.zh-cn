# C6296
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C6296：for 循环定义错误: 循环体仅执行一次  
  
 此警告意味着 for 循环可能不会按预期方式工作。  当索引无符号，而且循环从零往下计数时，循环的主体只运行一次。  
  
## 示例  
 下面的代码生成此警告：  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i--)  
   {  
      // code ...  
   }  
}  
```  
  
 若要更正此警告，请使用下面的代码：  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i++)  
   {  
      // code ...  
   }  
}  
```
