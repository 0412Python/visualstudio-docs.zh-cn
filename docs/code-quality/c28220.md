# C28220
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28220：批注中的参数应为整数表达式  
  
 此警告意味着为批注的参数应为整数表达式，并且，遇到某些其他类型。  这通常表示不正确的宏代码注释。
