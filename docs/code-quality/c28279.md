# C28279
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28279：对于符号，已找到“begin”，但没有匹配的“end”  
  
 注释语言支持开始和结束 \(`{` 和 `}` C中\)构造结构，并且此配对不对称。  如果使用了宏，此情况可避免。
