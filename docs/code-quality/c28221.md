# C28221
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28221：批注中的参数应为字符串表达式  
  
 此警告意味着为注释的参数应是字符串，以及曾遇到的其他类型。  这通常指示不正确代码的注释宏并且在正常使用时不应可见。
