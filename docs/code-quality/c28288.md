# C28288
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28288：对于函数， \_At\_\(\) 注解中存在语法错误\(无效的参数名\)  
  
 当 `SAL_at` \(`__drv_at`\) 注释被使用并且参数表达式无法在当前上下文解释，代码分析工具报告此警告。  此可能包含一个拼错的参数或成员名称，或 “Return” 或 “this” 关键字拼写错误。
