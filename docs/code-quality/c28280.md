# <a name="c28280"></a>C28280
警告 C28280： 对于符号，结束已找到没有与其匹配的开始

 批注语言支持 begin 和 end (`{`和`}`在 C 中) 构造，并且此配对收到不平衡。 如果使用了宏，则此情况可避免。
