# <a name="ca0055"></a>CA0055
无法加载文件：\<路径 >

 代码分析工具无法加载分析指定的目标文件。 如果找不到文件或文件的访问被拒绝，将发生这种情况。 通常，如果用户选择重写在 Visual Studio 中，输入程序集属性，或手动调用 fxcopcmd.exe 出现此情况。
