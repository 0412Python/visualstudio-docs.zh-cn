# <a name="ca0055"></a>CA0055
无法加载文件：\<路径 >  
  
 代码分析工具无法加载分析指定的目标文件。 如果未找到文件或文件的访问被拒绝，将发生这种情况。 通常情况下，发生这种情况在用户选择重写中的输入程序集属性[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)]，或手动调用 fxcopcmd.exe。
