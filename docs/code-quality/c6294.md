# <a name="c6294"></a>C6294
警告 C6294: for 循环： 初始条件不满足测试。 未执行的循环正文  
  
 此警告意味着无法执行 for 循环，因为终止条件为 true。 此警告意味着不正确捕获程序员的意图。  
  
## <a name="example"></a>示例  
 下面的示例代码将生成此警告，因为 MAX_VALUE 为 0:  
  
```  
#define MAX_VALUE 0  
void f()  
{  
  int i;  
  for (i = 0; i < MAX_VALUE; i++)  
  {  
    // code   
  }  
}  
```  
  
 下面的示例代码通过 MAX_VALUE 将值更改为 25 来更正此警告  
  
```  
#define MAX_VALUE 25  
void f()  
{  
  int i;  
  for (i = 0; i < MAX_VALUE; i++)  
  {  
    // code   
  }  
}  
```
