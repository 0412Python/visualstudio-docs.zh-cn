# C28232
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告C28232: pre、post 或 deref 不适用于任何批注。  
  
 此警告意味着 `_Pre_`、 `_Post_` 或 `_Deref_` 运算符在表达式显示注释，不带注释后面的功能；修饰符将忽略，但这指示一不正确代码的注释。
