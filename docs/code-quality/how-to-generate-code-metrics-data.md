# 如何：生成代码度量数据
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

您可以决定是为整个解决方案还是为选定项目生成代码度量结果。  
  
 **要求**  
  
 Visual Studio 2012 旗舰版或 Visual Studio 2012 高级专业版  
  
### 为整个解决方案生成代码度量结果  
  
-   单击**“分析菜单”**，然后单击**“为解决方案计算代码度量值”**。  
  
     \- 或 \-  
  
-   在**“解决方案资源管理器”**中右击解决方案，再单击**“计算代码度量值”**。  
  
     \- 或 \-  
  
-   在**“代码度量结果”**窗口中，单击**“为解决方案计算代码度量值”**按钮。  
  
     此时将生成结果并显示**“代码度量结果”**窗口。  
  
### 为一个或多个选定项目生成代码度量结果  
  
1.  在**“解决方案资源管理器”**中，选择一个或多个项目。  
  
2.  单击**“分析”**，再单击**“为选定项目生成代码度量值”**。  
  
     \- 或 \-  
  
     右击您的选择项，然后单击**“计算代码度量值”**。  
  
 此时将生成结果并显示**“代码度量结果”**窗口。  
  
### 查看结果详细信息  
  
-   展开**“层次结构”**列中的树。
