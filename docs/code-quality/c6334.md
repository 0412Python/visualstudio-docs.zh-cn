# <a name="c6334"></a>C6334
警告 C6334: sizeof 运算符应用于具有运算符的表达式可能产生意外的结果  
  
 此警告意味着的滥用`sizeof`运算符。 `sizeof`运算符，当应用于表达式中，生成的结果表达式的类型的大小。  
  
 例如，在以下代码：  
  
```  
char     a[10];  
size_t   x;  
  
x = sizeof (a - 1);  
```  
  
 `x`将被赋予值 4，不月 9 日，因为生成的表达式不再指向数组的指针`a`，但只是一个指针。  
  
## <a name="example"></a>示例  
 下面的代码将生成此警告：  
  
```  
void f( )  
{     
  size_t x;  
  char a[10];  
  
  x= sizeof (a - 4);  
  // code...  
}  
```  
  
 若要更正此警告，请使用下面的代码：  
  
```  
void f( )     
{  
  size_t x;  
  char a[10];  
  
  x= sizeof (a) - 4;  
  // code...  
}   
```
