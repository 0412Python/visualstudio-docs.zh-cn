# C28350
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28350：批注 \<annotation\> 描述非条件性适用的情况。  
  
 通常当注释应用于 C\/C\+\+ 类型检查的位置时生成此警告。
