# <a name="c6386"></a>C6386
警告 C6386： 缓冲区溢出： 访问\<缓冲区名称 >，可写大小是\<size1 > 个字节，但\<size2 > 可能写入字节： 行： x，y

 此警告指示指定的缓冲区的可写范围可能会比使用对其进行写入的索引小。 这可能导致缓冲区溢出。

## <a name="example"></a>示例
 下面的代码将生成这两个此警告和[C6201](../code-quality/c6201.md):

```
#define MAX 25

void f ( )
{
  char ar[MAX];
  //Code ...
  ar[MAX] = '\0';
}
```

 若要更正这两个警告，请使用以下代码：

```
#define MAX 25

void f ( )
{
   char a[MAX];
   // code...
   a[MAX - 1] = '\0';
}
```
