# C28160
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28160：错误批注  
  
 当 `__drv_error` 遇到注释后，报告此警告。  此注释用于标记应修复的编码做法，并可用来使用 `__drv_when` 批注指示参数的特定组合。
