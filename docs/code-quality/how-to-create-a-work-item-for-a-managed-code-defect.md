# 如何：为托管代码缺陷创建工作项
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

您可以使用工作项跟踪功能从 [!INCLUDE[vsPreLong](../code-quality/includes/vsprelong_md.md)]内记录工作项。  若要使用此功能，您的项目必须是 [!INCLUDE[esprfound](../code-quality/includes/esprfound_md.md)] 中团队项目的一部分。  
  
### 为托管代码缺陷创建工作项  
  
1.  在 **代码分析** 窗口，选择一个警告。  
  
2.  选择**Actions（行动）**，然后选择 **创建工作项** 并选择要创建的工作项类型。  
  
     将创建一个新的工作项，以便您指定缺陷信息。  
  
### 为多个托管代码缺陷创建工作项  
  
1.  在**“错误列表”**中，选择多个警告，然后右击这些警告。  
  
2.  指向**“创建工作项”**，然后单击要创建的工作项的类型。  
  
     将为所有选定警告创建一个工作项，以便您指定 bug 信息。
