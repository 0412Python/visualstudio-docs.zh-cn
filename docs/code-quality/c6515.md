# <a name="c6515"></a>C6515
警告 C6515： 无效的批注：\<名称 > 属性仅用于指针类型的值  
  
 此警告指示在指针上使用的属性已应用到非指针类型。 批注属性的列表，请参阅[批注属性](http://msdn.microsoft.com/en-us/f77b4370-6bda-4294-bd2a-e7d0df182a3d)。  
  
## <a name="example"></a>示例  
 下面的代码将生成此警告：  
  
```cpp  
#include <sal.h>  
  
void f(_Readable_bytes_(c) char pc,  size_t c)  
{  
    // code ...  
}  
```  
  
 若要更正此警告，请使用下面的代码：  
  
```  
#include <sal.h>  
  
void f(_Readable_bytes_(c) char * pc,  size_t c)  
{  
    // code ...  
}  
```
