# <a name="c28199"></a>C28199
警告 C28199： 可能使用未初始化的内存

 此消息意味着变量的地址已被采用，但未发现其分配。

 指定的变量正在使用而未进行显式初始化，但它的地址在某一时刻已被采用，这表明它可能以一种代码分析工具不可见的方式进行了初始化。

 如果变量在函数之外进行初始化，则此警告可能有误。

 如果参数具有 `_Out_` 或 `_Inout_` 批注并且变量未初始化，则代码分析工具将报告此警告。
