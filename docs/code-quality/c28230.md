# C28230
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28230：参数类型没有成员。  
  
 此警告意味着为批注的参数尝试访问 `struct`，`class`，或者 `union` 并且此成员不存在。
