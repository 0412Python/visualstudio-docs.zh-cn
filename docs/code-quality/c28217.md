# C28217
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28217：对于函数，批注的参数数量与在文件中找到的数量不匹配  
  
 注释在当前行和消息中的行是不一致的。  如果标准宏用于注释则它时不可能的；此警告在正常使用中不可见。
