# 测量托管代码的复杂性和可维护性
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

随着现代软件应用程序复杂性的增加，使代码保持可靠和可维护的难度也增加了。  在最近几年，许多软件测量方法（称作代码度量）已经得到发展，可以帮助开发人员了解他们的哪些代码需要返工或进一步测试。  
  
 开发人员可以使用 [!INCLUDE[vsprvsts](../code-quality/includes/vsprvsts_md.md)] 生成测量托管代码复杂性和可维护性的代码度量数据。  可以为整个解决方案或单个项目生成代码度量数据。  
  
 本节介绍如何在 [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] 中生成和使用代码度量。
