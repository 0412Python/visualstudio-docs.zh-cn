# <a name="c28216"></a>C28216
警告 C28216: _Check_return\_批注仅应用于特定函数参数的后置条件。  
  
 `_Check_return_` 批注应用在了除 `__post` 外的上下文中；它可能需要一个 `__post`（或 `__drv_out`）修饰符，或者可能未正确放置。
