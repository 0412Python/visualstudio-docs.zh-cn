# C28216
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28216： \_Check\_return\_注释仅适用于特定函数参数的后置条件。  
  
 `_Check_return_` 批注应用在一个上下文中而不是除 `__post`；它可能需要一个`__post` （或 `__drv_out`\) 修饰符，或者可能没有正确放置它。
