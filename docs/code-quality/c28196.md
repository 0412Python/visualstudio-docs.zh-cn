# C28196
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28196：未满足要求。（该表达式未计算为 true。）  
  
 此警告意味着函数的函数原型添加分析具有 `__notnull`、`__null` 或 `__drv_valueIs` 上的 `_Out_` 参数或返回值，但是，该值返回注释是不一致的。
