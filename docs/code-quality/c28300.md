# C28300
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28300：\<parameter\_name\>:运算符 \<operator\_name\> 的不兼容类型的表达式操作数  
  
 当操作数类型参数不与运算符匹配时，将报告此警告。
