# C28219
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28219：批注中的参数应为批注预期的枚举成员  
  
 该注释的一个参数应是名为 `enum` 类型的成员，这样，其他符号遇到时；使用 `enum` 类型的成员。  这通常表示不正确的宏代码注释。
