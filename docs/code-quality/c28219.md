# <a name="c28219"></a>C28219
警告 C28219:批注中的批注参数所需的枚举成员

 批注参数应为命名的成员`enum`遇到了类型和一些其他符号; 使用的成员`enum`类型。 这通常表示存在错误编码的批注宏。
