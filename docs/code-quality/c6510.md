# C6510
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C6510:无效的批注：NullTerminated属性。元素为整型或指针类型的缓冲区只能使用：\<函数\> \<参数\>。  
  
 这个警告表示了不正确的使用**NullTerminated**属性（那些以`_z`结尾的）。  只能对指针或数组类型使用此属性类型。  
  
## 示例  
 下面的代码生成此警告：  
  
```  
#include <sal.h>  
  
void f(_In_z_ char x)  
{  
    // code ...  
}  
```  
  
 若要更正此警告，请使用下面的代码：  
  
```  
#include <sal.h>  
  
void f(_In_z_ char * x)  
{  
    // code ...  
}  
```
