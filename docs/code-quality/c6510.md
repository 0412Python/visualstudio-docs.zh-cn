# <a name="c6510"></a>C6510
警告 C6510： 无效的批注: NullTerminated 属性可能只能用于其元素的缓冲区的整型或指针类型： 函数\<函数 >\<参数 >。

 此警告意味着的使用不当**NullTerminated**属性 (那些以`_z`)。 只能对指针或数组类型使用此属性类型。

## <a name="example"></a>示例
 下面的代码将生成此警告：

```
#include <sal.h>

void f(_In_z_ char x)
{
    // code ...
}
```

 若要更正此警告，请使用下面的代码：

```
#include <sal.h>

void f(_In_z_ char * x)
{
    // code ...
}
```
