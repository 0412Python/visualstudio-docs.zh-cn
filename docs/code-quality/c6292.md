# <a name="c6292"></a>C6292
警告 C6292: for 循环： 从最大值计算  
  
 此警告意味着 for 循环可能不会按预期。  
  
 当循环计数从最大值，但是具有较低的终止条件，则它时发生。 仅后发生整数溢出时，此循环将终止。  
  
## <a name="example"></a>示例  
 下面的代码将生成此警告：  
  
```  
void f( )  
{  
   int i;  
  
   for (i = 100; i >= 0; i++)  
   {  
      // code ...  
   }  
}  
```  
  
 若要更正此警告，请使用下面的代码：  
  
```  
void f( )  
{  
   int i;  
  
  for (i = 100; i >= 0; i--)  
   {  
      // code ...  
   }  
}  
```
