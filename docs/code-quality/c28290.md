# C28290
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28290：函数的批注包含的外部对象多于参数的实际数量  
  
 当为一个包含多于实际参数数的外部参数的函数做注解时，代码分析工具报告此警告。
