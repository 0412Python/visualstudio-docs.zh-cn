# C28291
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28291：对于函数 \<x\> 在参数 \<数字\>，在 deref 级别为 0 时发送 null\/notnull 毫无意义。  
  
 当取消引用级别零参数的后置条件指定具有空\/非无效属性时，代码分析工具报告此警告。  此错误发生因为在取消引用级别零的值不能更改。
