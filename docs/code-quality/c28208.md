# C28208
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28208：函数 \<function\> 早前已通过位于 \<文件\>（\<行\>）中的不同参数列表进行了定义。  一些分析工具将产生不正确的结果  
  
 当已知函数的定义不匹配另一个事件时，将报告此警告。
