# <a name="c28208"></a>C28208
警告 C28208： 函数\<函数 > 之前已定义了在不同的参数列表与\<文件 > (\<行 >)。 某些分析工具会生成不正确的结果

 当函数的已知的定义与另一个匹配项不匹配时，将报告此警告。
