# <a name="ca0504"></a>CA0504
规则目录被忽略，因为它不存在

 中指定一个目录时，会生成警告 CA504 **CodeAnalysisRuleDirectories**找不到项目或生成的配置文件中的属性。 若要解决此问题，请输入有效的目录路径。
