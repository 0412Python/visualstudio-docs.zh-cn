# <a name="c28194"></a>C28194
警告 C28194： 了函数声明为别名中变量的值并将未这样的情况下退出  
  
 此警告意味着所分析函数的函数原型有`__drv_isAliased`批注，这表示它将*别名*指定自变量 （即，将值分配它时保留下来的方式从函数返回）。 但是，此函数不会沿批注所指示的路径用别名来命名此自变量。 大多数用别名来命名某个变量的函数都会将它的值保存到全局数据结构。
