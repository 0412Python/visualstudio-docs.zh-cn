# C28351
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28351：批注 \<annotation\> 描述不能使用动态值（变量）的情况。  
  
 当注释应用于 C\/C\+\+ 类型检查的位置，此警告被报告。  
  
 在 `_When_` 中的表达式应计算为常数。  忽略 `_When_`。
