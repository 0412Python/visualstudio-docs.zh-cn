# <a name="c6295"></a>C6295
警告 C6295: for 循环：\<变量 > 的值为"最小值"到"max"的范围。 无限期地执行循环  
  
 此警告意味着 for 循环可能不会按预期。 For 循环使用 >= 对照零 (0) 测试无符号值。 在结果始终为 true，因此循环是无限期。  
  
## <a name="example"></a>示例  
 下面的代码将生成此警告：  
  
```  
void f( )  
{  
  for (unsigned int i = 100; i >= 0; i--)   
  {  
    // code ...  
  }  
}  
```  
  
 若要更正此警告，请使用下面的代码：  
  
```  
void f( )  
{  
   for (unsigned int i = 100; i > 0; i--)  
   {  
      // code ...  
   }  
}  
```
