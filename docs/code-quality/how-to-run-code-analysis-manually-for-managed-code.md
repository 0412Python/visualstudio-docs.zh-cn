# <a name="how-to-run-code-analysis-manually-for-managed-code"></a>如何：手动运行托管代码的代码分析
代码分析工具为您提供了有关在源代码中可能存在的缺陷的信息。 你可以使用代码项目中，每个生成自动运行代码分析，还可以手动运行代码分析。 在项目属性页代码分析页上指定的规则时运行代码分析签入。 有关更多信息，请参见[如何：配置托管的代码项目的代码分析](../code-quality/how-to-configure-code-analysis-for-a-managed-code-project.md)

### <a name="to-run-code-analysis-manually"></a>若要手动运行代码分析

1.  在中**解决方案资源管理器**，单击该项目。

2.  上**分析**菜单上，单击**上运行代码分析***项目名称*。
