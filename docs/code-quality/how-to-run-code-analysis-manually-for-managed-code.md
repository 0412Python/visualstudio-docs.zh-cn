# 如何：手动运行托管代码的代码分析
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

代码分析工具为您提供了有关源代码中可能存在的缺陷的信息。  可以在生成每个代码项目时自动运行代码分析，也可以手动运行代码分析。  运行代码分析时选中的规则是在项目属性页的“代码分析”页上指定的。  有关更多信息，请参阅[如何：配置托管代码项目的代码分析](../code-quality/how-to-configure-code-analysis-for-a-managed-code-project.md)。  
  
### 手动运行代码分析  
  
1.  在**“解决方案资源管理器”**中，单击该项目。  
  
2.  在**“分析”**菜单上，单击**“运行代码分析对于”** *Project Name*。
