# C28109
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28109：无法在调用函数时保留变量  
  
 程序当调用此函数时保留了不应保留的资源。  通常，它表示此资源无意中获得两次。  代码分析工具报告此警告，当在同一个函数中获得和释放资源时。  
  
## 示例  
 下面的代码示例生成此警告：  
  
```  
ExAcquireResourceLite(resource, true);  
...  
ExAcquireResourceLite(resource, true);  
```  
  
 下面的代码示例避免此警告：  
  
```  
ExAcquireResourceLite(resource, true);  
```
