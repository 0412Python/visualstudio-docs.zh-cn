# C28223
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28223：对于批注，参数未找到预期的标记\/标识符  
  
 此警告意味着为注释的参数应是标识符，以及曾遇到的其他符号。  这通常表示不正确的宏代码注释。  在此位置使用 C 或 C\+\+ 关键字将导致此错误。
