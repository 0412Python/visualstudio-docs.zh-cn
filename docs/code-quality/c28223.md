# <a name="c28223"></a>C28223
警告 C28223： 未找到批注，预期的标记/标识符参数  
  
 此警告意味着批注参数应为标识符，并遇到了一些其他符号。 这通常表示存在错误编码的批注宏。 在此位置中的 C 或 c + + 关键字的使用会导致此错误。
