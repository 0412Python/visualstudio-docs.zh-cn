# <a name="c28263"></a>C28263
警告 C28263： 条件批注中的语法错误中找到内部批注  
  
 当指定函数的返回值具有条件值时，代码分析工具将报告此警告。 此警告意味着批注而不是所分析的代码中存在错误。 如果函数声明位于头文件中，则应纠正批注，以便其与头文件相匹配。  
  
 指定函数和参数的结果列表中有多个无条件值。  
  
 通常，这意味着使用了多个无条件 `_Null_` 或 `__drv_valueIs` 批注来指定结果值。
