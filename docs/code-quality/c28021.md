# C28021
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28021：参数 \<param\>（用 \<anno\> 批注）必须是指针  
  
 当注解的对象不是指针类型时，将报告此警告。  此注释不能使用 `void` 或整型。
