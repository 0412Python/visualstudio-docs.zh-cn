# C28272
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28272：检查时针对函数参数的批注与函数声明不一致  
  
 此警告意味着错误在注释，不在被分析的代码。  出现在函数定义上的注释与出现在声明上的不一致。  这两个注释应该被解析为匹配。
