# C28226
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28226：注解也不能为 PrimOp\(在当前声明中\)  
  
 此警告意味着被命名的注释被声明为 PrimOp，并且为先前声明的标准注释。  如果标准宏用于注释则它时不可能的；此警告在正常使用中不可见。
