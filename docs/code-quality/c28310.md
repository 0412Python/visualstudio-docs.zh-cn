# <a name="c28310"></a>C28310
警告 28310： 上的 < annotation_name > 批注\<函数 >\<参数 > 没有 SAL 版本。

 源代码中使用的所有 SAL 批注都应具有通过使用 SAL_name 应用的批注版本。 这需要在宏定义中进行更正。

 此警告对于每个声明仅报告一次。 有关更多过时的 SAL 的信息，请查看声明的其他部分。
