# <a name="c6281"></a>C6281
警告 6281-运算顺序不正确： 关系运算符具有优先权要高于位运算符  
  
 此警告意味着运算符的优先级可能的错误。 这可能会产生不正确的结果。 你应该检查优先，并使用括号来明确指定优先级。 关系运算符 (\<，>， \<=、 > =、 = =、 ！ =) 具有更高的优先级高于位运算符 (& &#124; ^)。  
  
## <a name="example"></a>示例  
 下面的代码将生成此警告：  
  
```  
#include <stdlib.h>  
#define FORMAT 1  
#define TYPE 2  
  
void f(int input)  
{  
  if (FORMAT & TYPE != input)  
  {  
    // code...  
  }  
}  
```  
  
 下面的代码使用括号来更正此警告：  
  
```  
#include <stdlib.h>  
#define FORMAT 1  
#define TYPE 2  
  
void f(int input)  
{  
  if ((FORMAT & TYPE) != input)  
  {  
    // code...  
  }  
}  
```
