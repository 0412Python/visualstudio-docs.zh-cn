# <a name="c28225"></a>C28225
警告 C28225： 未在批注中找到所需的参数的正确数目

 此警告意味着所命名的批注用于的参数数目不正确。 如果正对批注使用标准宏，则这应不是可能实现的；此警告在正常使用时应不显示。
