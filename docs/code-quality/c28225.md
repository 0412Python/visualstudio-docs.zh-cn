# C28225
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28225：在批注中找到的所需参数的数量不正确  
  
 此警告意味着批注使用具有命名参数的错误数目。  如果标准宏用于注释时应该是不可能的；此警告在典型使用中不期望可见。
