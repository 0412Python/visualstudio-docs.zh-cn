# <a name="c28022"></a>C28022
警告 C28022:函数类\<classlist1 > 在此函数不匹配的函数类\<classlist2 > 上使用定义它的 typedef。

 当批注中存在错误时将报告此警告。 Typedef 和函数本身具有`_Function_class_`注释，但它们不匹配。 如果同时使用它们必须匹配。
