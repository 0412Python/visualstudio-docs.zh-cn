# <a name="c28022"></a>C28022
警告 C28022： 函数类\<classlist1 > 对该函数不匹配函数类\<classlist2 > 上用于定义它的 typedef。  
  
 当批注中存在错误时将报告此警告。 Typedef 和该函数本身具有`_Function_class_`批注，但它们不匹配。 如果同时使用这它们必须与匹配。
