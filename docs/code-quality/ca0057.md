# <a name="ca0057"></a>CA0057
冲突不能映射到当前的目标和规则集

 如果使用项目文件手动调用 FxCopCmd.exe 会发生此错误或 / 导入具有不能映射到当前组的目标和规则的冲突的 XML 报表。

 仅当执行分析时，才会出现使用该工具的命令行版本。
