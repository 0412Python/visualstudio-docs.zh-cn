# <a name="ca0057"></a>CA0057
冲突无法映射到当前的目标和规则集

 如果使用项目文件手动调用 FxCopCmd.exe，则会发生此错误或 / 导入的发生冲突，无法映射到当前的目标和规则集的 XML 报告。

 分析完成时，才会使用该工具的命令行版本。
