# <a name="c6274"></a>C6274
警告 C6274： 作为参数传递的非字符\<数 > 字符对的调用的需要时\<函数 >  
  
 此警告意味着格式字符串，指定一个字符，不需要 (例如，`%c`或`%C`规范)，但所传递的非整数，如浮点数、 字符串或结构。 此缺陷很可能导致错误的输出。  
  
## <a name="example"></a>示例  
 下面的代码将生成此警告：  
  
```  
#include <stdio.h>  
#include <string.h>  
  
void f(char str[])  
{  
   char buff[5];  
  
   sprintf(buff,"%c",str);  
}  
```  
  
 若要更正此警告，请使用下面的代码：  
  
```  
#include <stdio.h>  
#include <string.h>  
  
void f(char str[])  
{  
   char buff[5];  
  
   sprintf(buff,"%c",str[0]);  
}  
```  
  
 下面的代码使用安全的字符串操作函数`sprintf_s`，若要更正此警告：  
  
```  
#include <stdio.h>  
#include <string.h>  
  
void f(char str[])  
{  
   char buff[5];  
  
   sprintf_s(buff,5,"%c", str[0]);  
}  
```
