# C28222
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 28222:\_Yes\_、\_No\_ 或 \_Maybe\_ 注释为期望的  
  
 此警告意味着期望的注释的参数应是 `_Yes_`, `_No_`, 或 `_Maybe_` 符号之一，以及其他曾遇到的符号。  这通常表示不正确的宏代码注释。
