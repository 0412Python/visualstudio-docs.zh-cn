# <a name="c28235"></a>C28235
警告 C28235: 函数无法作为批注单独存在

 此警告意味着试图使用未在批注上下文声明为批注的函数。 这包括在独立上下文中使用基元操作 (PrimOp)。 如果正对批注使用标准宏，则这应不是可能实现的。 此警告在正常使用时应不显示。
