# C28275
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28275:\_Macro\_value\_的参数为空  
  
 此警告在模型文件内提示有内部错误，而不在代码分析时提示。  *macroValue*函数调用时不带参数。
