# <a name="c6303"></a>C6303
警告 C6303： 格式字符串不匹配： 宽字符字符串作为参数传递\<数 > 何时在调用需要字符串\<函数 >

 此警告指示格式字符串指定需要字符字符串。 但是，传递的是宽字符字符串。 此缺陷有可能会导致崩溃或某种形式的损坏。

## <a name="example"></a>示例
 下面的示例代码将生成此警告：

```
#include <stdio.h>

void f()
{
  wchar_t buff[5] = L"hi";

  printf("%s", buff);
}
```

 若要更正此警告，请使用`%ls`中下面的示例代码所示：

```
#include <stdio.h>

void f()
{
  wchar_t buff[5] = L"hi";

  printf("%ls", buff);
}
```

 下面的示例代码使用安全的字符串操作函数`printf_s`若要更正此警告：

```
#include <stdio.h>

void f()
{
  wchar_t buff[5] = L"hi";

  printf_s("%ls",buff);
}
```
