# <a name="c28213"></a>C28213
警告 C28213:`_Use_decl_annotations_`批注必须用于引用，而不进行修改上, 一个声明。 \<为什么 >

 当引用之前的声明，但没有必要的 `_Use_decl_annotations_` 批注时，将报告此警告。
