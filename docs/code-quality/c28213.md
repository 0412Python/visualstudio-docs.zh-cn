# C28213
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28213：`_Use_decl_annotations_` 批注必须用于引用\(无需修改\)以前的声明。\<原因\>  
  
 当某个声明引用，但没有必需 `_Use_decl_annotations_` 注释时，报告此警告。
