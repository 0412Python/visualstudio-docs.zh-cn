# <a name="c6323"></a>C6323
警告 C6323-使用布尔类型的算术运算符

 如果布尔数据类型上使用算术运算符，则会发生此警告。 使用的运算符不正确可能会产生不正确的结果。 它还指示程序员的目的不会反映在代码中。

## <a name="example"></a>示例
 下面的代码将生成此警告：

```
int test(bool a, bool b)
{
    int c = a + b;     //C6323
    return c;
}
```

 若要更正此警告，请使用正确的数据类型和运算符。

```
int test(int a, int b)
{
    int c = a + b;
    return c;
}
```
