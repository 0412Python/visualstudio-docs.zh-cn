# C28210
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 28210：`_On_failure_` 上下文的批注不得位于显式的 pre 上下文中  
  
 在函数返回后，注释 `_On_failure_` 后面必须显式或隐式来指示在 `__post` 上下文，即，函数返回后应用。使用 `_drv_out` 确保这一点。
