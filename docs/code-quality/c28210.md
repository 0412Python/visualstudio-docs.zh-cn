# <a name="c28210"></a>C28210
警告 28210： 的批注`_On_failure_`上下文不得位于显式的 pre 上下文

 必须在 `_On_failure_` 上下文中显式或隐式指明批注 `__post`，也就是说，要将其应用在函数返回值之后。  使用 `_drv_out` 可确保这一点。
