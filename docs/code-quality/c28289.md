# C28289
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28289: 对于函数: ReadableTo 或 WritableTo 作为参数没有限制规范。  
  
 当函数\/参数注释为已知的错编时，代码分析工具报告此警告。
