# <a name="c6313"></a>C6313
警告 C6313： 运算符不正确： 零值标志不能使用按位进行测试-和。 使用相等性测试检查零值标志

 此警告指示零的常量值已作为的按位的自变量提供的和 (`&`) 在测试上下文中的运算符。 生成的表达式是常量并计算结果为 false;结果都不同非预期。

 这通常由于使用按位-和要测试其值为零的标志。 若要测试零值标志，测试相等性必须执行，例如，使用`==`或`!=`。

## <a name="example"></a>示例
 下面的代码将生成此警告：

```
#define FLAG 0

void f(int Flags )
{
  if (Flags & FLAG)
  {
    // code
  }
}
```

 若要更正此警告，请使用下面的代码：

```
#define FLAG 0

void f(int Flags )
{
  if (Flags == FLAG)
  {
    // code
  }
}
```
