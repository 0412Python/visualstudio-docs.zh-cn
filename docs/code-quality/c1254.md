# C1254
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C1254：插件版本不匹配：版本与 PREfast 驱动程序的版本不匹配  
  
 代码分析工具报告此警告，当错误出现在插件内部版本时，而不是在分析的代码中。
