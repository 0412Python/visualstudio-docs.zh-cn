# C28204
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28204：\<function\>:只批注一种此类重载和位于 \<文件名\>（\<行\>）处一种重载；而对于 \<filename\>:必须批注两者或均不批注。  
  
 当注释中存在错误时报告此警告。
