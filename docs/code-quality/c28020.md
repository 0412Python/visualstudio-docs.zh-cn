# <a name="c28020"></a>C28020
警告 C28020： 表达式\<expr > 不在此调用为 true

 将报告此警告时 _Satisfies\_列出的表达式不为 true。 这通常表示存在错误的参数。

 如果在函数声明时发生此警告，则表示批注指明了一种不可能的条件。
