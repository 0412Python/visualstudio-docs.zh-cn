# C28020
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28020：表达式 \<expr\> 对于此调用不适用  
  
 在列出的\_Satisfies\_表达式不为 True 时，报告此警告。  这通常指示了不正确的参数。  
  
 如果在函数声明时发生，此注解表示一个不可能的情况。
