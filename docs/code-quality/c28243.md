# C28243
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28243：参数上函数的批注需要的取消引用次数多于已批注的实际类型所允许的次数  
  
 `__deref`运算符上的注释是多于指针由参数类型定义的级别数。  通过改变无论是在注释的数量或所引用的参数的指针水平更正此警告。
