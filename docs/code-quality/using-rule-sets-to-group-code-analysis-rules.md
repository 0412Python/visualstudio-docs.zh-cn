# <a name="use-rule-sets-to-group-code-analysis-rules"></a>使用规则集组合代码分析规则

在 Visual Studio 中配置代码分析时，你可以从内置的列表中选择*规则集*。 规则集是标识目标的问题和特定条件的代码分析规则的逻辑分组。 例如，可以应用一种旨在扫描公开可用 Api 代码的规则集，也可以应用只包括最少建议规则的规则集。 你还可以应用包含所有规则的规则集。

你可以自定义规则集通过添加或删除规则，或者通过更改规则严重级别以显示为警告或中的错误**错误列表**。 自定义的规则集可以满足为特定的开发环境的需要。 自定义规则集时，规则集页将提供搜索和筛选工具来帮助你在过程中。
