# C28211
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28211：SAL\_context 需要静态上下文名称  
  
 此警告意味着为 `_Static_context_` 的注释操作数必须是一工具定义的上下文中的名称。  此警告在正常使用不应当发生。
