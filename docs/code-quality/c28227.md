# C28227
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28227：批注也不能为 PrimOp（请参见以前的声明）  
  
 此警告意味着被命名的注释被声明为普通注释，并且为先前声明 PrimOp。  如果标准宏用于注释时应该是不可能的；此警告在典型使用中不期望可见。
