# <a name="c28227"></a>C28227
警告 C28227： 批注也不能为 PrimOp （请参阅以前的声明）  
  
 此警告意味着所命名的批注即将被声明为普通批注，和先前已声明为 PrimOp。 如果正对批注使用标准宏，则这应不是可能实现的；此警告在正常使用时应不显示。
