# <a name="c28239"></a>C28239
警告 C28239：参数的批注同时具有值 (stringValue 或 longValue) 和 paramn=xxx。 仅使用 paramn=xxx  
  
 PREfast 模型文件发生了内部错误。 此警告在正常使用时不应出现。
