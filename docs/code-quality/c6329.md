# <a name="c6329"></a>C6329
警告 C6329： 返回到调用值\<函数 > 不应检查对\<数量 >

 程序正在将一个数字与调用 `CreateFile` 返回的值进行比较。 如果 `CreateFile` 成功，它将返回打开的对象句柄。 如果失败，则返回 `INVALID_HANDLE_VALUE`。

## <a name="example"></a>示例
 此代码会导致警告：

```cpp
if (CreateFile() == NULL)
    {
        return;
    }
```

## <a name="example"></a>示例
 此代码可更正错误：

```cpp
if (CreateFile() == INVALID_HANDLE_VALUE)
    {
        return;
    }
```
