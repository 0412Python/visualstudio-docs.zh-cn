# C28212
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28212：批注需要指针表达式  
  
 此警告意味着为注释的编号参数 \(不是标注的函数\) 应该是指针或数组类型，除非遇到某些其他类型。  需要更正的注释。
