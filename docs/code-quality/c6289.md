# <a name="c6289"></a>C6289
警告 C6289： 运算符不正确： 通过互相排斥 &#124; &#124;始终为非零常数。 你未打算使用 （& a) 和相反？  
  
 此警告表示测试表达式中变量所测试的对照两个不同的常数，该结果取决于哪个条件为 true。 这始终评估为 true。  
  
 此问题通常由于使用`||`代替了`&&`，但也可以通过使用导致`!=`其中`==`重新运行安装程序。  
  
## <a name="example"></a>示例  
 下面的代码将生成此警告：  
  
```  
void f(int x)  
{  
  if ((x != 1) || (x != 3))  
  {  
    // code   
  }  
}  
```  
  
 若要更正此警告，请使用下面的代码：  
  
```  
void f(int x)  
{  
  if ((x != 1) && (x != 3))  
  {  
    // code   
  }  
}  
  
/* or */  
void f(int x)  
{  
  if ((x == 1) || (x == 3))  
  {  
    // code   
  }  
}  
```
