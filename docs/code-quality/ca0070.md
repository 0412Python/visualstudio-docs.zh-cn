# <a name="ca0070"></a>CA0070
代码分析用于分析程序集并遇到权限特性和至少一个以下条件为 true 时，将发生此错误：

-   代码分析找不到包含属性的程序集。

-   代码分析没有加载程序集的权限。

-   代码分析来加载的程序集不包含该属性。

 在大多数情况下，可以通过确保你具有的最新版本的代码分析工具，从而解决此问题。
