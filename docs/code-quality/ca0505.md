# <a name="ca0505"></a>CA0505

不推荐使用的属性将被忽略，因为取代属性定义。

不推荐使用和取代代码分析属性在项目中具有值，或生成配置文件时，会生成警告 CA505。 若要解决此问题，请删除不推荐使用的属性。
