# <a name="ca0505"></a>CA0505
不推荐使用的属性将被忽略，因为定义取代属性。  
  
 如果指定的弃用和取代代码分析属性包含在项目中的值，或生成配置文件时，会生成警告 CA505。 若要解决此问题，删除不推荐使用的属性。
