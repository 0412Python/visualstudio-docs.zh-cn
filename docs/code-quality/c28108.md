# <a name="c28108"></a>C28108
警告 C28108： 变量包含意外的资源  
  
 驱动程序正在使用的资源中的预期的 C 语言类型，但具有不同的语义类型。  
  
## <a name="example"></a>示例  
 下面的代码示例将生成此警告：  
  
```  
KeAcquireInStackSpinLock(spinLock, lockHandle);  
...  
KeReleaseSpinLock(spinLock, 0);  
```  
  
 下面的代码示例可避免此警告：  
  
```  
KeAcquireInStackSpinLock(spinLock, lockHandle);  
...  
KeReleaseInStackSpinLock(lockHandle);  
```
