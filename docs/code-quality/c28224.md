# C28224
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28224：批注需要参数  
  
 此警告意味着此注解使用时没有含参数，但需要至少一个参数。  如果标准宏用于注释则它时不可能的；此警告在正常使用中不可见。
