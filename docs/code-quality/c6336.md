# <a name="c6336"></a>C6336
警告 C6336： 算术运算符的优先级高于问号运算符，请使用括号来阐明优先级  
  
 此警告意味着可能运算符优先级问题。 '+'，'-'，' *' 和 '/' 运算符的优先级高于？ 运算符。 如果表达式中的优先级不正确，请使用圆括号更改运算符优先级。  
  
## <a name="example"></a>示例  
 下面的代码将生成此警告：  
  
```  
int Count();  
  
void f(int flag)  
{  
  int result;  
  result = Count() + flag ? 1 : 2;  
  // code...  
}  
```  
  
 若要更正此警告，请添加括号，如下面的代码中所示：  
  
```  
int Count();  
  
void f(int flag)  
{  
  int result;  
  result = Count() + (flag ? 1 : 2);  
  // code...  
}  
```
