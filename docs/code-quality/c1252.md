# C1252
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C1252：插件之间存在循环依赖项或缺少依赖项：需要 GUID  
  
 代码分析工具报告此警告，当错误出现在插件内部依赖性时，而不是在分析的代码中。
