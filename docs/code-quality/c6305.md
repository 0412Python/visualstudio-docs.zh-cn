# <a name="c6305"></a>C6305
警告 C6305： 之间 sizeof 和 countof 数量可能不匹配

 此警告意味着变量控股`sizeof`结果正在加上或减去从指针或`countof`表达式。 这会导致意外缩放指针算法。

## <a name="example"></a>示例
 下面的代码将生成此警告：

```
void f(int *p)
{
  int cb=sizeof(int);
  //code...
  p +=cb; // warning 6305
}
```

 若要更正此警告，请使用下面的代码：

```
void f(int *p)
{
  // code...
  p += 1;
}
```
