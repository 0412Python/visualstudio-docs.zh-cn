# C6305
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C6305：sizeof 和 countof 量值之间可能不一致  
  
 此警告意味着正在指针或 `countof` 表达式中添加或删除存放 `sizeof`  结果的变量。  这将导致指针算法的意外缩放。  
  
## 示例  
 下面的代码生成此警告：  
  
```  
void f(int *p)   
{   
  int cb=sizeof(int);   
  //code...   
  p +=cb; // warning 6305  
}  
```  
  
 若要更正此警告，请使用下面的代码：  
  
```  
void f(int *p)   
{  
  // code...  
  p += 1;   
}  
```
