# C28246
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

警告 C28246：函数“\<name\>”的批注\- 参数“\<parameter\>”与参数的类型不匹配  
  
 在创建注释时，\_\_deref 运算符应用于非指针类型。
