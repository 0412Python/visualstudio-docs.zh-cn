# <a name="c28218"></a>C28218
警告 28218:对于函数参数，批注的参数不匹配，在文件中找到

 在当前行和消息中的行上的批注不一致。 如果对批注使用标准宏，则这种情况应该是不可能的；正常使用时不应出现此警告。
