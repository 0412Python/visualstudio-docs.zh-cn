# <a name="developing-workflow-applications-targeting-the-net-30-or-net-35-framework"></a>开发面向 .NET 3.0 或 .NET 3.5 Framework 的工作流应用程序

Visual Studio 2010 提供 Windows 工作流设计器用于创建面向.NET Framework 4 的 Windows Workflow Foundation (WF) 应用程序。 该设计器用于图形构造和调试的工作流应用程序中使用熟悉的 Visual Studio 用户界面。

Visual Studio 2010 还提供对可用于面向.NET Framework 版本 3.5 或 WinFX 旧工作流设计器的访问。 本节包含有关使用此旧设计器的指南。
