# <a name="state-machine-activity-designers"></a>状态机活动设计器
[!INCLUDE[wfd1](../workflow-designer/includes/wfd1_md.md)]包含很多在构造工作流时可以使用的活动设计器。 本节包含系统提供的、用于创建和管理状态机工作流的活动设计器。 以下主题介绍这些设计器以及如何使用它们。  
  
## <a name="in-this-section"></a>本节内容  
 [状态机](../workflow-designer/statemachine-activity-designer.md)  
 包含使用熟悉的状态机范式的状态和模型工作流的集合。  
  
 [状态](../workflow-designer/state-activity-designer.md)  
 表示状态机可具有的状态。  
  
 [FinalState](../workflow-designer/finalstate-activity-designer.md)  
 表示终止状态机工作实例的一个状态。  
  
 [转换](../workflow-designer/transition-activity-designer.md)  
 表示两个状态间的转换。
