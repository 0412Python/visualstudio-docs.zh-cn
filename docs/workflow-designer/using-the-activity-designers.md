# <a name="using-the-activity-designers"></a>使用活动设计器

Windows 工作流设计器包含大量的系统提供构造活动和工作流时可以使用的活动。 本节介绍系统提供的可用活动及其设计器的类别。 有关每个类别内各活动及其相应设计器的主题将介绍这些活动的用途并提供使用指南。
