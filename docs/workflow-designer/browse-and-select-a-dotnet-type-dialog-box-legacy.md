# <a name="browse-and-select-a-net-type-dialog-box-legacy"></a>“浏览并选择 .NET 类型”对话框（旧版）
本主题介绍如何使用**浏览并选择.NET 类型**旧的 Windows 工作流设计器中的对话框。 在需要面向 [!INCLUDE[wfd2](../workflow-designer/includes/wfd2_md.md)] 或 [!INCLUDE[netfx35_long](../workflow-designer/includes/netfx35_long_md.md)] 时，请使用旧 [!INCLUDE[vstecwinfx](../workflow-designer/includes/vstecwinfx_md.md)]。

 在**属性**窗口中，当你选择引用的程序集中，省略号采用.NET Framework 类型的属性**[…]**将出现在该属性的文本框末尾。 单击**[…]**打开**浏览并选择.NET 类型**对话框。 在此对话框中，可以从引用的程序集的一个树视图中选取一个类型。 例如，如果你使用活动设计器中，在**属性**窗口中，单击**基类**省略号**[…]**若要从引用程序集树中选择另一个活动的基类。

 下表描述的用户界面 (UI) 元素**浏览并选择.NET 类型**对话框。

|UI 元素|描述|
|----------------|-----------------|
|**类型名称：**|当前选定的类型的名称。|
|**Type**|左窗格显示引用程序集的树视图。 右窗格显示在左窗格中所选引用程序集的选择的可用类型。|
