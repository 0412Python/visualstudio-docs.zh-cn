# “主题、工作流设计器和选项”对话框（旧版）
本主题介绍如何使用旧 [!INCLUDE[wfd1](../workflow-designer/includes/wfd1_md.md)] 中的**“主题”、“工作流设计器”、“选项”**对话框。在需要面向 [!INCLUDE[netfx35_long](../workflow-designer/includes/netfx35_long_md.md)] 或 [!INCLUDE[vstecwinfx](../workflow-designer/includes/vstecwinfx_md.md)] 时，请使用旧 [!INCLUDE[wfd2](../workflow-designer/includes/wfd2_md.md)]。  
  
 主题定义工作流的背景和前景颜色、样式、图标及其他可视元素。可以保存主题以供其他工作流重复使用。可以使用**“主题、工作流设计器和选项”**对话框选择默认主题和已保存的自定义主题。若要打开此对话框，请在**“工作流”**菜单上单击**“选择自定义主题”**。或者，右击工作流设计图面并单击**“选择自定义主题”**。  
  
 下表描述**“主题、工作流设计器和选项”**对话框的用户界面 \(UI\) 元素。  
  
|UI 元素|说明|  
|-----------|--------|  
|**新建**|单击打开 [“主题配置”对话框（旧版）](../workflow-designer/theme-configuration-dialog-box-legacy.md) 以创建一个新主题。|  
|**修改…**|单击打开 [“主题配置”对话框（旧版）](../workflow-designer/theme-configuration-dialog-box-legacy.md) 以修改选定的主题。|  
|**浏览**|单击以浏览一个使用 .wtm 文件扩展名的工作流主题文件。|  
|**选择主题**|主题列表的主题名称列标题。从此列表中选择一个主题，单击**“修改”**以编辑工作流的选定主题，或单击**“确定”**以使用工作流的选定主题。|  
|**说明**|主题列表的主题说明列标题。|  
|**位置**|主题列表的主题位置列标题。|
