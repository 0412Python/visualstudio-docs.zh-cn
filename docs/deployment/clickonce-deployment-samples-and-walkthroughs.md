# <a name="clickonce-deployment-samples-and-walkthroughs"></a>ClickOnce 部署示例和演练
本部分包含示例应用程序、 示例代码和分步演练介绍了语法、 结构和技术用于部署 Windows 窗体、 WPF 和控制台应用程序。  
  
 示例代码用于教学目的，并且不应在已部署的解决方案，无需修改即可使用。 具体而言，必须将安全性考虑更高版本中不考虑。
