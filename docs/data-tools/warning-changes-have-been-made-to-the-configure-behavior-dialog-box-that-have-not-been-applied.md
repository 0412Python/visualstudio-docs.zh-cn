# <a name="warning-changes-have-been-made-to-the-configure-behavior-dialog-box-that-have-not-been-applied"></a>警告： 进行了更改到尚未应用的配置行为对话框

警告。 尚未应用对“配置行为”对话框所做的更改。 是否要应用您所做的更改?

**配置行为**对话框中，您可以配置为所有可用类的插入、 更新和删除行为。 当你选择一个新时，会出现此消息**类**和**行为**组合和以前的更改具有尚未应用。

## <a name="to-apply-the-change-and-continue"></a>应用更改并继续

- 单击**是**将更改应用到所选**类**和**行为**。

## <a name="to-cancel-the-previous-change-and-continue"></a>取消以前的更改并继续

- 单击**否**取消对所选更改**类**和**行为**。
