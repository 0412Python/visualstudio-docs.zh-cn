# 警告。 尚未应用对“配置行为”对话框所做的更改
警告。尚未应用对“配置行为”对话框所做的更改。是否要应用您所做的更改?  
  
 使用**“配置行为”**对话框，可以为所有可用类配置插入、更新和删除行为。当选择**“类”**和**“行为”**的新组合并且尚未应用以前的更改时，将出现此消息。  
  
### 应用更改并继续  
  
-   单击**“是”**应用对所选**“类”**和**“行为”**的更改。  
  
### 取消以前的更改并继续  
  
-   单击**“否”**取消对所选**“类”**和**“行为”**的更改。
