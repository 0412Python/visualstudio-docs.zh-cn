# <a name="the-connection-property-in-the-application-settings-file-is-missing-or-incorrect"></a>应用程序设置文件中的连接属性缺失或不正确

应用程序设置文件中缺少连接属性或连接属性不正确。 中的连接字符串 *.dbml*文件已使用在其原位置。

*.Dbml*文件包含对连接字符串中找不到应用程序设置文件的引用。 此消息仅供参考;连接字符串设置将创建何时**确定**单击。

若要对此消息做出响应，请选择**确定**。 中包含的连接信息 *.dbml*文件添加到应用程序设置。
