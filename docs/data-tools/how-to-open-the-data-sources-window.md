# 如何：打开“数据源”窗口
**“数据源”**窗口显示项目中的数据源。  使用**“数据源”**窗口创建数据绑定控件的用户界面，方法是将各个项从该窗口拖动到项目中的窗体上。  请参见[在 Visual Studio 中将 Windows 窗体控件绑定到数据](../data-tools/bind-windows-forms-controls-to-data-in-visual-studio.md)。  
  
 可通过运行**“数据源配置”**向导来填充**“数据源”**窗口。  有关更多信息，请参见[数据源配置向导](../data-tools/media/data-source-configuration-wizard.png)。  
  
### 打开数据源窗口  
  
-   在菜单栏上，依次选择**“视图”**、**“其他窗口”**、**“数据源”**（或选择 Shift\+Alt\+D 键）。  
  
    > [!IMPORTANT]
    >  只有当项目支持创建和使用数据源时，你才可以向该项目添加数据源。  例如，你不能在 Windows 应用商店应用的一个项目中打开**“数据源”**窗口。
