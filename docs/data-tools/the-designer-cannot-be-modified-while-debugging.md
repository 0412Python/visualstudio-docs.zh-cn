# <a name="the-designer-cannot-be-modified-while-debugging"></a>调试时无法修改设计器

当尝试在修改项时，会出现此消息**O/R 设计器**应用程序在调试模式下的运行时。 在调试模式下运行应用程序时**O/R 设计器**是只读的。

若要更正此错误，请选择**停止调试**上**调试**菜单。 应用程序将停止调试，并可以修改中的项**O/R 设计器**。
