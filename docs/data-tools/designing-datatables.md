# 设计数据表
[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] 提供用于创建和编辑数据表 \(**DataTable**\) 的设计时工具。  有关 **DataTable** 的概述，请参见 [DataTable](../Topic/DataTables.md)。  
  
 下列主题介绍了如何创建数据表、使用**“数据集设计器”**将数据表添加到类型化数据集以及创建和编辑定义数据表的数据列 \(**DataColumn**\)。  
  
> [!NOTE]
>  显示的对话框和菜单命令可能会与“帮助”中的描述不同，具体取决于您现用的设置或版本。  若要更改设置，请在**“工具”**菜单上选择**“导入和导出设置”**。  有关更多信息，请参见 [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/zh-cn/22c4debb-4e31-47a8-8f19-16f328d7dcd3)。  
  
## 本节内容  
 [如何：创建数据表](../data-tools/how-to-create-data-tables.md)  
 提供使用**“数据集设计器”**创建新的 <xref:System.Data.DataTable> 的步骤。  
  
 [如何：向数据表添加列](../Topic/How%20to:%20Add%20Columns%20to%20a%20DataTable.md)  
 提供在现有 <xref:System.Data.DataTable> 中创建新的 <xref:System.Data.DataColumn> 的步骤。  
  
 [演练：在数据集设计器中创建数据表](../data-tools/walkthrough-creating-a-datatable-in-the-dataset-designer.md)  
 提供分步说明，介绍如何创建 <xref:System.Data.DataTable> 以及如何定义构成其结构的 <xref:System.Data.DataColumn>。  
  
## 参考  
 <xref:System.Data.DataSet>  
 表示数据的内存中缓存。  
  
 <xref:System.Data.DataTable>  
 表示一个内存中数据表。  
  
 <xref:System.Data.DataColumn>  
 表示 <xref:System.Data.DataTable> 中的列的架构。  
  
 <xref:System.Data.DataRelation>  
 表示两个 <xref:System.Data.DataTable> 对象之间的父\/子关系。
