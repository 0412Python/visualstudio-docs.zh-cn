# <a name="could-not-retrieve-schema-information-for-database-object-object-name"></a>无法检索数据库对象的架构信息\<对象名称 >
在对象时，通常会出现此消息**服务器资源管理器/数据库资源管理器**复制到剪贴板，从数据库中删除然后再粘贴到设计器上。 因为该数据库对象不再存在，所以显示此消息。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   单击**确定**以关闭对话框。
