# XElement 类动态属性
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

本节介绍 <xref:System.Xml.Linq.XElemen?displayProperty=fullName>[t](assetId:///t?qualifyHint=False&autoUpgrade=True) 类的动态属性。  
  
## 本节内容  
  
|主题|说明|  
|--------|--------|  
|[特性](../designers/attribute-xelement-dynamic-property.md)|获取一个索引器，用于检索与指定扩展名对应的属性。|  
|[元素](../designers/element-xelement-dynamic-property.md)|获取一个索引器，用于检索与指定扩展名对应的子元素。|  
|[Elements](../designers/elements-xelement-dynamic-property.md)|获取一个索引器，用于检索与指定扩展名匹配的当前元素的子元素。|  
|[Descendants](../designers/descendants-xelement-dynamic-property.md)|获取一个索引器，用于检索与指定扩展名匹配的当前元素的所有子代元素。|  
|[值](../designers/value-xelement-dynamic-property.md)|获取或设置一个元素的内容。|  
|[Xml](../designers/xml-xelement-dynamic-property.md)|获取元素的未格式化的 XML 表示形式。|
