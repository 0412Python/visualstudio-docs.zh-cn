# <a name="create-modern-desktop-applications-with-windows-presentation-foundation"></a>使用 Windows Presentation Foundation 创建现代桌面应用程序

Visual Studio 中的 Windows Presentation Foundation (WPF) 向开发人员提供了一个统一的编程模型，可用于在 Windows 上生成现代业务线桌面应用程序。  

若要进行深入演练，请从 [WPF 入门](../designers/getting-started-with-wpf.md)开始，同时学习和完成其次要主题。

有关可用工具的比较，请参阅[在 Visual Studio 和 Blend for Visual Studio 中设计 XAML](../designers/designing-xaml-in-visual-studio.md)。
