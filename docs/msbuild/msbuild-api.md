# <a name="use-the-msbuild-api"></a>使用 MSBuild API

MSBuild 提供公共 API 外围，因此程序可以执行生成操作和检查项目。

可在 [Microsoft.Build 命名空间](https://msdn.microsoft.com/library/mt476050(v=vs.110).aspx)上找到 MSBuild API 相关文档。
