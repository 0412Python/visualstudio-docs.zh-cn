# <a name="endtrackingcontext"></a>EndTrackingContext
结束当前跟踪上下文。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT WINAPI EndTrackingContext();  
```  
  
## <a name="return-value"></a>返回值  
 如果跟踪上下文结束，返回带 SUCCEEDED 位组的 HRESULT。  
  
## <a name="requirements"></a>要求  
 标头：FileTracker.h
