# <a name="adding-extensions-to-dsl-definitions"></a>向 DSL 定义中添加扩展
DSL 定义扩展允许您创建域特定语言 (DSL) 扩展的包。 DSL 扩展，它包含在 Visual Studio 集成扩展 (VSIX) 中，可以安装在用户的计算机上，DSL 的方式相同。 可以动态启用和禁用在运行时中的其他功能。 Dsl 便无需专门为扩展，并且扩展可以被设计为更高版本或者由第三方而无需更改扩展的 DSL。  
  
 其他功能可以包括以下︰  
  
-   模型和演示文稿的元素的属性  
  
-   形状和连接线的修饰器  
  
-   类、 关系、 形状和连接线  
  
-   验证约束  
  
-   工具箱项和选项卡  
  
 扩展 DSL 的用户可以创建并保存包含其他功能的实例的模型和这些可以读取由其他用户的已安装相应的扩展名。 未安装该扩展的用户不能使用这些附加功能，但它们可以更新并将模型保存而不会丢失的其他功能。
