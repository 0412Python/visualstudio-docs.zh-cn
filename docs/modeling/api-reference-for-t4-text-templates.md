# T4 文本模板的 API 参考
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

文本模板 API 允许您调用和自定义 [text templates（文本模板）](../modeling/code-generation-and-t4-text-templates.md) 的转换。  
  
## 命名空间  
  
|命名空间|用途|  
|----------|--------|  
|<xref:Microsoft.VisualStudio.TextTemplating>|包含用于文本模板转换功能的类。  文本模板转换引擎集成在 Visual Studio 中，并可将文本模板文件转换为生成的文本输出文件。|  
|<xref:Microsoft.VisualStudio.TextTemplating.Modeling>|提供与 UML 模型和域特定语言相关的文本转换功能，例如访问 [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] ModelBus。|  
|<xref:Microsoft.VisualStudio.TextTemplating.VSHost>|提供对 [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] 中的文本模板化服务的访问。|
