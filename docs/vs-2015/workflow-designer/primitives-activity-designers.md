# <a name="primitives-activity-designers"></a>基元活动设计器
各基元活动设计器用于创建和配置 <xref:System.Activities> 命名空间中的多种活动，如 Assign 和 Delay。 本节中包含的主题介绍如何使用 [!INCLUDE[wfd2](../includes/wfd2-md.md)] 活动和模板设计器。  
  
## <a name="in-this-section"></a>本节内容  
 [Assign](../workflow-designer/assign-activity-designer.md)  
 用于创建和配置 <xref:System.Activities.Statements.Assign> 活动。  
  
 [Delay](../workflow-designer/delay-activity-designer.md)  
 用于创建和配置 <xref:System.Activities.Statements.Delay> 活动。  
  
 [InvokeMethod](../workflow-designer/invokemethod-activity-designer.md)  
 用于创建和配置 <xref:System.Activities.Statements.InvokeMethod> 活动。  
  
 [WriteLine](../workflow-designer/writeline-activity-designer.md)  
 用于创建和配置 <xref:System.Activities.Statements.WriteLine> 活动。
