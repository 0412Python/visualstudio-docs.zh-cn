# <a name="concurrency-namespace-concurrency-visualizer"></a>Concurrency 命名空间（并发可视化工具）
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主题的最新版本，请参阅[并发 Namespace （并发可视化工具）](https://docs.microsoft.com/visualstudio/profiling/concurrency-namespace-concurrency-visualizer)。  
  
`Concurrency` 命名空间提供可让你访问 C++ 的并发运行和并发编程框架的类和函数。 有关详细信息，请参阅[并发运行时](http://msdn.microsoft.com/library/874bc58f-8dce-483e-a3a1-4dcc9e52ed2c)。  
  
## <a name="syntax"></a>语法  
  
```  
namespace Concurrency;  
```  
  
## <a name="members"></a>成员  
  
### <a name="namespaces"></a>命名空间  
  
|名称|描述|  
|----------|-----------------|  
|[diagnostic 命名空间](../profiling/diagnostic-namespace.md)|`diagnostics` 命名空间提供用于发出并行可视化工具标记的功能。|  
  
## <a name="requirements"></a>要求  
 **标头：** cvmarkersobj.h
