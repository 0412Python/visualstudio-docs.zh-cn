# <a name="vspackage-essentials"></a>VSPackage 要点
VSPackage 是软件模块，它们通过 UI 元素、服务、项目、编辑器和设计器来组成并扩展 Visual Studio。 Vspackage 是的中央体系结构单元[!INCLUDE[vsprvs](../includes/vsprvs-md.md)]。 [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] 本身多半以 Vspackage 集合的形式进行编写。  
  
 若要创建[!INCLUDE[vsprvs](../includes/vsprvs-md.md)]扩展应用程序，首先创建 VSPackage。 然后可以添加菜单命令、 工具窗口、 文档窗口或其他代码来扩展[!INCLUDE[vsprvs](../includes/vsprvs-md.md)]IDE。  
  
 要使用 VSPackage，需要安装 Visual Studio SDK。 要按照本演练的步骤操作，必须安装 Visual Studio SDK。 有关详细信息，请参阅[Visual Studio SDK](../extensibility/visual-studio-sdk.md)。
