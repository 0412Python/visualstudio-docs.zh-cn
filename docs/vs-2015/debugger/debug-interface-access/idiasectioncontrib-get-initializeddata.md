# <a name="idiasectioncontribgetinitializeddata"></a>IDiaSectionContrib::get_initializedData
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

本主题的最新版本，请参阅[idiasectioncontrib:: Get_initializeddata](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasectioncontrib-get-initializeddata)。  
  
检索一个标志，指示的部分是否包含初始化的数据。  
  
## <a name="syntax"></a>语法  
  
```cpp#  
HRESULT get_initializedData (   
   BOOL* pRetVal  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pRetVal`  
 [out]返回`TRUE`的部分包含初始化的数据; 否则，返回`FALSE`。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`。 返回`S_FALSE`如果此属性不受支持。 否则，返回错误代码。
