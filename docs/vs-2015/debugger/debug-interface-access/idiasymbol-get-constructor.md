# <a name="idiasymbolgetconstructor"></a>IDiaSymbol::get_constructor
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

本主题的最新版本，请参阅[idiasymbol:: Get_constructor](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasymbol-get-constructor)。  
  
检索指定用户定义数据类型是否具有构造函数或析构函数的标志。  
  
## <a name="syntax"></a>语法  
  
```cpp#  
HRESULT get_constructor (   
   BOOL* pRetVal  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pRetVal`  
 [out]返回`TRUE`如果用户定义数据类型的构造函数或析构函数; 否则，返回`FALSE`。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`; 否则为返回`S_FALSE`或错误代码。  
  
> [!NOTE]
>  返回值为`S_FALSE`表示该属性不是可用于符号。
