# <a name="how-to-debug-com-servers"></a>如何：调试 COM 服务器
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主题的最新版本，请参阅[如何： 调试 COM 服务器](https://docs.microsoft.com/visualstudio/debugger/how-to-debug-com-servers)。  
  
调试 COM 服务器应用程序会引起一系列不总是容易解决的独特问题。  
  
 如果不必或不想为容器应用程序使用调试信息，那么开始调试服务器应用程序可分三步进行。  
  
### <a name="to-debug-a-server-application-without-container-information"></a>在没有容器信息的情况下调试服务器应用程序  
  
1.  像对待普通的应用程序一样开始调试服务器。  
  
2.  按需要设置断点。  
  
3.  启动容器应用程序。
