# <a name="how-to-change-fonts"></a>如何：更改字体
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

您可以更改字体、 字体样式和 Spy + + 窗口的字体大小。  
  
### <a name="to-change-font-options"></a>若要更改字体选项  
  
1. 从**视图**菜单中，选择**字体**。  
  
2. 选择字体、 字体样式和字体大小。  
  
3. 单击 **“确定”**。  
  
   选择**保存为默认字体**将导致所有将来 Spy + + windows 以使用此字体。
