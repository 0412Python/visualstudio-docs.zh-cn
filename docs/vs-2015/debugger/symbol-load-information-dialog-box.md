# <a name="symbol-load-information-dialog-box"></a>“符号加载信息”对话框
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主题的最新版本，请参阅[符号加载信息对话框](https://docs.microsoft.com/visualstudio/debugger/symbol-load-information-dialog-box)。  
  
**符号加载信息**右键单击打开对话框**模块**窗口，然后选择**符号加载信息**。  
  
## <a name="uielement-list"></a>UIElement 列表  
 **信息**  
 显示搜索符号文件 （PDB 文件） 基于 Visual Studio 安装设置和对中的符号设置所做的更改目录**选项**对话框。  
  
 **符号设置**  
 单击此按钮可更改中的符号设置**选项**对话框的 (**调试**类别中，**符号**页)。 在该页上，你可以添加或移除符号位置、指定本地缓存位置（如果使用的是符号服务器）并控制搜索符号位置的时间。
