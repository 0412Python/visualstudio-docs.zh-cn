# <a name="help-menu-commands"></a>“帮助”菜单命令
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主题的最新版本，请参阅[帮助菜单命令](https://docs.microsoft.com/visualstudio/debugger/help-menu-commands)。  
  
**帮助**菜单提供命令来打开帮助文件。  
  
 **索引**  
 将内容屏幕打开 Spy + + 帮助的。  
  
 **使用的帮助**  
 打开介绍如何使用帮助的帮助文件。  
  
 **关于 Spy + +**  
 显示 Spy + + 中的软件版本和版权信息。
