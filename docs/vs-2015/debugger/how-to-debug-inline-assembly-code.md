# <a name="how-to-debug-inline-assembly-code"></a>如何：调试内联程序集代码
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

请注意]
>  显示的对话框和菜单命令可能会与“帮助”中的描述不同，具体取决于你的当前设置或版本。 若要更改设置，请在“工具”菜单上选择“导入和导出设置”。 有关详细信息，请参阅 [在 Visual Studio 中自定义开发设置](http://msdn.microsoft.com/en-us/22c4debb-4e31-47a8-8f19-16f328d7dcd3)。  
  
 调试器为调试内联程序集代码中，提供两个窗口**反汇编**窗口和**注册**窗口。  
  
## <a name="procedure"></a>过程  
  
#### <a name="to-debug-inline-assembly-code"></a>调试内联程序集代码  
  
1.  使用**反汇编**窗口查看程序集指令。  
  
2.  使用**注册**窗口查看寄存器内容。
