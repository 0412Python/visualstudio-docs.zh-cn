# <a name="what39s-new-in-the-xslt-debugger"></a>什么&#39;s XSLT 调试程序中的新增功能
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Visual Studio 2010 中新增了以下功能。  
  
## <a name="xslt-profiler"></a>XSLT 探查器  
 XSLT 探查器是一种工具，开发人员可用来创建详细的 XSLT 性能报告，从而对 XSLT 代码中与性能相关的问题进行衡量、评估并将这些问题作为目标。 有关详细信息，请参阅[XSLT Profiler](../xml-tools/xslt-profiler.md)。
