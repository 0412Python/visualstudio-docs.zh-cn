# <a name="mparent-field"></a>m_parent 字段
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

本主题的最新版本，请参阅[m_parent 字段](https://docs.microsoft.com/visualstudio/extensibility/debugger/m-parent-field)。  
  
有关支持字段<xref:System.Threading.Tasks.Task?displayProperty=fullName>父属性。  
  
 **Namespace**：<xref:System.Threading.Tasks?displayProperty=fullName>  
  
 **程序集：** mscorlib （在 mscorlib.dll 中)  
  
 无法从.NET Framework 来访问此内部成员，因为以下语法提供通用中间语言 (CIL)。  
  
## <a name="syntax"></a>语法  
  
```  
.field private initonly class System.Threading.Tasks.Task m_parent  
```
