# <a name="idebugprogram2detach"></a>IDebugProgram2::Detach
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

从该程序的调试引擎中分离。  
  
## <a name="syntax"></a>语法  
  
```cpp#  
HRESULT Detach(   
   void   
);  
```  
  
```csharp  
int Detach();  
```  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`; 否则为返回错误代码。  
  
## <a name="remarks"></a>备注  
 分离的程序继续运行，但它不再是调试会话的一部分。 没有更多程序调试事件发送后分离调试引擎。
