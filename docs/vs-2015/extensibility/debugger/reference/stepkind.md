# <a name="stepkind"></a>STEPKIND
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

指定用于单步执行步骤种类。  
  
## <a name="syntax"></a>语法  
  
```cpp#  
enum enum_STEPKIND {   
   STEP_INTO      = 0,  
   STEP_OVER      = 1,  
   STEP_OUT       = 2,  
   STEP_BACKWARDS = 3  
};  
typedef DWORD STEPKIND;  
```  
  
```csharp  
public enum enum_STEPKIND {   
   STEP_INTO      = 0,  
   STEP_OVER      = 1,  
   STEP_OUT       = 2,  
   STEP_BACKWARDS = 3  
};  
```  
  
## <a name="members"></a>成员  
 STEP_INTO  
 单步执行函数。  
  
 跨度  
 过程执行函数的步骤。  
  
 STEP_OUT  
 跳出函数。  
  
 STEP_BACKWARDS  
 执行某一函数向后的步骤。  
  
## <a name="remarks"></a>备注  
 作为参数传递[步骤](../../../extensibility/debugger/reference/idebugprocess3-step.md)方法。
