# <a name="idebugreference2setreferencetype"></a>IDebugReference2::SetReferenceType
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

本主题的最新版本，请参阅[IDebugReference2::SetReferenceType](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugreference2-setreferencetype)。  
  
设置引用类型。 留待将来使用。  
  
## <a name="syntax"></a>语法  
  
```cpp#  
HRESULT SetReferenceType (   
   REFERENCE_TYPE dwRefType  
);  
```  
  
```csharp  
int SetReferenceType (   
   enum_REFERENCE_TYPE dwRefType  
);  
```  
  
#### <a name="parameters"></a>参数  
 `dwRefType`  
 [in]中的值[REFERENCE_TYPE](../../../extensibility/debugger/reference/reference-type.md)枚举，用于指定引用类型。  
  
## <a name="return-value"></a>返回值  
 始终返回 `E_NOTIMPL`。
