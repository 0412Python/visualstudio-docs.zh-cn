# <a name="asyncvoidmethodbuildermobjectidfordebugger-field"></a>AsyncVoidMethodBuilder.m_objectIdForDebugger 字段
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

本主题的最新版本，请参阅[AsyncVoidMethodBuilder.m_objectIdForDebugger 字段](https://docs.microsoft.com/visualstudio/extensibility/debugger/asyncvoidmethodbuilder-m-objectidfordebugger-field)。  
  
表示调试器用于唯一标识此生成器的延迟初始化的对象。  
  
 **命名空间：** <xref:System.Runtime.CompilerServices?displayProperty=fullName>  
  
 **程序集：** mscorlib （在 mscorlib.dll 中)  
  
 无法从.NET Framework 来访问此内部成员，因为以下语法提供通用中间语言 (CIL)。  
  
## <a name="syntax"></a>语法  
  
```  
.field private object m_objectIdForDebugger  
```
