# <a name="c28284"></a>C28284
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28284： 对于符号，当前不支持谓词非函数符号  
  
 此警告意味着发现与函数之外的对象有关的条件批注（谓词，`__drv_when`）。
