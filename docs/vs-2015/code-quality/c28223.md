[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主题的最新版本，请参阅[C28223](https://docs.microsoft.com/visualstudio/code-quality/c28223)。  
  
警告 C28223： 找不到预期的标记/标识符对于批注，参数  
  
 此警告意味着批注参数应为标识符，并且遇到了一些其他符号。 这通常表示存在错误编码的批注宏。 在此位置中的 C 或 c + + 关键字的使用将导致此错误。
