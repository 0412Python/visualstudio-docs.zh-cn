# <a name="c28285"></a>C28285
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28285： 对于函数，参数中存在语法错误  
  
 当在模型文件中遇到一个可能的错误时，代码分析工具将报告此警告。 一些源文件错误也会造成此类错误。
