# <a name="c28196"></a>C28196
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主题的最新版本，请参阅[C28196](https://docs.microsoft.com/visualstudio/code-quality/c28196)。  
  
警告 C28196： 未满足的要求。 （表达式不的计算结果为 true。）  
  
 此警告意味着所分析函数的函数原型有基于 `__notnull` 参数或返回值的 `__null`、`__drv_valueIs` 或 `_Out_`，但返回值与该批注不一致。
