# <a name="c28217"></a>C28217
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28217： 对于函数，批注的参数数目不会不与在文件中找到的  
  
 在当前行和消息中的行上的批注不一致。 如果对批注使用标准宏，则这种情况应该是不可能的；正常使用时不应出现此警告。
