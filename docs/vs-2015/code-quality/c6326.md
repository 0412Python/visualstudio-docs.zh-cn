# <a name="c6326"></a>C6326
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主题的最新版本，请参阅[C6326](https://docs.microsoft.com/visualstudio/code-quality/c6326)。  
  
警告 C6326： 可能的常量进行比较  
  
 此警告表明潜在的另一个常量，它是冗余的代码与常量比较。 您必须检查以确保你的意图正确捕获在代码中。 在某些情况下，可以简化测试条件来实现相同的结果。  
  
## <a name="example"></a>示例  
 下面的代码生成此警告，因为两个常量进行比较：  
  
```  
#define LEVEL    
const int STD_LEVEL = 5;  
  
const int value =   
#ifdef LEVEL  
10;  
#else   
5;  
#endif  
  
void f()  
{  
  if( value > STD_LEVEL)  
  {  
    // code...  
  }  
  else  
  {  
    // code...  
  }  
}  
```  
  
 下面的代码演示使用 #ifdef 语句来确定应执行的代码更正此警告的一种方法：  
  
```  
#define LEVEL    
const int STD_LEVEL = 5;  
  
const int value =   
#ifdef LEVEL  
10;  
#else   
5;  
#endif  
  
void f ()  
{  
#ifdef LEVEL  
  {  
    // code...  
  }  
#else  
  {  
    // code...  
  }  
#endif  
}   
```
