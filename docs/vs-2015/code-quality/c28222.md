# <a name="c28222"></a>C28222
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主题的最新版本，请参阅[C28222](https://docs.microsoft.com/visualstudio/code-quality/c28222)。  
  
警告 28222: （_y)\_，（_n)\_，或 _Maybe\_批注所需  
  
 此警告意味着批注参数应是符号 `_Yes_`、`_No_` 或 `_Maybe_` 中的一种，但遇到了其他符号。 这通常表示存在错误编码的批注宏。
