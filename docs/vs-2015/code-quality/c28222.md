# <a name="c28222"></a>C28222
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 28222:\_是\_，\_否\_，或\_也许\_批注所需  
  
 此警告意味着批注参数应是符号 `_Yes_`、`_No_` 或 `_Maybe_` 中的一种，但遇到了其他符号。 这通常表示存在错误编码的批注宏。
