# <a name="c28279"></a>C28279
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28279： 对于符号，而无需匹配的 end 已找到 begin  
  
 批注语言支持 begin 和 end（C 中的 `{` 和 `}`）构造，并且此配对不对称。 如果使用了宏，则此情况可避免。
