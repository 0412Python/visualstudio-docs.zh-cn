# <a name="c28226"></a>C28226
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主题的最新版本，请参阅[C28226](https://docs.microsoft.com/visualstudio/code-quality/c28226)。  
  
警告 C28226：批注也不能为 PrimOp (在当前声明中)  
  
 此警告意味着所命名的批注即将被声明为 PrimOp，并且先前已声明为正常批注。 如果对批注使用标准宏，则这种情况应该是不可能的；正常使用时不应出现此警告。
