# <a name="c28241"></a>C28241
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28241： 无法识别参数上函数的批注  
  
 使用无法识别的批注名称。 如果正对批注使用标准宏，则这应不是可能实现的；此警告在正常使用时应不显示。
