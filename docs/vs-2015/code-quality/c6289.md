# <a name="c6289"></a>C6289
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主题的最新版本，请参阅[C6289](https://docs.microsoft.com/visualstudio/code-quality/c6289)。  
  
警告 C6289： 运算符不正确： 基于&#124;&#124;始终是一个非零常量。 是否原本要使用 & &？  
  
 此警告表明测试表达式中一个变量所测试的针对两个不同的常量和结果取决于哪个条件为 true。 这始终为 true。  
  
 此问题通常由使用`||`来代替`&&`，但也可以通过使用导致`!=`其中`==`的初衷。  
  
## <a name="example"></a>示例  
 下面的代码生成此警告：  
  
```  
void f(int x)  
{  
  if ((x != 1) || (x != 3))  
  {  
    // code   
  }  
}  
```  
  
 若要更正此警告，请使用以下代码：  
  
```  
void f(int x)  
{  
  if ((x != 1) && (x != 3))  
  {  
    // code   
  }  
}  
  
/* or */  
void f(int x)  
{  
  if ((x == 1) || (x == 3))  
  {  
    // code   
  }  
}  
```
