# <a name="c28213"></a>C28213
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主题的最新版本，请参阅[C28213](https://docs.microsoft.com/visualstudio/code-quality/c28213)。  
  
警告 C28213:`_Use_decl_annotations_`批注必须用于引用，无需修改，之前的声明。 \<原因 >  
  
 当引用之前的声明，但没有必要的 `_Use_decl_annotations_` 批注时，将报告此警告。
