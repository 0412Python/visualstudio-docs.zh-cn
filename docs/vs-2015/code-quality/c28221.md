# <a name="c28221"></a>C28221
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主题的最新版本，请参阅[C28221](https://docs.microsoft.com/visualstudio/code-quality/c28221)。  
  
警告 C28221： 批注中的参数应为表达式的字符串  
  
 此警告表明批注参数应为字符串，但遇到了其他类型。 这通常表示存在错误编码的批注宏，此警告在正常使用时不应出现。
