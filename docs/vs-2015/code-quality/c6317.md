# <a name="c6317"></a>C6317
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6317： 运算符不正确： 逻辑非 （！） 不可互换使用 1 的补数 （~）  
  
 此警告意味着逻辑非 (`!`) 要应用到一个常量，它可能是一个位标志。 逻辑非的结果是布尔值;不应将应用按位的和 (`&`) 为布尔值的运算符。 使用 1 的补数 (`~`) 运算符来翻转标志。  
  
## <a name="example"></a>示例  
 下面的代码生成此警告：  
  
```  
#define FLAGS   0x4004  
  
void f(int i)  
{  
  if (i & !FLAGS) // warning  
  {   
    // code  
  }  
}  
```  
  
 若要更正此警告，请使用以下代码：  
  
```  
#define FLAGS   0x4004  
  
void f(int i)  
{  
  if (i & ~FLAGS)  
  {  
    // code  
  }  
}  
```
