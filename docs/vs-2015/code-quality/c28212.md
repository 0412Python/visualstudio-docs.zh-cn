# <a name="c28212"></a>C28212
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主题的最新版本，请参阅[C28212](https://docs.microsoft.com/visualstudio/code-quality/c28212)。  
  
警告 C28212： 批注需要指针表达式  
  
 此警告意味着所编号的批注参数（不是要批注的函数）应该是指针或数组类型，但却遇到某些其他类型。 此批注需要更正。
