本主题的最新版本，请参阅[C28283](https://docs.microsoft.com/visualstudio/code-quality/c28283)。  
  
警告 C28283： 对于符号，指定的大小规范尚不支持  
  
 此警告表明大小规范“sentinel”批注接收了非零值。 实际上，调用方尝试说明字符串被二进制零以外的字符终止。
