# <a name="c28243"></a>C28243
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主题的最新版本，请参阅[C28243](https://docs.microsoft.com/visualstudio/code-quality/c28243)。  
  
警告 C28243： 批注需要参数上的函数取消引用不是实际类型所允许  
  
 批注 `__deref` 运算符的数量多于由参数类型定义的指针级别数。 通过改变批注中的数量或所引用参数的指针级别，可更正此警告。
