# <a name="c28215"></a>C28215
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28215: typefix 无法应用于已具有 typefix 的参数  
  
 将 `__typefix` 批注应用到已有该批注的参数是一个错误。 `__typefix` 批注只在少数特殊情况下使用，此警告在正常使用时不应出现。
