# <a name="c28291"></a>C28291
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28291： 发送 null/notnull 在 deref 级别 0 是对于函数无意义\<x > 参数为\<数 >  
  
 当取消引用级别零参数的后置条件指定为具有 null/非 null 属性时，代码分析工具将报告此警告。 之所以发生此错误，是因为取消引用级别零下的值不能更改。
