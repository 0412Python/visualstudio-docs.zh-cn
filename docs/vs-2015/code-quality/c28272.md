# <a name="c28272"></a>C28272
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主题的最新版本，请参阅[C28272](https://docs.microsoft.com/visualstudio/code-quality/c28272)。  
  
警告 C28272： 对于函数，在检查参数时批注与函数声明不一致  
  
 此警告表明批注而不是所分析的代码中存在错误。 显示在函数定义上的批注与显示在声明上的批注不一致。 这两个批注在解析后应匹配。
