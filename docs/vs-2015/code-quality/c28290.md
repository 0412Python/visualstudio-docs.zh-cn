# <a name="c28290"></a>C28290
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C28290： 函数的批注包含多个外部对象数量多于实际的参数数量  
  
 当函数批注包含的外部项超过实际的参数数量时，代码分析工具将报告此警告。
