# <a name="xml-documentation-comments-javascript"></a>XML 文档注释 (JavaScript)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

在 JavaScript 中，可以通过在源代码内的特殊注释字段中添加 XML 标记来为代码创建文档。 IntelliSense 使用这些 XML 标记提供有关 JavaScript 对象的信息。  
  
 代码注释的位置因具体标记而异。  
  
 支持以下标记：  
  
-   [\<不推荐使用 >](../ide/deprecated-javascript.md)  
  
-   [\<字段 >](../ide/field-javascript.md)  
  
-   [\<loc >](../ide/loc-javascript.md)  
  
-   [\<param >](../ide/param-javascript.md)  
  
-   [\<returns>](../ide/returns-javascript.md)  
  
-   [\<签名 >](../ide/signature-javascript.md)  
  
-   [\<summary>](../ide/summary-javascript.md)  
  
-   [\<value>](../ide/value-javascript.md)  
  
-   [\<var >](../ide/var-javascript.md)
