# <a name="locate-information"></a>找到信息
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Help Viewer 提供几种不同方式供你在帮助中找到信息。 根据所查找的信息类型，某些策略可能会比其他策略更加有效。  
  
|有关信息|使用|  
|----------------------------|---------|  
|用户界面中的一个元素|F1 键|  
|目录中与当前主题同一区域的其他主题|工具栏上的“在目录中显示主题”按钮|  
|功能或技术|索引|  
|语言元素，如方法和事件|搜索|
