|主题|描述|  
|-----------|-----------------|

|[如何：执行异步数据服务查询](http://msdn.microsoft.com/library/902a2dc1-d0e9-4b00-90a8-becc4cb1f6a7)|演示如何以异步方式保存通过数据服务所做的更改。|
