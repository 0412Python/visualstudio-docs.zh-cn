# <a name="cannot-create-an-association-ltassociation-namegt---property-types-do-not-match"></a>无法创建关联&lt;关联名称&gt;-属性类型不匹配
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

本主题的最新版本，请参阅[无法创建关联&lt;关联名称&gt;-属性类型不匹配](https://docs.microsoft.com/visualstudio/data-tools/cannot-create-an-association-association-name-property-types-do-not-match)。  
  
  
无法创建关联\<关联名称 >-属性类型不匹配。 属性不具有匹配的类型：\<属性名称 >。  
  
 关联定义由所选**关联属性**中**关联编辑器**对话框。 关联双方的属性必须具有相同的数据类型。  
  
 消息中列出的属性不具有相同的数据类型。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
1.  检查消息并记下消息中报告的属性。  
  
2.  单击**确定**以关闭对话框。  
  
3.  检查**关联属性**并选择相同的数据类型的属性。  
  
4.  单击 **“确定”**。
