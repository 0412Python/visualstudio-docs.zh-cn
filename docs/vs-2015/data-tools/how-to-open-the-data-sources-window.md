# <a name="how-to-open-the-data-sources-window"></a>如何：打开“数据源”窗口
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

**数据源**窗口显示在项目中的数据源。 您使用**数据源**窗口通过将项从该窗口拖动你的项目中的窗体上创建数据绑定控件的用户界面。 请参阅[绑定 Windows 窗体控件添加到 Visual Studio 中的数据](../data-tools/bind-windows-forms-controls-to-data-in-visual-studio.md)。  
  
 您可以填充**数据源**窗口中的通过运行**数据源配置**向导。 有关详细信息请参阅[数据源配置向导](http://msdn.microsoft.com/library/c4df7de5-5da0-4064-940c-761dd6d9e28f)。  
  
### <a name="to-open-the-data-sources-window"></a>打开数据源窗口  
  
-   在菜单栏上依次选择**视图**，**其他 Windows**，**数据源**（或按 Shift + Alt + D 键）。  
  
    > [!IMPORTANT]
    >  只有当项目支持创建和使用数据源时，你才可以向该项目添加数据源。 例如，不能打开**数据源**为 Windows 应用商店应用项目中的窗口。
