> [!WARNING]
> 启用现有的角色的诊断时，部署包时已禁用已设置任何扩展。 这些方法包括：
>
> * Microsoft 监视代理诊断
> * Microsoft Azure 安全监视
> * Microsoft 反恶意软件                 
> * Microsoft Monitoring Agent
> * Microsoft 服务 Profiler 代理      
> * Windows Azure 域扩展        
> * Windows Azure 诊断扩展   
> * Windows Azure 的远程桌面扩展
> * Windows Azure 日志收集器
>
> 在部署更新的角色后，可以重置你的扩展通过 Azure 门户或 PowerShell。
>
