# 如何：安装 Office 主互操作程序集
  当你安装 Office 时，安装 Microsoft Office 主互操作程序集 \(PIA\)。  
  
### 安装 Office 时安装 PIA  
  
1.  确保具有不超过 2.0 版本的 .NET Framework。  
  
2.  安装 Microsoft Office，并确保为你想要扩展的应用程序选择“.NET 可编程性支持”功能（默认安装中包括此功能）。  
  
    > [!WARNING]  
    >  默认情况下，当你生成 PIA 时它们就已嵌入到了你的解决方案中，因此你不必将 PIA 分发给用户来作为使用 VSTO 外接程序或自定义项的先决条件。
