# 非托管 API 参考（Visual Studio 中的 Office 开发）
  从 2007 Microsoft Office system 开始，Office 应用程序使用 [IManagedAddin 接口](../vsto/imanagedaddin-interface.md) 接口来调用 [!INCLUDE[vsto_runtime](../vsto/includes/vsto-runtime-md.md)] 中附带的 VSTO 外接程序加载程序组件。 此组件用于帮助加载托管 VSTO 外接程序。 可以通过实现此接口来创建自己的 VSTO 外接程序加载程序组件。  
  
## 本节内容  
 [IManagedAddin 接口](../vsto/imanagedaddin-interface.md)  
 一种 COM 接口，可以通过实现该接口来在 Office 应用程序中加载和卸载 VSTO 托管外接程序。
