# <a name="uninstall-visual-studio"></a>卸载 Visual Studio
此页面介绍如何卸载 Visual Studio（一个面向开发人员的工作效率工具集成套件）。  

1.  在计算机上找到 Visual Studio 安装程序。  

     例如，在运行 Windows 10 周年更新的计算机上，单击“启动”，然后滚动到字母“M”，在此处可以看到其列示为“Microsoft Visual Studio 安装程序”。  

     ![00-ModifyingDev15Prev5-FindTheVisualStudioInstaller](../install/media/00-modifyingdev15prev5-findthevisualstudioinstaller.png)

2.  单击以启动安装程序，单击“详细信息”![](media/vs2017uninstall-UninstallIcon.png)图标，然后单击“卸载”。  

     ![修改 Visual Studio 2017 RC；启动或修改](media/vs2017uninstall-ChooseUninstallFromInstaller.png "修复或卸载 Visual Studio 2017 RC")  

3.  单击“确定”确认所做选择。

     ![修改 Visual Studio 2017 RC；启动或修改](media/vs2017uninstall-UninstallConfirm.png "确认卸载 Visual Studio 2017 RC")  

如果以后改变主意，想重新安装 Visual Studio 2017 RC，只需再次启动 Visual Studio 安装程序，然后从选择屏幕中单击“安装”。

> [!IMPORTANT]
> 虽然一般情况下支持在生产环境中使用 Visual Studio 2017 RC，但安装 UI 中标记为“预览”的工作负载和组件不支持在生产环境中使用。
