1. 若要连接到 GitHub 存储库，请运行 Visual Studio 安装程序，选择“修改”，然后选择“单个组件”选项卡。向下滚动到“代码工具”部分，选择“适用于 Visual Studio 的 GitHub 扩展”，然后选择“修改”。

    ![在 Visual Studio 安装程序中选择 GitHub 扩展](../media/installation-github-extension.png)
