# <a name="export-button-concurrency-visualizer"></a>“导出”按钮（并发可视化工具）
通过“导出”按钮，可以将调用堆栈作为 .csv 文件导出，以用于自己的记录或与其他工具（如 Microsoft Excel）一起使用。
