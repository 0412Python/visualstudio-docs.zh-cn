# 不显示线程活动（线程视图）
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

此区域显示有关当前可见时间范围内的非隐藏线程的数据。  
  
 如果信息不可见，请检查以下设置：  
  
-   缩放级别是否太高？  尝试缩小，或滚动以在范围内显示更多线程活动。  
  
-   是否隐藏太多线程？  如果是这样，尝试显示所有线程  
  
-   如果选择了**“仅我的代码”**，则只能查看有关您的代码的数据。  尝试清除设置以确定是否有任何系统线程活动。  
  
-   确保“降噪”设置为低阈值。
