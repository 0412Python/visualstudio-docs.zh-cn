# 可见时间线分析
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

线程阻塞视图的可视时间线配置文件提供要报告的统计信息和链接。  当您放大、缩小、水平滚动、隐藏或显示通道时，活动图例中的数字将更改以反映视图中的当前内容。  若要查看有关图例中的某项的报告，请单击该项。
