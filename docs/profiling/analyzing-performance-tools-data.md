# 对分析工具数据进行分析
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

通过 [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] 分析工具的性能报告，您可以查看并分析应用程序的性能问题。  本节概述了性能报告和可用于检查性能数据的视图。  
  
## 常规任务  
  
|任务|相关内容|  
|--------|----------|  
|**使用性能规则快速发现问题：**分析工具性能规则可以发现常见问题，并帮助您轻松导航到有问题的源代码。  详细帮助主题通常可以建议一个解决方案。|-   [使用性能规则对数据进行分析](../profiling/using-performance-rules-to-analyze-data.md)|  
|**了解报告视图详细信息：**分析工具报告视图为分析运行的进程、线程、模块和函数提供了聚合性能数据。  显示的数据取决于用于收集数据的分析方法。|-   [分析工具报告视图](../profiling/performance-report-views.md)|  
|**对报告视图进行配置、排序和筛选**：您可以指定要在报告中显示的数据列并对其进行排序，对报告行进行排序，对数据进行筛选以仅包括您指定的时间段。|-   [自定义分析工具报表视图](../profiling/customizing-performance-tools-report-views.md)|
