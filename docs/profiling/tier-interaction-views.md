# 层交互视图
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

层交互分析提供有关多层应用程序中函数的执行时间的其他信息，这些应用程序通过 [!INCLUDE[vstecado](../data-tools/includes/vstecado_md.md)] 与数据库通信。  收集的数据仅用于同步函数调用。  在所有分析方法中都可以加入层交互数据。
