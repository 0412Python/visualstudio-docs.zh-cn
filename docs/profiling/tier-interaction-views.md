# <a name="tier-interaction-views"></a>层交互视图
层交互分析提供有关多层应用程序函数执行时间的附加信息，这些应用程序通过 [!INCLUDE[vstecado](../data-tools/includes/vstecado_md.md)] 服务与数据库进行通信。 仅针对同步函数调用收集数据。 可以在所有分析方法中包含层交互数据。
