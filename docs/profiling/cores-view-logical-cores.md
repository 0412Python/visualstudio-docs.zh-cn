# 内核视图的逻辑内核数
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

每个通道名称代表被分析系统上存在的一个逻辑内核。  该命名方法从 0 开始向上计数到逻辑内核的总数。  例如，四核系统从逻辑内核 0 计数到逻辑内核 3。
