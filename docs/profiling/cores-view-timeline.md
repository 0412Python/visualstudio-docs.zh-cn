# 内核视图时间线
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

时间线中的每一行代表分析系统上的一个逻辑处理器内核。  对于每一行，水平轴显示在给定时间点在逻辑内核上运行的线程。  可以将鼠标指针悬停在时间线中的相关颜色上，以返回标识该线程的工具提示。  为了帮助标识线程，窗口底部的图例显示每种颜色所代表的内容。  使用缩放工具可以放大和缩小，方法是单击并拖动鼠标，或者按 Ctrl 并移动鼠标滚轮。  在内核视图和线程视图之间切换时，将保持缩放一致性。
