# 收集其他性能数据
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

使用 [!INCLUDE[vsPreShort](../code-quality/includes/vspreshort_md.md)]分析工具，除可以收集使用分析方法收集的数据外，还可以记录和查看其他数据。  
  
## 常规任务  
  
|任务|相关内容|  
|--------|----------|  
|**收集应用程序中 ADO.NET 调用的性能数据。**添加有关应用程序对数据库进行的同步调用的数据。|-   [收集层交互数据](../profiling/collecting-tier-interaction-data.md)|  
|**收集 Windows 性能计数器数据。**将系统性能计数器作为分析标记添加到分析数据。  可以使用标记筛选报告。|-   [如何：收集 Windows 计数器数据](../profiling/how-to-collect-windows-counter-data.md)|  
|**收集 Windows 事件跟踪数据。**除了分析数据之外，还会收集 Windows 事件跟踪 \(ETW\) 数据以记录应用程序和系统事件。|-   [如何：收集 Windows 事件跟踪 \(ETW\) 数据](../Topic/How%20to:%20Collect%20Event%20Tracing%20for%20Windows%20\(ETW\)%20Data.md)|
