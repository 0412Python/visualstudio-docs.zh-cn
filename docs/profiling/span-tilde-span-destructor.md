# <a name="spanspan-destructor"></a>span::~span 析构函数

## <a name="syntax"></a>语法  
  
```cpp  
~span();  
```  
  
## <a name="requirements"></a>要求  
 **标头：** cvmarkersobj.h  
  
 **命名空间：** Concurrency::diagnostic
