# Concurrency 命名空间（并发可视化工具）
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

`Concurrency` 命名空间提供了一些类和函数，可以利用它们访问并发运行时（针对 C\+\+ 的并发编程框架）。  有关详细信息，请参阅[并发运行时](/visual-cpp/parallel/concrt/concurrency-runtime)。  
  
## 语法  
  
```  
namespace Concurrency;  
```  
  
## 成员  
  
### 命名空间  
  
|名称|说明|  
|--------|--------|  
|[diagnostic 命名空间](../profiling/diagnostic-namespace.md)|`diagnostics` 命名空间提供用于并发可视化工具标记的功能。|  
  
## 要求  
 **页眉：**cvmarkersobj.h
