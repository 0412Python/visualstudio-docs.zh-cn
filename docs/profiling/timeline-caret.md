# 时间线插入符号
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

当在时间线上选择执行线程段时，此插入符号显示在该段上方。  “当前堆栈”选项卡上显示的调用堆栈是时间上与您单击该段的时间最接近的一个调用堆栈。  脱字用于关联的调用堆栈，显示在 **当前** 选项卡下，使用即时，则该示例后。  插入符号显示调用堆栈的确切位置，即最近的调用堆栈，用户选择位置。
