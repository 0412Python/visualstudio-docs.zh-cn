# <a name="cvleavespan-function"></a>CvLeaveSpan 函数
标记范围的结束位置。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT CvLeaveSpan(  
   _In_ PCV_SPAN pSpan  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pSpan`  
 上一 CvEnterSpan* 调用返回的范围对象 不能为 NULL。  
  
## <a name="return-value"></a>返回值  
 成功写入消息时返回 S_OK。 出现任何错误时返回错误代码。 使用 SUCCEEDED/FAILED 宏检查错误条件。
