# 线程就绪连接器
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

单击阻塞时间段以查看调用堆栈及其解除阻塞的堆栈时，还可能会显示线程就绪连接器。  如果解除阻塞的事件在当前进程中的另一个线程上发生，则线程就绪连接器可直观地标识允许阻塞的线程继续执行的线程和执行时间段。
