# “缩放”控件（使用率视图）
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

缩放控件可帮助您放大和缩小 CPU 使用率图表，从而使您能够关注特定的感兴趣区域。  由于此控件在视图的中心区域执行放大，  因此，应在放大之前将感兴趣的区域置于中心。  
  
 可以将在 CPU 使用率图表或活动图的 GPU 鼠标指针将创建一个突出显示的区域。  释放鼠标按钮后，时间线视图会放大选定的范围。
