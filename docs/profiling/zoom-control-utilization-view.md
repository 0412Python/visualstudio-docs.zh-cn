# <a name="zoom-control-utilization-view"></a>“缩放”控件（使用率视图）
缩放控件可帮助您放大 CPU 使用率图表，从而使您能够关注特定的感兴趣区域。 此控件可在视图的中心区域放大。 所以在放大之前将感兴趣的区域移到中心位置。  
  
 在 CPU 使用率图表或 GPU 活动图中，可以拖动鼠标指针创建一个突出显示的区域。 释放鼠标按钮后，视图会放大选定的范围。
