# 如何：自定义分析工具报告视图列

-   [“函数”视图](../profiling/functions-view.md)  
  
-   [“行”视图](../profiling/lines-view.md)  
  
-   [“指令指针”\(IP\) 视图](../profiling/instruction-pointers-ips-view.md)

-   [“进程”视图](../profiling/process-view.md)  
  
### 自定义报告视图  
  
1.  打开报告视图。  
  
2.  右击报告视图图面，然后单击**“添加\/移除列”**。  
  
     将出现**“添加\/移除列”**对话框。  
  
3.  选择要添加的列。  
  
4.  单击箭头按钮对列进行重新排序。  
  
5.  单击**“确定”**。
