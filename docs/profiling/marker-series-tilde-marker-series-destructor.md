# <a name="markerseriesmarkerseries-destructor"></a>marker_series::~marker_series 析构函数
销毁 marker_series 对象并释放所有已分配的资源。  
  
## <a name="syntax"></a>语法  
  
```cpp  
~marker_series();  
```  
  
## <a name="requirements"></a>要求  
 **标头：** cvmarkersobj.h  
  
 **命名空间：** Concurrency::diagnostic
