# “函数”视图
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

“函数”视图列出分析期间调用的函数。  
  
 默认情况下，按名称列出函数。  若要按模块查看函数，请在“函数”视图窗口中单击鼠标右键，然后选择**“在模块视图中显示”**。  这会显示“模块”视图。  
  
 若要添加或移除列，请在报告窗口中单击鼠标右键，然后选择**“添加\/移除列”**。  此外，还可以通过单击列名对数据进行排序。  有关更多信息，请参见 [如何：自定义报告视图列](../profiling/how-to-customize-report-view-columns.md)。  
  
 “函数”视图中有哪些列取决于用于收集数据的分析方法（采样或检测），以及在分析运行期间是否收集了 .NET 内存数据。
