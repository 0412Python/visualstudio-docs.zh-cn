# 如何：使用分析工具手动创建性能会话
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

您可以手动创建性能会话。  不需要在 [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] 中打开项目，就可以执行该操作。  有关更多信息，请参见[配置性能会话](../profiling/configuring-performance-sessions.md)。  
  
### 手动创建性能会话  
  
1.  在**“分析”**菜单上指向**“探查器”**，然后单击**“新建性能会话”**。  
  
     随即将向**“性能资源管理器”**中添加一个空性能会话。  
  
2.  右击**“目标”**，然后单击**“添加目标二进制文件”**。  
  
3.  在**“添加目标二进制文件”**对话框中，选择文件名，然后单击**“打开”**。  
  
     将添加一个新的二进制文件。
