# <a name="idiaframedatagetvirtualaddress"></a>IDiaFrameData::get_virtualAddress
检索该帧的代码的虚拟地址 (VA)。  
  
## <a name="syntax"></a>语法  
  
```C++  
HRESULT get_virtualAddress (   
   ULONGLONG* pRetVal  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pRetVal`  
 [out]返回框架代码的虚拟地址。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回 `S_OK`。 返回`S_FALSE`如果此属性不受支持。 否则，返回错误代码。
