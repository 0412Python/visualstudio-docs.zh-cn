# <a name="idiasectioncontribgetlength"></a>IDiaSectionContrib::get_length
检索一个部分中的字节数。  
  
## <a name="syntax"></a>语法  
  
```C++  
HRESULT get_length (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pRetVal`  
 [out]返回的部分中的字节数。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回 `S_OK`。 返回`S_FALSE`如果此属性不受支持。 否则，返回错误代码。
