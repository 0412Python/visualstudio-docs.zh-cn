# <a name="idiasectioncontribgetlength"></a>IDiaSectionContrib::get_length
检索部分中的字节的数。  
  
## <a name="syntax"></a>语法  
  
```C++  
HRESULT get_length (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pRetVal`  
 [out]在某个部分中返回字节的数。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`。 返回`S_FALSE`如果不支持此属性。 否则，返回错误代码。
