# <a name="idiasymbolgetstride"></a>IDiaSymbol::get_stride
检索矩阵或 strided 数组的步幅。  
  
## <a name="syntax"></a>语法  
  
```C++  
HRESULT get_stride(   
   DWORD* pRetVal);  
```  
  
#### <a name="parameters"></a>参数  
 `pRetVal`  
 [out]指向的指针`DWORD`保存 stride。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`; 否则为返回`S_FALSE`或错误代码。
