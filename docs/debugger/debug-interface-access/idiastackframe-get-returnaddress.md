# <a name="idiastackframegetreturnaddress"></a>IDiaStackFrame::get_returnAddress
检索的框架的寄信人地址。  
  
## <a name="syntax"></a>语法  
  
```C++  
HRESULT get_returnAddress (   
   ULONGLONG* pRetVal  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pRetVal`  
 [out]返回的框架的寄信人地址。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`。 返回`S_FALSE`如果不支持的属性。 否则，返回错误代码。
