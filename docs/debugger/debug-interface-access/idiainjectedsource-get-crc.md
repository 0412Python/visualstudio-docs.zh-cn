# <a name="idiainjectedsourcegetcrc"></a>IDiaInjectedSource::get_crc
检索循环冗余检查 (CRC) 计算出的源代码的字节数。  
  
## <a name="syntax"></a>语法  
  
```C++  
HRESULT get_crc (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pRetVal`  
 [out]返回 CRC 的源代码的字节从计算。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`。 返回`S_FALSE`如果不支持此属性。 否则，返回错误代码。
