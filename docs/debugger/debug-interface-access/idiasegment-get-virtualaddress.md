# <a name="idiasegmentgetvirtualaddress"></a>IDiaSegment::get_virtualAddress
检索部分的开头的虚拟地址 (VA)。

## <a name="syntax"></a>语法

```C++
HRESULT get_virtualAddress ( 
   ULONGLONG* pRetVal
);
```

#### <a name="parameters"></a>参数
 `pRetVal`

[out]返回部分的开头的 VA。

## <a name="return-value"></a>返回值
 如果成功，则返回 `S_OK`。 返回`S_FALSE`如果此属性不受支持。 否则，返回错误代码。
