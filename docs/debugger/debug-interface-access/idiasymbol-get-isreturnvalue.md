# IDiaSymbol::get_isReturnValue
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

指定变量是否具有一个返回值。  
  
## 语法  
  
```cpp  
HRESULT get_isReturnValue(   
   BOOL* pRetVal);  
```  
  
#### 参数  
 `pRetVal`  
 \[in\]指定要 `BOOL` 的指针变量是否具有一个返回值。  
  
## 返回值  
 如果成功，则返回 `S_OK`，否则返回 `S_FALSE` 或者错误值代码。
