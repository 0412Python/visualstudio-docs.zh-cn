# <a name="idiasymbolgetisreturnvalue"></a>IDiaSymbol::get_isReturnValue
指定该变量是否包含返回值。  
  
## <a name="syntax"></a>语法  
  
```C++  
HRESULT get_isReturnValue(   
   BOOL* pRetVal);  
```  
  
#### <a name="parameters"></a>参数  
 `pRetVal`  
 [out]一个指向`BOOL`，它指定该变量是否包含返回值。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`; 否则为返回`S_FALSE`或错误代码。
