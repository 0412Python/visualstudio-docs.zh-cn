# <a name="idiasymbolgetbasesymbolid"></a>IDiaSymbol::get_baseSymbolId
检索从中基于指针的符号 ID。  
  
## <a name="syntax"></a>语法  
  
```C++  
HRESULT get_baseSymbolId(   
   DWORD *pRetVal);  
```  
  
#### <a name="parameters"></a>参数  
 `pRetVal`  
 [out]指向的指针`DWORD`，用于保存从其基于指针的符号 ID。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`; 否则为返回`S_FALSE`或错误代码。
