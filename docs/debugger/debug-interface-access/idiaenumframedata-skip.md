# <a name="idiaenumframedataskip"></a>IDiaEnumFrameData::Skip
跳过指定的数目的帧枚举序列中的数据元素。  
  
## <a name="syntax"></a>语法  
  
```C++  
HRESULT Skip (   
   ULONG celt  
);  
```  
  
#### <a name="parameters"></a>参数  
 celt  
 [in]在枚举顺序中跳过的帧数据元素的数目。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`; 否则为返回`S_FALSE`是否存在要跳过没有更多记录。
