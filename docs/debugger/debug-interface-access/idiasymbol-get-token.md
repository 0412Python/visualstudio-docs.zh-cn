# <a name="idiasymbolgettoken"></a>IDiaSymbol::get_token
检索一个托管的函数或变量的元数据标记。  
  
## <a name="syntax"></a>语法  
  
```C++  
HRESULT get_token (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pRetVal`  
 [out]返回一个托管的函数或变量的元数据标记。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`; 否则为返回`S_FALSE`或错误代码。  
  
> [!NOTE]
>  返回值`S_FALSE`意味着属性不是可用于符号。
