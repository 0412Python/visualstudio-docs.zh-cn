# <a name="idiaenumsymbolsskip"></a>IDiaEnumSymbols::Skip
跳过指定的数目的枚举序列中的符号。  
  
## <a name="syntax"></a>语法  
  
```C++  
HRESULT Skip (   
   ULONG celt  
);  
```  
  
#### <a name="parameters"></a>参数  
 celt  
 [in]要跳过的枚举序列中的符号数。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`; 否则为返回`S_FALSE`如果没有更多的符号，以跳过。
