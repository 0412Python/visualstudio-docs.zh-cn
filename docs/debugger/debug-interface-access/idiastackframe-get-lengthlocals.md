# <a name="idiastackframegetlengthlocals"></a>IDiaStackFrame::get_lengthLocals
检索局部变量推送到堆栈上的字节的数。  
  
## <a name="syntax"></a>语法  
  
```C++  
HRESULT get_lengthLocals (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pRetVal`  
 [out]返回局部变量的字节数。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回 `S_OK`。 返回`S_FALSE`如果不支持的属性。 否则，返回错误代码。
