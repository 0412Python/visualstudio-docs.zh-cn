# IDiaEnumStackFrames::Reset
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

将枚举序列重置到开始处。  
  
## 语法  
  
```cpp#  
HRESULT Reset();  
```  
  
## 返回值  
 返回 `S_OK`。
