# <a name="idiasymbolgetnested"></a>IDiaSymbol::get_nested
检索指定用户定义数据类型是否嵌套的标志。  
  
## <a name="syntax"></a>语法  
  
```C++  
HRESULT get_nested (   
   BOOL* pRetVal  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pRetVal`  
 [out]返回`TRUE`如果嵌套的用户定义数据类型; 否则，返回`FALSE`。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`; 否则为返回`S_FALSE`或错误代码。  
  
> [!NOTE]
>  返回值为`S_FALSE`表示该属性不是可用于符号。
