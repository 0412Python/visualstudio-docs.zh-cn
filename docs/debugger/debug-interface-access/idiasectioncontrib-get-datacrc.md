# <a name="idiasectioncontribgetdatacrc"></a>IDiaSectionContrib::get_dataCrc
检索的部分中的数据的循环冗余检查 (CRC)。  
  
## <a name="syntax"></a>语法  
  
```C++  
HRESULT get_dataCrc (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pRetVal`  
 [out]返回的部分中的数据的 CRC。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回 `S_OK`。 返回`S_FALSE`如果此属性不受支持。 否则，返回错误代码。
