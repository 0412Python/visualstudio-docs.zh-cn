# <a name="idiasymbolgetmanaged"></a>IDiaSymbol::get_managed
检索一个标志，指定是否引用托管代码符号。  
  
## <a name="syntax"></a>语法  
  
```C++  
HRESULT get_managed (   
   BOOL* pRetVal  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pRetVal`  
 [out]返回`TRUE`如果符号所引用的托管代码; 否则，返回`FALSE`。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`; 否则为返回`S_FALSE`或错误代码。  
  
> [!NOTE]
>  返回值为`S_FALSE`表示该属性不是可用于符号。
