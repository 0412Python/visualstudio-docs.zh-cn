# <a name="idiasymbolgetbuiltinkind"></a>IDiaSymbol::get_builtInKind
检索 HLSL 类型的内置类型。  
  
## <a name="syntax"></a>语法  
  
```C++  
HRESULT get_buildInKind(   
   DWORD* pRetVal);  
```  
  
#### <a name="parameters"></a>参数  
 `pRetVal`  
 [out]指向的指针`DWORD`保存是内置类型的 HLSL 类型。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`; 否则为返回`S_FALSE`或错误代码。
