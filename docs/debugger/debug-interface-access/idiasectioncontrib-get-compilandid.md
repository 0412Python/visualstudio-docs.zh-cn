# <a name="idiasectioncontribgetcompilandid"></a>IDiaSectionContrib::get_compilandId
检索部分的编译单位标识符。  
  
## <a name="syntax"></a>语法  
  
```C++  
HRESULT get_compilandId (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pRetVal`  
 [out]返回部分中的编译单位标识符。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回 `S_OK`。 返回`S_FALSE`如果此属性不受支持。 否则，返回错误代码。
