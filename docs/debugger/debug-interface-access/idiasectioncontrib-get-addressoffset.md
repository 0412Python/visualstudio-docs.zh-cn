# <a name="idiasectioncontribgetaddressoffset"></a>IDiaSectionContrib::get_addressOffset
检索的贡献地址的偏移量的部分。  
  
## <a name="syntax"></a>语法  
  
```C++  
HRESULT get_addressOffset (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pRetVal`  
 [out]返回的贡献地址的偏移量的部分。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`。 返回`S_FALSE`如果不支持此属性。 否则，返回错误代码。
