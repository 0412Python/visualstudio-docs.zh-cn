# <a name="idiasymbolgetbasedataslot"></a>IDiaSymbol::get_baseDataSlot
检索基本数据槽。  
  
## <a name="syntax"></a>语法  
  
```C++  
HRESULT get_baseDataSlot(   
   DWORD* pRetVal);  
```  
  
#### <a name="parameters"></a>参数  
 `pRetVal`  
 [out]一个指向`DWORD`，它持有的基本数据槽。  
  
## <a name="return-value"></a>返回值  
 如果成功，则返回`S_OK`; 否则为返回`S_FALSE`或错误代码。
