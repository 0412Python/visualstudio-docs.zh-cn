# <a name="windows-tab-window-properties-dialog-box"></a>“窗口属性”对话框 ->“窗口”选项卡
使用**Windows**选项卡来显示在 windows 上的信息与所选的窗口。 若要显示[窗口属性对话框](../debugger/window-properties-dialog-box.md)，焦点移到[Windows 视图](../debugger/windows-view.md)窗口。 在树中，选择任何窗口节点，然后选择**属性**从**视图**菜单。  
  
 以下设置子网上有**Windows**选项卡：  
  
|条目|描述|  
|-----------|-----------------|  
|**下一个窗口**|（如果没有下一个窗口"无"） 的窗口树视图中所示的相同序列 （Z 顺序） 中的下一个同级窗口的句柄。 选择此项可查看下一步窗口的属性。|  
|**上一个窗口**|在窗口树视图 （如果没有上一个窗口"无"） 中所示的相同序列 （Z 顺序） 的上一个同级窗口句柄。 选择此项可查看上一个窗口的属性。|  
|**父窗口**|此窗口 （如果不存在父"无"） 的父窗口的句柄。 选择此项可查看父窗口的属性。|  
|**第一个子级**|此窗口中第一个子窗口，序列 （Z 顺序） （如果不有任何子窗口"无"） 窗口树视图中显示的句柄。 选择此值可查看的第一个子窗口的属性。|  
|**所有者窗口**|此窗口的所有者窗口的句柄。 通常，应用程序的主窗口拥有系统模式对话框窗口，例如 （如果没有任何所有者"无"）。 选择此项可查看所有者窗口的属性。|
