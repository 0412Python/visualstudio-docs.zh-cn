# <a name="spy-views"></a>Spy++ 视图

## <a name="in-this-section"></a>本节内容  
 [窗口视图](../debugger/windows-view.md)  
 显示可用的 windows 和控件的树视图。  
  
 [消息视图](../debugger/messages-view.md)

 [进程视图](../debugger/processes-view.md)  
 显示活动进程的树视图。
