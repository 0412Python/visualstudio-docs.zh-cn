# <a name="symbol-load-information-dialog-box"></a>“符号加载信息”对话框
右键单击“模块”窗口并选择“符号加载信息”后，将打开“符号加载信息”对话框。

## <a name="uielement-list"></a>UIElement 列表
 **信息**显示了搜索符号文件 （PDB 文件） 基于 Visual Studio 安装设置和对中的符号设置所做的更改目录**选项**对话框。

 **符号设置**单击此按钮可更改中的符号设置**选项**对话框的 (**调试**类别中，**符号**页)。 在该页上，你可以添加或移除符号位置、指定本地缓存位置（如果使用的是符号服务器）并控制搜索符号位置的时间。
