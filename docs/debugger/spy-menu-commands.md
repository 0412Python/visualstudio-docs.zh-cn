# “监视”菜单命令
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

“监视”菜单提供用于处理 Spy\+\+ 窗口和视图的各个命令。  
  
 **Windows**  
 打开新的窗口视图。  
  
 **进程**  
 打开新的进程视图。  
  
 **线程**  
 打开新的线程视图。  
  
 **日志消息**  
 打开新的消息视图，并打开该视图的[“消息选项”对话框](../debugger/message-options-dialog-box.md)。  
  
 **查找窗口**  
 显示“查找窗口”对话框以使用鼠标选择窗口。  
  
 **Exit**  
 关闭 Spy\+\+ 实用工具。
