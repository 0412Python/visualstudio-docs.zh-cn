# 调试准备：Windows 服务
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Windows 服务是一种在 Microsoft Windows 的后台中运行的程序。  Telnet 服务和 Windows 时间服务（它更新计算机的可见时钟）都属于 Windows 服务。  Windows 服务不能从 [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] 内部运行；它必须在服务控制管理器的上下文中运行。  有关更多信息，请参见[创建 Windows 服务](../Topic/How%20to:%20Create%20Windows%20Services.md)、[调试 Windows 服务应用程序](../Topic/How%20to:%20Debug%20Windows%20Service%20Applications.md)和 [Windows 服务应用程序](../Topic/Developing%20Windows%20Service%20Applications.md)。
