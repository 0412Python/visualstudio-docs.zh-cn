# How to: Expand and Collapse Spy++ Trees
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

可以通过两种方法展开和折叠窗口、进程和线程视图：单击窗口中的相应图标或使用**“树”**菜单。  树中的 \+ 和 – 图标与它们在 Visual C\+\+ 项目窗口中的含义相同。  
  
 **“树”**菜单包含四个命令：  
  
|菜单命令|说明|  
|----------|--------|  
|**展开一层**|将当前选定的项展开到下一层。|  
|**展开分支**|完全展开当前选定的项。|  
|**全部展开**|完全展开窗口中的所有项。|  
|**折叠**|完全折叠当前选定的项。|  
  
> [!TIP]
>  如果展开某个进程，则将看到该进程拥有的所有线程。  如果展开某个线程，则将看到该线程拥有的所有窗口列表。  
  
### 展开或折叠 Spy\+\+ 树  
  
1.  突出显示窗口、进程或线程视图中的某个项。  
  
2.  从**“树”**菜单中选择某个展开或折叠命令。
