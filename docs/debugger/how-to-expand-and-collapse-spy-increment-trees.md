# <a name="how-to-expand-and-collapse-spy-trees"></a>如何：展开和折叠 Spy++ 树
你可以展开和折叠的 Windows、 进程和线程视图，使用两种方法： 通过单击图标在窗口中或通过使用**树**菜单。 + 和-与它们在 Visual c + + 项目窗口中，执行操作在树中的图标。  
  
 **树**菜单包含四个命令：  
  
|菜单命令|描述|  
|------------------|-----------------|  
|**展开一层**|到下一步级别展开当前选定的项。|  
|**展开分支**|完全展开当前选定的项。|  
|**全部展开**|完全展开窗口中的所有项目。|  
|**折叠**|完全折叠当前选定的项。|  
  
> [!TIP]
>  如果你展开一个进程，你将看到该进程拥有的所有线程。 如果你展开一个线程，你将看到它所拥有的所有窗口的列表。  
  
### <a name="to-expand-or-collapse-spy-trees"></a>若要展开或折叠 Spy + + 树  
  
1.  突出显示 Windows、 进程或线程视图中的项之一。  
  
2.  从**树**菜单上，选择一个展开或折叠命令。
