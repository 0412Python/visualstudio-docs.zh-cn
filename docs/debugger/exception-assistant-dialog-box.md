# “异常助手”对话框
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

**“异常助手”**对话框在引发运行时异常时出现。  **“异常助手”**显示异常类型，提供其他信息和到疑难解答提示的链接，提供搜索其他联机帮助的方法，并允许用户执行某些操作（如查看异常的详细信息）。  
  
## 任务列表  
 若要查看有关对遇到的异常类型进行疑难解答的主题，请单击**“疑难解答提示”**窗格中显示的提示消息之一。  
  
 若要执行与该异常相关的操作，请单击操作窗格中显示的操作之一。  
  
 有关如何启用或禁用异常助手的信息，请参阅[“选项”对话框 \-\>“调试”\-\>“常规”](../debugger/general-debugging-options-dialog-box.md)。  
  
## 用户界面元素列表  
 **异常的类型**  
 显示所引发异常的类型。  
  
 **其他信息**  
 显示有关异常的其他信息。  
  
 **疑难解答指南**  
 显示指向疑难解答提示的链接，这些提示可帮助您发现异常的源。  
  
 **操作**  
 列出可执行的操作（例如，查看有关异常对象的详细信息）。  
  
 **获取联机帮助**  
 使您可以搜索其他联机帮助。
