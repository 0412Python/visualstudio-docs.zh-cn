# <a name="vsgdbgvsgdbg-destructor"></a>VsgDbg::~VsgDbg（析构函数）
销毁实例`VsgDbg`类。 如果主动记录图形信息时，完成图形日志文件并将其关闭，并释放主动捕获图形信息时使用的资源。  
  
## <a name="syntax"></a>语法  
  
```C++  
~VsgDbg();  
```
