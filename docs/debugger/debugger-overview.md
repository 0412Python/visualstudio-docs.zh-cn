# <a name="overview-of-the-visual-studio-debugger"></a>Visual Studio 调试器的概述

Visual Studio 调试程序允许你观察程序的运行时行为并确定逻辑错误的位置。 该调试器可用于所有的 Visual Studio 编程语言及其关联的库。 使用调试程序时，可以中断执行以逐句通过代码和计算应用程序中的变量。

调试器可以设置断点、 检查变量和设置监视、 查看调用堆栈，在远程设备上进行调试，调试多线程应用程序，并利用许多其他功能。
