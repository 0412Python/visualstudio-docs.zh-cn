# <a name="visualizer-api-reference"></a>可视化工具 API 参考
可视化工具 API 是专为希望为 Visual Studio 调试器编写可视化工具的用户而提供的。 可视化工具是一种小应用程序，它扩展了 Visual Studio 调试器用户界面的功能。 可视化工具可以显示 （和 （可选） 编辑） 可视化工具旨在为其特定类型的数据对象。
