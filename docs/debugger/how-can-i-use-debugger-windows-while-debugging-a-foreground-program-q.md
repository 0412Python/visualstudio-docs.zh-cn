# 调试前台程序时如何使用调试器窗口？
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

## 问题描述  
 我在尝试调试屏幕绘制问题。  若要观察该问题，必须将程序保持在前台，这意味着不能访问调试窗口。  我该怎么办？  
  
## 解决方案  
 如果有另一台计算机，则可以使用远程调试。  通过两台计算机的设置，当在主机上运行调试器时，可以监视远程计算机上的屏幕绘制。  有关远程调试的更多信息，请参见[设置远程调试](../Topic/Set%20Up%20the%20Remote%20Tools%20on%20the%20Device.md)。
