# <a name="help-menu-commands"></a>“帮助”菜单命令
**帮助**菜单提供命令来打开帮助文件。  
  
 **Tuple**  
 将内容屏幕打开 Spy + + 帮助的。  
  
 **使用帮助**  
 打开介绍如何使用帮助的帮助文件。  
  
 **关于 Spy++**  
 显示 Spy + + 中的软件版本和版权信息。
