# <a name="stale-code-warning-dialog-box"></a>“陈旧代码警告”对话框
将显示此对话框时所做更改为本机代码**编辑并继续**未能立即应用。 因此，当前堆栈帧中的某些本机代码是过时的、陈旧的。 有关详细信息，请参阅[如何： 使用陈旧代码](/visualstudio/debugger/edit-and-continue-visual-cpp#bkmk_how_to_work_with_stale_code)。  
  
 **不再显示此对话框**  
 如果选中此复选框，则“编辑并继续”以后将在不要求权限的情况下应用代码更改。 您可以打开此警告再次转到**选项**对话框中，打开**调试**文件夹，单击**编辑并继续**页，然后选择**就陈旧的代码，则发出警告**。
