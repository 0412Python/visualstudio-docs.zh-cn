# “窗口属性”对话框 -&gt;“进程”选项卡
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

使用“进程”选项卡可以标识所选窗口的进程。  若要显示[“窗口属性”对话框](../debugger/window-properties-dialog-box.md)，请将焦点移动到[窗口视图](../debugger/windows-view.md)窗口。  在树中选择任何窗口节点，然后从“视图”菜单中选择“属性”。  
  
 “进程”选项卡上提供了下列设置：  
  
|Entry|说明|  
|-----------|--------|  
|**进程 ID**|拥有创建此窗口的线程的进程的 ID。  选择此值可查看该进程的属性。|  
|**线程 ID**|创建此窗口的线程的 ID。  选择此值可查看该线程的属性。|
