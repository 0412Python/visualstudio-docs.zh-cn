# <a name="process-tab-window-properties-dialog-box"></a>“窗口属性”对话框 ->“进程”选项卡
使用**进程**选项卡以识别的所选的窗口过程。 若要显示[窗口属性对话框](../debugger/window-properties-dialog-box.md)，将焦点移动到 [Windows 视图](../debugger/windows-view.md)窗口。 在树中选择任意窗口节点，然后从“视图”菜单选择“属性”。   
  
 以下设置位于**进程**选项卡：  
  
|条目|说明​​|  
|-----------|-----------------|  
|**进程 ID**|拥有此窗口在创建线程的进程的 ID。 选择此值以查看此过程的属性。|  
|**线程 ID**|创建此窗口的线程的 ID。 选择此值以查看此线程的属性。|
