# <a name="choose-breakpoints-dialog-box"></a>“选择断点”对话框
**选择断点**对话框将打开 when you try to 二义性输入中设置断点**新断点**对话框。 不明确的一个常见示例是输入的重载的函数名称。  
  
 **断点**  
 可以设置的列表可能断点基于不明确的输入。 可以选择一个或多个要设置的断点。  
  
 **All**  
 选择中列出的所有断点**断点**框。  
  
 **无**  
 清除中列出的所有断点**断点**框。
