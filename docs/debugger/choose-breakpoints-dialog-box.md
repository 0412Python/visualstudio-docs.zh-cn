# “选择断点”对话框
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

在尝试为**“新建断点”**对话框中的二义性输入设置断点时，将打开**“选择断点”**对话框。  常见的二义性输入是重载的函数名。  
  
 **断点**  
 列出根据该二义性输入可能设置的断点。  可以选择设置一个或多个断点。  
  
 **全部**  
 选择在**“断点”**框中列出的所有断点。  
  
 **无**  
 清除在**“断点”**框中列出的所有断点。
