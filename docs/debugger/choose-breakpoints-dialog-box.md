# <a name="choose-breakpoints-dialog-box"></a>“选择断点”对话框
**选择断点**对话框随即打开，当你尝试从二义性输入中设置断点时**新断点**对话框。 二义性输入的一个常见示例是重载的函数名称。  
  
 **断点**  
 可以设置的列表可能断点基于二义性的输入。 你可以选择一个或多个设置的断点。  
  
 **All**  
 选择中列出的所有断点**断点**框。  
  
 **无**  
 清除中列出的所有断点**断点**框。
