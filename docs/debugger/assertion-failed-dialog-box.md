# “断言失败”对话框
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

断言语句指定在程序的某些特定点应为真的条件。  如果该条件不为 true，则断言失败、程序的执行中断且此对话框将出现。  
  
|单击|到|  
|--------|-------|  
|重试|调试断言或获取有关断言的帮助。|  
|忽略|忽略断言并继续运行该程序。|  
|中止|暂停执行程序，并结束调试会话。|
