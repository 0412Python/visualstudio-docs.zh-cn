# <a name="search-tools-for-spy-views"></a>Spy++ 视图的搜索工具
使用这些对话框来指定 Spy + + 视图窗口将显示哪些数据：  
  
## <a name="in-this-section"></a>本节内容  
 [“查找窗口”对话框](../debugger/find-window-dialog-box.md)  
 用于查看的属性或特定的窗口中的消息。  
  
 [“窗口搜索”对话框](../debugger/window-search-dialog-box.md)  
 用于查找特定窗口在窗口视图中的节点。  
  
 [“消息搜索”对话框](../debugger/message-search-dialog-box.md)  
 用于查找特定消息在消息视图中的节点。  
  
 [“进程搜索”对话框](../debugger/process-search-dialog-box.md)  
 用于查找进程视图中的特定进程的节点。  
  
 [“线程搜索”对话框](../debugger/thread-search-dialog-box.md)  
 用于查找特定线程在线程视图中的节点。
