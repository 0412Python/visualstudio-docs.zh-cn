# 错误：仅当使用 Microsoft .NET Framework 2.0 或更高版本时，才支持混合模式调试
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

若要调试混合的本机代码和托管代码，必须安装有 [!INCLUDE[dnprdnshort](../code-quality/includes/dnprdnshort_md.md)] 2.0、3.0、3.5 或 4 版。  [!INCLUDE[dnprdnshort](../code-quality/includes/dnprdnshort_md.md)] 的早期版本不支持混合模式调试。  
  
### 更正此错误  
  
-   将 [!INCLUDE[dnprdnshort](../code-quality/includes/dnprdnshort_md.md)] 升级到 2.0、3.0、3.5 或 4 版。
