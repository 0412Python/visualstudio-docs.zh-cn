# “消息”菜单命令
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

“消息”菜单提供用于控制消息视图的命令。  
  
 **开始\/停止记录**  
 开始或停止记录消息。  
  
 **记录选项**  
 打开“消息选项”对话框，您可在该对话框中指定消息搜索条件。  
  
 **清除日志**  
 清除活动消息视图的内容。
