# How to: Display Window Properties
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

若要查找有关[窗口视图](../debugger/windows-view.md)中所列窗口的更多信息，请使用[“窗口属性”对话框](../debugger/window-properties-dialog-box.md)。  
  
### 打开“窗口属性”对话框  
  
1.  选择项，然后从“视图”菜单中选择“属性”。  
  
 属性对话框不是模式对话框。  当您在视图窗口中单击各个项时，将使用每个所选项的信息自动刷新该对话框。
