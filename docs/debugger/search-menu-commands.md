# <a name="search-menu-commands"></a>“搜索”菜单命令
**搜索**菜单提供用于搜索特定窗口的命令。  
  
 **查找窗口/进程/线程/消息**  
 此时将打开**窗口**/**进程**/**线程**/**消息搜索**对话框中，可以在其中使用**这些**或指定搜索条件。  
  
 **查找下一个**  
 搜索下一个匹配的窗口、 进程、 线程或消息的当前视图。 仅当不是唯一有效的搜索结果时，此菜单项 （及相关的工具栏按钮） 才可用。  
  
 **查找上一个**  
 搜索上一个匹配的窗口、 进程、 线程或消息的当前视图。 仅当不是唯一有效的搜索结果时，此菜单项 （及相关的工具栏按钮） 才可用。
