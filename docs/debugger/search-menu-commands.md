# “搜索”菜单命令
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

“搜索”菜单提供用于搜索特定窗口的命令。  
  
 **查找窗口\/进程\/线程\/消息**  
 打开“窗口搜索”\/“进程搜索”\/“线程搜索”\/“消息搜索”对话框，您可在这些对话框中使用“查找程序工具”或指定搜索条件。  
  
 **查找下一个**  
 在当前视图中搜索下一个匹配的窗口、进程、线程或消息。  仅当有效搜索结果不唯一时，此菜单项（及相关工具栏按钮）才可用。  
  
 **查找上一个**  
 在当前视图中搜索上一个匹配的窗口、进程、线程或消息。  仅当有效搜索结果不唯一时，此菜单项（及相关工具栏按钮）才可用。
