# How to: Control Messages View
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

使用 Spy\+\+，您可以对[消息视图](../debugger/messages-view.md)窗口的内容进行相当多的控制。  您随时都可以[开始和停止](../debugger/how-to-start-and-stop-the-message-log-display.md)显示消息，并且可以指定：  
  
-   要查看的消息类型。  
  
-   要监视的窗口。  
  
-   消息行的显示格式。  
  
 可以从[“消息选项”对话框](../debugger/message-options-dialog-box.md)获取这些设置，它们仅适用于活动[消息视图](../debugger/messages-view.md)窗口。  若要打开此对话框，请从“监视”菜单中选择“日志消息”。  
  
 消息选项提供了三个选项卡式窗格。  有关更多信息，请参见[“消息选项”对话框 \-\>“窗口”选项卡](../debugger/windows-tab-message-options-dialog-box.md)、[“消息选项”对话框 \-\>“消息”选项卡](../debugger/messages-tab-message-options-dialog-box.md)和[“消息选项”对话框 \-\>“输出”选项卡](../debugger/output-tab-message-options-dialog-box.md)。
