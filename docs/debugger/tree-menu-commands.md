# “树”菜单命令
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

“树”菜单提供用于展开或折叠视图的命令。  
  
 **展开一层**  
 通过显示当前选定项下的一层，在系统视图中展开该项。  
  
 **展开分支**  
 展开当前选定项下的所有层。  
  
 **全部展开**  
 展开并显示系统视图中的所有层。  
  
 **折叠**  
 折叠当前选定项下的所有层。
