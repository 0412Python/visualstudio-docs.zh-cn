# How to: Start and Stop the Message Log Display
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

当[消息视图](../debugger/messages-view.md)窗口处于活动状态时，Spy\+\+“消息”菜单上将显示“开始记录”或“停止记录”选项，且这两个选项对应的“开始记录”或“停止记录”工具栏按钮将变为活动状态。  
  
### 开始显示消息日志  
  
1.  从“消息”菜单中选择“开始记录”。  
  
### 停止显示消息日志  
  
1.  从“消息”菜单中选择“停止记录”。
