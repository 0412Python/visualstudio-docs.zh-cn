# How to: Display Spy++ Views
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

当 Spy\+\+ 启动时，它将以[窗口视图](../debugger/windows-view.md)打开，该视图是包含计算机上所有可用窗口和控件的树视图。  此根窗口的标题为“窗口 1”。还有其他三种可用的 Spy\+\+ 视图：[消息视图](../debugger/messages-view.md)、[进程视图](../debugger/processes-view.md)和[线程视图](../debugger/threads-view.md)。  可从**“监视”**菜单显示所需视图。  有关更多信息，请参见 [Spy\+\+ 视图](../debugger/spy-increment-views.md)。
