# How to: Change Fonts
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

可以更改 Spy\+\+ 窗口的字体、字形和字号。  
  
### 更改字体选项  
  
1.  从**“视图”**菜单中选择**“字体”**。  
  
2.  选择字体、字形和字号。  
  
3.  单击**“确定”**。  
  
 选择**“保存为默认字体”**可使所有将来的 Spy\+\+ 窗口都使用此字体。
