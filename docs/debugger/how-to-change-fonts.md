# <a name="how-to-change-fonts"></a>如何：更改字体
你可以更改字体、 字形和字号 Spy + + windows。  
  
### <a name="to-change-font-options"></a>若要更改字体选项  
  
1.  从**视图**菜单上，选择**字体**。  
  
2.  选择字体、 字形和字体大小。  
  
3.  单击 **“确定”**。  
  
 选择**保存为默认字体**将导致所有将来 Spy + + windows 用于此字体。
