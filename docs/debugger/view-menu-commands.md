# <a name="view-menu-commands"></a>“视图”菜单命令
**视图**菜单提供用于控制命令 Spy + + 用户界面。  
  
 **字体**  
 打开**字体**对话框中，你可以在其中选择当前的 Spy + + 视图中存在使用的字体。  
  
 **工具栏**  
 隐藏或显示工具栏。  
  
 **状态栏**  
 隐藏或显示状态栏。  
  
 **属性**  
 打开**属性**活动视图的对话框。
