# “视图”菜单命令
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

“视图”菜单提供用于控制 Spy\+\+ 用户界面的命令。  
  
 **字体**  
 打开“字体”对话框，可在其中选择要在当前 Spy\+\+ 视图中使用的字体。  
  
 **工具栏**  
 隐藏或显示工具栏。  
  
 **Status Bar**  
 隐藏或显示状态栏。  
  
 **属性**  
 打开活动视图的“属性”对话框。
