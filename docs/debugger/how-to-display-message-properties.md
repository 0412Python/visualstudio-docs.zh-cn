# How to: Display Message Properties
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

若要查找有关[消息视图](../debugger/messages-view.md)中所列消息的更多信息，请使用[“消息属性”对话框](../debugger/message-properties-dialog-box.md)。  
  
### 打开“消息属性”对话框  
  
1.  从消息视图中显示的树中选择一个消息。  
  
2.  从“视图”菜单中选择“属性”。  
  
 属性对话框不是模式对话框，因此，您可以单击视图窗口中的其他项，且该对话框将显示有关所选项的信息。
