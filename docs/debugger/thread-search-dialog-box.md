# “线程搜索”对话框
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

使用此对话框可以在[线程视图](../debugger/threads-view.md)中查找并选择特定线程的节点。  若要显示此对话框，请将焦点移动到“线程”视图窗口。  然后从“搜索”菜单中选择“查找线程”。  
  
 下列设置可用：  
  
 **线程**  
 要搜索的线程 ID。  
  
 **模块**  
 要搜索的模块字符串。  
  
 **搜索方向: 向上或向下**  
 搜索的初始方向。
