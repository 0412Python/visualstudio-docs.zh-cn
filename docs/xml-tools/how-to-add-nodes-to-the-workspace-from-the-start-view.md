# 如何：从起始视图向工作区中添加节点
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

本主题介绍如何从[起始视图](../xml-tools/start-view.md)向[XML 架构设计器工作区](../xml-tools/xml-schema-designer-workspace.md)中添加全局节点。  
  
### 从起始视图向工作区中添加节点  
  
1.  请按照[如何：创建和编辑 XSD 架构文件](../xml-tools/how-to-create-and-edit-an-xsd-schema-file.md)中的步骤操作。  
  
2.  在 XML 编辑器内右击并选择**“视图设计器”**。  
  
     此时将显示起始视图。  
  
     请注意，**“架构集详细信息”**部分中有两 \(**2**\) 个**“全局复杂类型”**。  
  
3.  单击**“全局复杂类型”**标题右侧的**“添加”**链接。  
  
     此时，`USAddress` 和 `PurchaseOrderType` 复杂类型节点将显示在图形视图的设计图面上。
