# <a name="xslt-profiler"></a>XSLT 探查器

XSLT 探查器是一个性能分析探查器工具，可帮助您开发和调试 XSLT 文档。 可以测量并通过创建详细的 XSLT 性能报告评估 XSLT 代码中的与性能相关问题。 XSLT 探查器包含 XSL 和 XSLT 样式表优化的有用提示，这对需要最佳性能的基于 XSLT 的应用程序必不可少。

XSLT Profiler 是 Visual Studio 的一部分并可从**XML**菜单。

![XSLT 探查器](../xml-tools/media/profile-xslt-menu.png)

> [!NOTE]
> XSLT Profiler 是仅在 Visual Studio Enterprise edition 中可用。
