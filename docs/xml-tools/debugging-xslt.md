# <a name="debugging-xslt"></a>调试 XSLT
您可以使用 Visual Studio 调试程序来调试 XSLT。 调试器支持设置断点、查看 XSLT 执行状态等。 Visual Studio Team System 和 Professional Edition 中支持 XSLT 调试。
