# <a name="integration-of-xml-literals-with-xml-schema-explorer"></a>XML 文本与 XML 架构资源管理器的集成

Visual Basic 支持 XML 文本，这意味着，您可以直接在 Visual Basic 代码合并 XML 片段。 有关详细信息，请参阅[XML 文本概述](http://go.microsoft.com/fwlink/?LinkId=140325)。

## <a name="how-to"></a>如何

如果在 Visual Basic 项目中的 XSD 文件包含 XML 文本，可以查看中设置的 XML 架构**XML 架构资源管理器**。 若要查看与 XML 文本关联的架构集，右键单击 XML 文本或导入的 XML 命名空间中的 XML 节点，然后选择**在架构资源管理器中显示**。

![Visual Basic XML 文本；XML 架构资源管理器](../xml-tools/media/vbxmlliteralswithxmlschemaexplorer1.gif)

这将打开**XML 架构资源管理器**与 Visual Basic 文件并行。

![Visual Basic XML 文本；XML 架构资源管理器](../xml-tools/media/vbxmlliteralswithxmlschemaexplorer2.gif)
