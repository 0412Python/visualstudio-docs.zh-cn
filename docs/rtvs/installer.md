# <a name="welcome-to-r-tools-for-visual-studio"></a>欢迎使用针对 Visual Studio 的 R 工具

感谢安装针对 Visual Studio 的 R 工具 (RTVS)！ 请查看该工具，并使用下[资源](#resources)下的链接告诉我们你的想法。

![RTVS 屏幕截图](~/docs/rtvs/media/installer-screenshot.png)

## <a name="resources"></a>资源

- [文档](index.md)
- [安装步骤](installation.md)
- [在 GitHib 上提交和浏览 bug 以及功能请求](https://github.com/Microsoft/RTVS/issues)
- [浏览和编写源代码](https://github.com/Microsoft/RTVS/blob/master/LICENSE)
- 若要通过 Github（首选）之外的途径发送反馈，请使用 Visual Studio 中的“R 工具”>“反馈”菜单。

## <a name="microsoft-r-distributions"></a>Microsoft R 分发

RTVS 支持 CRAN R 和 Microsoft R 分发，其中包括与 SQL Server 2016 一起分发的 R 版本。 有关不同 Microsoft R 分发的详细信息，请参阅以下链接：

* [Microsoft R Open](https://mran.microsoft.com/download/)
* [Microsoft R Client](https://msdn.microsoft.com/microsoft-r/r-client-get-started)
* [Microsoft R Server](https://www.microsoft.com/server-cloud/products/r-server/)
