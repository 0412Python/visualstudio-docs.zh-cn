# <a name="exporting-subscription-information"></a>导出订阅信息

在 Visual Studio 订阅的[管理门户](https://manage.visualstudio.com)中，可以导出你的用户列表和有关其分配的详细信息。 此信息包括：用户的名称、电子邮件地址、备用电子邮件地址、订阅级别、分配日期、激活状态、到期日期、引用字段、启用的下载、国家/地区、语言、订阅状态和订阅 GUID。 

在一些情况下此功能非常有用。 例如，你可以轻松地跟踪分配和到期日期。 如果你正在从使用 BAN 跟踪订阅转变为使用 GUID 跟踪订阅，则可以将此报表与 Microsoft Excel 中的 VLOOKUP 公式一起使用，以正确匹配订阅者。  

要执行导出操作，只需选择“导出”选项卡，文件就会下载到本地计算机。 该文件将包含用户订阅所在的 Azure 帐户名称以及导出日期。
