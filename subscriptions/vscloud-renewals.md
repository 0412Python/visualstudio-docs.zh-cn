# <a name="buying-visual-studio-cloud-subscriptions---renewals"></a>购买 Visual Studio 云订阅 - 续订

系统将自动续订云订阅，续订费用将显示在你的 Azure 帐单中。  以下是可能遇到的一些示例。
## <a name="monthly-subscriptions"></a>月度订阅
Microsoft 在第一个月份将按比例收费。  

示例：如果你在 15 日购买，则你的第一个帐单将按约 50% 计费。  然后，后续月份都将会在每月的第 1 天收取整月费用。

## <a name="annual-subscriptions"></a>年度订阅
年度订阅将在购买日期周年后的第一个月续订。  第一次续订后，续订日期将保持一致。

示例：对于 2018 年 1 月 15 日购买的订阅，其续订日期为 2019 年 2 月 1 日。

## <a name="faq"></a>常见问题解答：
### <a name="q--what-if-i-dont-want-to-auto-renew"></a>问：如果我不想自动续订怎么办？
答：为避免产生续订费用，你需要在自动续订日期之前取消你的 Visual Studio 云订阅。  如果决定取消你的订阅，你仍然可以在订阅的剩余时间里访问你的权益。
