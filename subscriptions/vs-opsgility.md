# <a name="activating-the-opsgility-benefit-in-visual-studio-subscriptions"></a>激活 Visual Studio 订阅中的 Opsgility 权益

企业订阅包含对按需培训的一年使用权限。  它还提供八五折优惠的教员指导式培训，以及联机培训、联机动手实验室、证书路径等。  

1.  要使用 Opsgility 权益，请单击权益磁贴底部的“获取代码”链接。   你将收到一条通知消息，指示已成功检索到优惠券代码。  单击“激活”，然后继续。

![Opsgility 权益磁贴](_img\vs-opsgility\vs-opsgility-tile.png)

2.  随即重定向到 [Visual Studio Enterprise 的 Opsgility 联机培训](https://www.opsgility.com/vse)登录页。  （你可能会获得与支持专家开始聊天会话的机会。  如果不希望开始聊天，可以将该窗口最小化。）填写所需的联系人信息，然后单击“注册”。  

![Opsgility 权益登录页](_img\vs-opsgility\vs-opsgility-landing-cropped.png)

3.  就这么简单！  准备开始学习吧。  为帮助你入门，页面左侧提供了建议的学习路径列表。  请尽情体验吧！

![Opsgility 权益主页](_img\vs-opsgility\vs-opsgility-home-cropped.png)
