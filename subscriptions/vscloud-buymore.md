# <a name="buying-visual-studio-cloud-subscriptions---buying-additional-subscriptions"></a>购买 Visual Studio 云订阅 - 购买更多订阅

若要购买更多 Visual Studio 云订阅，请使用有权访问 Azure 订阅用于原始购买的帐户登录到 Visual Studio [订阅管理门户](https://manage.visualstudio.com/)。

1.  在左上角，验证已选择相应的 Azure 订阅。
2.  在左侧导航栏，在希望购买的订阅下选择“购买更多”。
3.  增加订阅数至所需数量，然后单击“继续”。
4.  查看订单，然后单击“确认”。
5.  在返回到“订阅管理”门户后，应会注意到可用订阅数量的增加。

要获得更多帮助，请参阅有关分配和管理订阅的[文档](/visualstudio/subscriptions/)

## <a name="faq"></a>常见问题解答：
### <a name="q--how-long-do-i-have-to-wait-before-i-can-assign-subscriptions"></a>问：需要等待多长时间才可以分配订阅？
答：订阅可在“订阅管理”门户中立即进行分配。
