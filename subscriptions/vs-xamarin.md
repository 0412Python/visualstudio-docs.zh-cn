# <a name="activating-the-xamarin-university-training-benefit-in-visual-studio-subscriptions"></a>激活 Visual Studio 订阅中的 Xamarin University 培训权益

借助专家入门视频（精选课堂视频和材料）在 C# 中构建本机 iOS 和 Android 应用。  

1.  若要使用 Xamarin University 培训权益，请单击权益磁贴底部的“获取代码”链接。   你将收到一条通知消息，指示已成功检索到优惠券代码。  单击“激活”，然后继续。

    ![Xamarin University 培训权益磁贴](_img\vs-xamarin\vs-xamarin-tile.png)

2.  在注册页上填写联系人信息，单击复选框，接受使用条款和隐私声明，然后单击“激活权益”。 

    ![Xamarin University 培训权益注册](_img\vs-xamarin\vs-xamarin-registration-resized.png)



3.  就这么简单！  现在，你已激活 Xamarin University 培训权益，请重定向到 Microsoft Visual Studio 页的 Xamarin University 移动培训，然后即可准备开始学习。  

    ![Xamarin University 培训权益主页](_img\vs-xamarin\vs-xamarin-home-resized.png)
