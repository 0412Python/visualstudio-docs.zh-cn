# <a name="buying-visual-studio-cloud-subscriptions---cancelling-subscriptions"></a>购买 Visual Studio 云订阅 - 取消订阅

若要取消 Visual Studio 云订阅，请使用有权限访问 Azure 订阅用于原始购买的帐户登录到 Visual Studio [订阅管理门户](https://manage.visualstudio.com/)。
1.  在左上角，验证已选择相应的 Azure 订阅。
2.  在左侧导航栏，在希望取消的订阅下选择“减少”。
3.  你将会看到一个窗口可调整与你帐户关联的订阅数量。
4.  减少订阅数至所需数量或者零 (0) 以取消，然后单击“继续”。
5.  查看订单，单击“确认”，这样就可以了！
## <a name="faq"></a>常见问题解答：
### <a name="q--can-i-still-use-a-subscription-after-i-cancel"></a>问：取消后是否仍然可以使用订阅？
答：是的，你仍然可以在订阅的剩余时间里访问你的权益。
