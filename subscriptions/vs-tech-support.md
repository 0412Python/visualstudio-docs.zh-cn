# <a name="activating-the-technical-support-benefit-in-visual-studio-subscriptions"></a>激活 Visual Studio 订阅中的技术支持权益

解决非生产环境中的中断修复问题。  （注意：必须激活权益，然后才能提交支持请求。）Visual Studio Enterprise 标准年度云订阅接收四个事件，但 Visual Studio Professional 标准年度云订阅接收两个事件。  

1.  若要开始使用技术支持权益，请单击“技术支持”磁贴上的“激活”。 

![技术支持权益磁贴](_img\vs-tech-support\vs-tech-support-tile.png)

2.  在[“技术支持”页](https://www.visualstudio.com/my/technical-support-vse)上，将会看到激活技术支持权益的说明。  你需要一个合同 ID 和访问 ID，我们的技术支持团队可为你设置。    
3.  如果尚未设置合同 ID 和访问 ID，请联系所在国家/地区的 [Microsoft 技术支持号码](http://support.microsoft.com/gp/customer-service-phone-numbers)，并告知他们需要设置 Visual Studio 订阅的技术支持合同。  这只需几分钟的时间。  他们将为你提供合同 ID 和访问 ID，你可能需要记录这些信息以供将来参考。  你还将收到一封包含该信息的电子邮件。  
 
4.  若要激活欧洲、中东和非洲的支持帐户，请从[此列表](http://support.microsoft.com/activatesupport)选择所在国家/地区。   对于其他的所有地理区域，请跳到步骤 7。
 
5.  “激活支持合同”页的说明将引导你完成该过程，该过程可能会因国家/地区而有所不同。  一般情况下，这些步骤包括下载和完善电子注册表单，并通过电子邮件发送给 Microsoft。  帐户处理完成并准备就绪之后，你将收到一封包含帐户详细信息的电子邮件。  
 
6.  设置好帐户之后，请转到步骤 8，了解如何联机提交支持事件。  
7.  在欧洲、中东和非洲以外的所有地理区域中，都可通过呼叫所在国家/地区的[技术支持团队](http://support.microsoft.com/gp/customer-service-phone-numbers)来设置帐户。  若要查找所在国家/地区的正确电话号码，只需单击与所在国家/地区名称对应的字母即可。  你将看到以该字母开头的国家/地区列表，以及电话号码和这些地方支持的语言。  如果未列出所在国家/地区，请选择提供首选语言支持的最邻近的国家/地区。  
 
8.  呼叫时，需要提供基本的联系信息，如姓名、电话号码和电子邮件地址。  还需要提供订阅 ID。  若要查找订阅 ID，请单击“[订阅者门户](https://my.visualstudio.com)”上的“[订阅](https://my.visualstudio.com/subscriptions)”选项卡。  此时，你将看到为订阅列出的订阅 ID。

![技术支持权益查找订阅 ID](_img\vs-tech-support\vs-tech-support-subID-cropped.png)

9.  电话呼叫结束后，你将收到一封确认电子邮件，其中包含为你创建的合同 ID 和访问 ID。

10. 合同 ID 和访问 ID 设置完成后，如果需要支持，可以呼叫同一个技术支持电话或[联机提交支持请求](http://support.microsoft.com/oas/)来设置帐户。  如果希望通过电话获得支持，仍然可以联机提交请求，Microsoft 支持部门专业人员之后会给你回电。

11. 若要[联机提交事件](http://support.microsoft.com/oas/)，请指定需要帮助的产品。  可从各种列表中进行选择，也可按名称进行搜索。  例如，如果要为 Visual Studio Enterprise 2017 提交事件，可以单击“开发人员工具”组中的“Visual Studio”。 
 
![技术支持权益选择事件产品](_img\vs-tech-support\vs-tech-support-select-product.png)
12. 单击 Visual Studio 产品列表中的“Visual Studio Enterprise 2017”。 
 
13. 在“创建事件”页上，从下拉列表中选择合适的问题类型。  对于本示例，我们选择“集成开发环境(IDE)”。

![技术支持权益创建事件](_img\vs-tech-support\vs-tech-support-create-incident.png)

14. 然后选择最符合你的疑问/问题的“类别”。  对于本示例，我们选择“架构师工具”。
15. 单击“启动请求”。 
 
16. 在下一页中，需要选择支付选项。  
- 选择“使用我的 MSDN/Visual Studio 订阅”。 
- 单击“使用其他合同”
- 输入访问 ID 和合同 ID
- 单击“添加”

17. 添加支持合同后，该合同将出现在专业技术支持合同列表中。  选择该合同，然后单击“继续”。
 
![技术支持权益支付](_img\vs-tech-support\vs-tech-support-payment.png)

18. 提供联系信息和首选联系方式，然后单击“继续”。  
 
19. 为支持事件提供一个标题，然后对问题进行描述。  

20. 回答一系列简短的问题来确定问题的严重性。  

21. 单击复选框接受服务条款。

22. 单击“提交”以提交问题。  
 
23. 此时，将看到事件已成功提交的确认消息。  分配给该事件的代理将会使用所选联系方式与你联系，以解决该问题。  非关键问题的响应时间是八个小时。 他们将为你提供服务请求号码，并协助你解决该问题。
