# <a name="jsfinalizecallback-typedef"></a>JsFinalizeCallback Typedef

## <a name="syntax"></a>语法  
  
```  
typedef void (CALLBACK *JsFinalizeCallback)(  
   _In_opt_ void *data  
);  
```  
  
#### <a name="parameters"></a>参数  
 数据
