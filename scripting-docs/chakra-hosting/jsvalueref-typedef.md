# <a name="jsvalueref-typedef"></a>JsValueRef Typedef
对 JavaScript 值的引用。  
  
## <a name="syntax"></a>语法  
  
```  
typedef JsRef JsValueRef;  
```  
  
## <a name="remarks"></a>备注  
 JavaScript 值为以下类型的值之一：未定义、Null、布尔值、字符串、数字或对象。  
  
## <a name="requirements"></a>要求  
 **标头：** jsrt.h
