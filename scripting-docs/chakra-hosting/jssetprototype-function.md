# JsSetPrototype 函数
设置对象的原型。  
  
## 语法  
  
```  
STDAPI_(JsErrorCode) JsSetPrototype(  
   _In_ JsValueRef object,  
   _In_ JsValueRef prototypeObject  
);  
```  
  
#### 参数  
 `object`  
 要更改其原型的对象。  
  
 `prototypeObject`  
 对象的新原型。  
  
## 返回值  
 如果该操作成功，则为代码 `JsNoError`，否则为失败代码。
