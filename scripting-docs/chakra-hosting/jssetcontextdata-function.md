# JsSetContextData 函数
设置 JsrtContext 的内部数据。  
  
## 语法  
  
```  
STDAPI_(JsErrorCode) JsSetContextData(  
  _In_ JsContextRef context,  
  _In_ void *data  
);  
  
```  
  
#### 参数  
 `context`  
 要在其中设置数据的上下文。  
  
 `data`  
 指向要设置的数据的指针。  
  
## 返回值  
 如果该操作成功，则为代码 `JsNoError`，否则为失败代码。  
  
## 备注  
 需要活动脚本上下文。  
  
## 要求  
 **标头：**jsrt.h
