# JsRuntimeVersion 枚举
运行时的版本。  
  
## 语法  
  
```  
enum JsRuntimeVersion;  
```  
  
## 成员  
  
### 值  
  
|名称|描述|  
|--------|--------|  
|`JsRuntimeVersion10`|创建 IE10 版本的运行时。|  
|`JsRuntimeVersion11`|创建 IE11 版本的运行时。|  
|`JsRuntimeVersionEdge`|在运行时创建计算机上存在的最高版本的运行时。|  
  
## 要求  
 **标头：**jsrt.h
