# <a name="jsruntimeversion-enumeration"></a>JsRuntimeVersion 枚举
运行时的版本。  
  
## <a name="syntax"></a>语法  
  
```  
enum JsRuntimeVersion;  
```  
  
## <a name="members"></a>成员  
  
### <a name="values"></a>值  
  
|名称|描述|  
|----------|-----------------|  
|`JsRuntimeVersion10`|创建 IE10 版本的运行时。|  
|`JsRuntimeVersion11`|创建 IE11 版本的运行时。|  
|`JsRuntimeVersionEdge`|在运行时创建计算机上存在的最高版本的运行时。|  
  
## <a name="requirements"></a>要求  
 **标头：** jsrt.h
