# JsGetUndefinedValue 函数
获取当前脚本上下文中的 `undefined` 值。  
  
## 语法  
  
```  
STDAPI_(JsErrorCode) JsGetUndefinedValue(  
   _Out_ JsValueRef *undefinedValue  
);  
```  
  
#### 参数  
 `undefinedValue`  
 `undefined` 值。  
  
## 返回值  
 如果该操作成功，则为代码 `JsNoError`，否则为失败代码。  
  
## 备注  
 需要活动脚本上下文。  
  
## 要求  
 **标头：**jsrt.h
