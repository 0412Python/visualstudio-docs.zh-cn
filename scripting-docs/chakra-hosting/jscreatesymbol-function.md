# JsCreateSymbol 函数
创建 JavaScript 符号。  
  
## 语法  
  
```  
STDAPI_(JsErrorCode) JsCreateSymbol(  
   _In_ JsValueRef description,  
   _Out_ JsValueRef *result  
);  
```  
  
#### 参数  
 `description`  
 符号的字符串说明。  可以为 null。  
  
 `result`  
 新的符号。  
  
## 返回值  
 如果该操作成功，则为代码 `JsNoError`，否则为失败代码。
