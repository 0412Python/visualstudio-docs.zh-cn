# JsGetPropertyIdType 函数
获取属性的类型。  
  
## 语法  
  
```  
STDAPI_(JsErrorCode) JsGetPropertyIdType(  
   _In_ JsPropertyIdRef propertyId,  
   _Out_ JsPropertyIdType* propertyIdType  
);  
```  
  
#### 参数  
 `propertyId`  
 要获取其类型的属性 ID。  
  
 `propertyIdType`  
 给定属性 ID 的 `JsPropertyIdType`。  
  
## 返回值  
 如果该操作成功，则为代码 `JsNoError`，否则为失败代码。
