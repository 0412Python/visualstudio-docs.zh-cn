# JsPropertyIdRef Typedef
一个属性标识符。  
  
## 语法  
  
```  
typedef JsRef JsPropertyIdRef;  
```  
  
## 备注  
 使用属性标识符（而不是字符串）来引用 JavaScript 对象的属性。  
  
## 要求  
 **标头：**jsrt.h
