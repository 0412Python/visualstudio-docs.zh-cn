# <a name="jspropertyidref-typedef"></a>JsPropertyIdRef Typedef
一个属性标识符。  
  
## <a name="syntax"></a>语法  
  
```  
typedef JsRef JsPropertyIdRef;  
```  
  
## <a name="remarks"></a>备注  
 使用属性标识符（而不是字符串）来引用 JavaScript 对象的属性。  
  
## <a name="requirements"></a>要求  
 **标头：** jsrt.h
