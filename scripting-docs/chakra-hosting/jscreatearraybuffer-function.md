# JsCreateArrayBuffer 函数
创建 JavaScript `ArrayBuffer` 对象。  
  
## 语法  
  
```  
STDAPI_(JsErrorCode) JsCreateArrayBuffer(  
   _In_ unsigned int byteLength,  
   _Out_ JsValueRef *result  
);  
```  
  
#### 参数  
 `byteLength`  
 ArrayBuffer 中的字节数。  
  
 `result`  
 新的 ArrayBuffer 对象。  
  
## 返回值  
 如果该操作成功，则为代码 `JsNoError`，否则为失败代码。
