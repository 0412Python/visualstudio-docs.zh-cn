# <a name="valueof-method-number"></a>valueOf 方法（数字）
返回指定数目的基元值。  
  
## <a name="syntax"></a>语法  
  
```  
  
number.valueOf()  
```  
  
#### <a name="parameters"></a>参数  
 此方法没有任何参数。  
  
## <a name="return-value"></a>返回值  
 返回数。  
  
## <a name="remarks"></a>备注  
 在下面的示例中，实例化 number 对象是此方法的返回值相同。  
  
```JavaScript  
var num = 1234;  
var s = num.valueOf();  
  
if (num === s)  
document.write("same");  
else  
document.write("different");  
  
// Output:  
// same  
  
```
