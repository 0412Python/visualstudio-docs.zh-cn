# toDateString 方法 (Date) (JavaScript)
以字符串值的形式返回一个日期。  
  
## 语法  
  
```  
  
objDate.toDateString( )  
```  
  
## 备注  
 必需的 *objDate* 引用是 `Date` 对象。  
  
 `toDateString` 方法返回一个包含日期的字符串值，该日期位于当前时区中，以一种方便、易读的格式表示。
