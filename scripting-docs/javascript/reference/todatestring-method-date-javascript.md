# <a name="todatestring-method-date-javascript"></a>toDateString 方法 (Date) (JavaScript)
返回一个字符串值作为日期。  
  
## <a name="syntax"></a>语法  
  
```  
  
objDate.toDateString( )  
```  
  
## <a name="remarks"></a>备注  
 所需*objDate*引用是`Date`对象。  
  
 `toDateString`方法返回包含日期，采用当前所在的时区，以一种方便的字符串值易读的格式。
