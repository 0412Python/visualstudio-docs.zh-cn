# Math.floor 函数 (JavaScript)
返回小于或等于其数值参数的最大整数。  
  
## 语法  
  
```  
  
Math.floor(number)   
```  
  
## 备注  
 必需的 *number*  参数是一个数值表达式。  
  
 返回值为小于或等于其数值参数的最大整数值。  
  
 **适用于**：[Math 对象](../../javascript/reference/math-object-javascript.md)
