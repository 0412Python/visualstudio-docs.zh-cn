# <a name="mathfloor-function-javascript"></a>Math.floor 函数 (JavaScript)
返回小于或等于其数值自变量的最大整数。  
  
## <a name="syntax"></a>语法  
  
```  
  
Math.floor(number)   
```  
  
## <a name="remarks"></a>备注  
 所需*数*自变量是一个数值表达式。  
  
 返回的值等于最大整数的整数值小于或等于其数值自变量。  
  
 **适用于**: [Math 对象](../../javascript/reference/math-object-javascript.md)
