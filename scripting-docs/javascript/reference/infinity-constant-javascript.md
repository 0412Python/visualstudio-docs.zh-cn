# <a name="infinity-constant-javascript"></a>Infinity 常量 (JavaScript)
一个数字，将大于最大浮点数。 返回的初始值为`Number.POSITIVE_INFINITY`。 负无穷大 (-无穷大) 小于最小的浮点数。  
  
## <a name="syntax"></a>语法  
  
```  
Infinity   
```  
  
## <a name="remarks"></a>备注  
 `Infinity`常量为属于`Global`对象，并可在初始化脚本引擎时。
