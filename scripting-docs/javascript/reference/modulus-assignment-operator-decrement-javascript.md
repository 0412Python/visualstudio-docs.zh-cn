# 取模赋值运算符 (JavaScript)
将变量值除以表达式值，并将余数赋给变量。  
  
## 语法  
  
```  
  
result %= expression  
```  
  
## 参数  
 `result`  
 任何变量。  
  
 `expression`  
 任何数值表达式。  
  
## 备注  
 使用 `%=` 运算符与指定以下内容完全相同：  
  
```  
result = result % expression  
```
