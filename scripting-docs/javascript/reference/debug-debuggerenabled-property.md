# <a name="debugdebuggerenabled-property"></a>Debug.debuggerEnabled 属性
确定是否为脚本上下文启用调试。 调试可能要启用或禁用已连接调试器。  
  
## <a name="syntax"></a>语法  
  
```  
  
var dbgEnabled = Debug.debuggerEnabled;  
```
