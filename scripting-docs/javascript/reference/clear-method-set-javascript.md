# <a name="clear-method-set-javascript"></a>clear 方法 (Set) (JavaScript)
从集内移除所有元素。  
  
## <a name="syntax"></a>语法  
  
```JavaScript  
setObj.clear()  
```  
  
#### <a name="parameters"></a>参数  
 `setObj`  
 必需。 要清除的集。
