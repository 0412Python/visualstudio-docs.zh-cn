# resolvedOptions 方法 (Intl.Collator)
返回包含排序程序对象的属性和值的对象。  
  
## 语法  
  
```  
collatorObj.resolvedOptions()  
```  
  
#### 参数  
 `collatorObj`  
 通过其检索信息的 `Collator` 对象。  
  
## 备注  
 对应于 `Collator` 对象的计算属性的返回对象的属性。
