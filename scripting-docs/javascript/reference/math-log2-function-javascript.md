# Math.log2 函数 (JavaScript)
返回数字以 2 为底的对数。  
  
## 语法  
  
```  
Math.log2(x)   
```  
  
## 返回值  
 如果 `x` 为 `NaN`，则此函数返回 `NaN`。  如果 `x` 小于零 \(0\)，则此函数返回 `NaN`。
