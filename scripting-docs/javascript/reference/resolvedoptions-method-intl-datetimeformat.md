# resolvedOptions 方法 (Intl.DateTimeFormat)
返回包含日期\/时间格式化程序对象的属性和值的对象。  
  
## 语法  
  
```  
dateTimeFormatObj.resolvedOptions()  
```  
  
#### 参数  
 `dateTimeFormatObj`  
 通过其检索信息的 `DateTimeFormat` 对象。  
  
## 备注  
 所返回对象的属性对应于 `DateTimeFormat` 的计算属性。
