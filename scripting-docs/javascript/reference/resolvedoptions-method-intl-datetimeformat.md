# <a name="resolvedoptions-method-intldatetimeformat"></a>resolvedOptions 方法 (Intl.DateTimeFormat)
返回包含日期/时间格式化程序对象的属性和值的对象。  
  
## <a name="syntax"></a>语法  
  
```  
dateTimeFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>参数  
 `dateTimeFormatObj`  
 `DateTimeFormat`要从中检索信息对象。  
  
## <a name="remarks"></a>备注  
 返回的对象的属性对应的计算属性`DateTimeFormat`对象。
