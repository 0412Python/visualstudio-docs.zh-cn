# <a name="valueof-method-array"></a>valueOf 方法 (Array)
返回指定对象的基元值。  
  
## <a name="syntax"></a>语法  
  
```  
  
array.valueOf()  
```  
  
#### <a name="parameters"></a>参数  
 此方法没有任何参数。  
  
## <a name="return-value"></a>返回值  
 返回的数组实例。  
  
## <a name="remarks"></a>备注  
 在下面的示例中，实例化的数组对象是此方法的返回值相同。  
  
```JavaScript  
var arr = [1, 2, 3, 4];  
var s = arr.valueOf();  
  
if (arr === s)  
document.write("same");  
else  
document.write("different");  
  
// Output:  
// same  
  
```
