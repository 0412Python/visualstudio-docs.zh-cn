# <a name="addition-operator--javascript"></a>加法运算符 (+) (JavaScript)
一个数值表达式的值添加到另一个，或串联两个字符串。  
  
## <a name="syntax"></a>语法  
  
```  
  
result = expression1 + expression2  
```  
  
## <a name="parameters"></a>参数  
 `result`  
 任何变量。  
  
 `expression1`  
 任何表达式。  
  
 `expression2`  
 任何表达式。  
  
## <a name="remarks"></a>备注  
 两个表达式的类型确定的行为 **+** 运算符。  
  
|如果|Then|  
|--------|----------|  
|两个表达式均数值或布尔|添加|  
|这两个表达式均为字符串|连接|  
|一个表达式是数值和另一种是一个字符串|连接|
