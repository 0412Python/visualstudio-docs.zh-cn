# Math.asinh 函数 (JavaScript)
返回数字的反双曲正弦。  
  
## 语法  
  
```  
  
Math.asinh(x)   
```  
  
## 备注  
 如果 `x` 为 `NaN`，则结果为 `NaN`。  
  
 **适用于**：[Math 对象](../../javascript/reference/math-object-javascript.md)
