# valueOf 方法 (Map) (JavaScript)
返回指定对象的原始值。  
  
## 语法  
  
```javascript  
mapObj.valueOf()  
```  
  
#### 参数  
 `mapObj`  
 必需。  `Map` 对象。  
  
## 属性值\/返回值  
 返回映射实例。
