# <a name="isfinite-function-javascript"></a>isFinite 函数 (JavaScript)
确定提供的数量是有限的。  
  
## <a name="syntax"></a>语法  
  
```  
isFinite(number)   
```  
  
## <a name="remarks"></a>备注  
 所需`number`自变量是任何数字值。  
  
 `isFinite`函数返回`true`如果`number`是任何值，而不`NaN`、 负无穷大或正无穷。 在这三种情况下，它将返回**false**。
