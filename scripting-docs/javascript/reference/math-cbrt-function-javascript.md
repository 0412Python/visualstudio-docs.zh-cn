# Math.cbrt 函数 (JavaScript)
返回数字的立方根。  
  
## 语法  
  
```  
  
Math.cbrt(  
number  
)   
```  
  
## 备注  
 如果 `number` 为 `NaN`，则结果将为 `NaN`。
