# <a name="mathcbrt-function-javascript"></a>Math.cbrt 函数 (JavaScript)
返回数字的立方根。  
  
## <a name="syntax"></a>语法  
  
```  
  
Math.cbrt(  
number  
)   
```  
  
## <a name="remarks"></a>备注  
 如果 `number` 为 `NaN`，则结果将为 `NaN`。
