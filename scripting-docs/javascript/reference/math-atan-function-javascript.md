# Math.atan 函数 (JavaScript)
返回一个数的反正切值。  
  
## 语法  
  
```  
  
Math.atan(number)   
```  
  
## 备注  
 必需的 `number` 参数是反正切值所需的数值表达式。  
  
 返回值是 `number` 参数的反正切值（以弧度为单位）。  
  
 **适用于**：[Math 对象](../../javascript/reference/math-object-javascript.md)
