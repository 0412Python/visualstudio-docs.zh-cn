# Global 对象 (JavaScript)
内部对象，其用途是将全局函数和常量收集到一个对象中。  
  
## 备注  
 `Global` 对象从不直接使用，并且不能使用 `new` 运算符来创建。  它在初始化脚本引擎时创建，从而使其函数和常量可以直接使用。
