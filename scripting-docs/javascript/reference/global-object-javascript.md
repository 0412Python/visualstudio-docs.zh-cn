# <a name="global-object-javascript"></a>Global 对象 (JavaScript)
内部对象，其用途是收集到一个对象的全局函数和常量。  
  
## <a name="remarks"></a>备注  
 `Global`对象永远不会直接使用，且无法使用创建`new`运算符。 初始化脚本引擎时，因此使其函数和常量可立即，则创建它。
