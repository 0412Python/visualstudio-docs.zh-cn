# valueOf 方法 (WeakMap) (JavaScript)
返回指定对象的原始值。  
  
## 语法  
  
```javascript  
weakmapObj.valueOf()  
```  
  
#### 参数  
 `weakmapObj`  
 必需。  `WeakMap` 对象。  
  
## 属性值\/返回值  
 返回 `WeakMap` 实例。
