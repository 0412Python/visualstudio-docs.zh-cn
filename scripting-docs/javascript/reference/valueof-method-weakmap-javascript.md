# <a name="valueof-method-weakmap-javascript"></a>valueOf 方法 (WeakMap) (JavaScript)
返回指定对象的基元值。  
  
## <a name="syntax"></a>语法  
  
```JavaScript  
weakmapObj.valueOf()  
```  
  
#### <a name="parameters"></a>参数  
 `weakmapObj`  
 必需。 一个 `WeakMap` 对象。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 返回`WeakMap`实例。
