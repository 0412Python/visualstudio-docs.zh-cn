# Math.sin 函数 (JavaScript)
返回一个数的正弦值。  
  
## 语法  
  
```  
  
Math.sin(radians)   
```  
  
## 备注  
 `radians` 参数是包含测量的角度（以弧度为单位）的数值表达式。  
  
 返回值为数值参数的正弦。
