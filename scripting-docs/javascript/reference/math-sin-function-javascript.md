# <a name="mathsin-function-javascript"></a>Math.sin 函数 (JavaScript)
返回数字的正弦值。  
  
## <a name="syntax"></a>语法  
  
```  
  
Math.sin(radians)   
```  
  
## <a name="remarks"></a>备注  
 `radians`自变量是一个包含以弧度计量的角度的数值表达式。  
  
 返回值是数值参数的正弦值。
