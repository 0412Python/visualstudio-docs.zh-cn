# <a name="mathlog1p-function-javascript"></a>Math.log1p 函数 (JavaScript)
返回 1 加上一个数字的的自然对数。  
  
## <a name="syntax"></a>语法  
  
```  
Math.log1p(x)   
```  
  
## <a name="return-value"></a>返回值  
 如果 `x` 为 `NaN`，则此函数返回 `NaN`。 如果 `x` 小于零 (0)，则此函数返回 `NaN`。
