# size 属性 (Map) (JavaScript)
返回映射中的元素数。  
  
## 语法  
  
```javascript  
sizeVar = mapObj.size  
```  
  
#### 参数  
 `sizeVar`  
 必需。  任何数量。  
  
 `mapObj`  
 必需。  任意 `Map` 对象。
