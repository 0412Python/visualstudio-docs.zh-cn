# <a name="valueof-method-set-javascript"></a>valueOf 方法 (Set) (JavaScript)
返回指定对象的基元值。  
  
## <a name="syntax"></a>语法  
  
```JavaScript  
setObj.valueOf()  
```  
  
#### <a name="parameters"></a>参数  
 `setObj`  
 必需。 一个 `Set` 对象。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 返回集实例。
