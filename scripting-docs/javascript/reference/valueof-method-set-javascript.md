# valueOf 方法 (Set) (JavaScript)
返回指定对象的原始值。  
  
## 语法  
  
```javascript  
setObj.valueOf()  
```  
  
#### 参数  
 `setObj`  
 必需。  `Set` 对象。  
  
## 属性值\/返回值  
 返回集实例。
