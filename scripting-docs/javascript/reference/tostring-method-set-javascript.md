# toString 方法 (Set) (JavaScript)
返回集的字符串表示形式。  
  
## 语法  
  
```javascript  
setObj.toString()  
```  
  
#### 参数  
 `setObj`  
 必需。  `Set` 对象。  
  
## 属性值\/返回值  
 集的字符串表示形式。
