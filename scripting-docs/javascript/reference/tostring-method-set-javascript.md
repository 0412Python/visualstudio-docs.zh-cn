# <a name="tostring-method-set-javascript"></a>toString 方法 (Set) (JavaScript)
返回的字符串表示形式一组。  
  
## <a name="syntax"></a>语法  
  
```JavaScript  
setObj.toString()  
```  
  
#### <a name="parameters"></a>参数  
 `setObj`  
 必需。 一个 `Set` 对象。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 集的字符串表示形式。
