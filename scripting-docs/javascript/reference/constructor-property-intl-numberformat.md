# <a name="constructor-property-intlnumberformat"></a>constructor 属性 (Intl.NumberFormat)
指定创建一个格式化程序的函数。  
  
## <a name="syntax"></a>语法  
  
```JavaScript  
numberFormatter.constructor  
```  
  
## <a name="remarks"></a>备注  
 所需`numberFormatter`是格式化程序的名称。  
  
 `constructor` 属性是具有原型的每个对象的原型的成员。 它包括所有内部 JavaScript 对象，`Global` 和 `Math` 对象除外。 `constructor` 属性包含对构造特定对象实例的函数的引用。
