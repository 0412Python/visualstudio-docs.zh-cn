# <a name="size-property-set-javascript"></a>size 属性 (Set) (JavaScript)
返回一组中的元素的数目。  
  
## <a name="syntax"></a>语法  
  
```JavaScript  
sizeVar = setObj.size  
```  
  
#### <a name="parameters"></a>参数  
 `sizeVar`  
 必需。 任意数值。  
  
 `setObj`  
 必需。 任意 `Set` 对象。
