# size 属性 (Set) (JavaScript)
返回集中的元素数。  
  
## 语法  
  
```javascript  
sizeVar = setObj.size  
```  
  
#### 参数  
 `sizeVar`  
 必需。  任何数量。  
  
 `setObj`  
 必需。  任意 `Set` 对象。
