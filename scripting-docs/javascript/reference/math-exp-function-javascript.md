# Math.exp 函数 (JavaScript)
返回 e（自然对数的底）的若干次幂。  
  
## 语法  
  
```  
  
Math.exp(number)   
```  
  
## 备注  
 必需的 `number` 参数是表示 *e* 的次幂的数值表达式。  
  
 该返回值是数字。  常数 *e* 是欧拉数，它约等于 2.71828，而 `number` 是提供的参数。  
  
 **适用于**：[Math 对象](../../javascript/reference/math-object-javascript.md)
