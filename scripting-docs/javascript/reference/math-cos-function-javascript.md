# <a name="mathcos-function-javascript"></a>Math.cos 函数 (JavaScript)
返回数字的余弦值。  
  
## <a name="syntax"></a>语法  
  
```  
  
Math.cos(radians)   
```  
  
## <a name="remarks"></a>备注  
 所需`radians`自变量是一个包含以弧度计量的角度的数值表达式。  
  
 返回值的数值自变量的余弦值`radians`。  
  
 **适用于**: [Math 对象](../../javascript/reference/math-object-javascript.md)
