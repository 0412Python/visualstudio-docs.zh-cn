# Math.cos 函数 (JavaScript)
返回数的余弦值。  
  
## 语法  
  
```  
  
Math.cos(radians)   
```  
  
## 备注  
 必需的 `radians` 参数是包含测量的角度（以弧度为单位）的数值表达式。  
  
 返回值为 `radians` 的数值参数的余弦值。  
  
 **适用于**：[Math 对象](../../javascript/reference/math-object-javascript.md)
