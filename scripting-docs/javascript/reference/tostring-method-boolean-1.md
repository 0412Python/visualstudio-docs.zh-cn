# <a name="tostring-method-boolean-1"></a>toString 方法 （布尔） 1
返回对象的字符串表示形式。  
  
## <a name="syntax"></a>语法  
  
```  
  
boolean.toString()  
```  
  
## <a name="parameters"></a>参数  
 `boolean`  
 必需。 要为其获取的字符串表示一个对象。  
  
## <a name="return-value"></a>返回值  
 如果布尔值为`true`，返回"true"。 否则，返回"false"。  
  
## <a name="remarks"></a>备注  
  
## <a name="example"></a>示例  
 下面的示例演示如何使用**toString**方法。  
  
```JavaScript  
var s = new Boolean(0);  
document.write(s.toString());  
  
// Output: false;  
  
```
