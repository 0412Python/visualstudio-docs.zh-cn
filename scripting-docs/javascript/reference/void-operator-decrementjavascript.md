# <a name="void-operator-javascript"></a>void 运算符 (JavaScript)
防止表达式返回值。  
  
## <a name="syntax"></a>语法  
  
```  
void expression   
```  
  
## <a name="remarks"></a>备注  
 `expression`自变量是任何有效[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]表达式。  
  
 `void`运算符计算它的表达式并返回`undefined`。 它可在其中应计算表达式，但结果不希望对脚本的其余部分可见的情况下。
