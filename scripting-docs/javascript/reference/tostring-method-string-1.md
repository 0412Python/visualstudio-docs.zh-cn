# toString 方法 (String)
返回字符串的字符串表示形式。  
  
## 语法  
  
```  
  
string.toString()  
```  
  
## 参数  
 `string`  
 必需。  要表示为字符串的数组。  
  
## 返回值  
 字符串的字符串表示形式。  
  
## 示例  
 下面的示例阐释了如何将 **toString** 方法用于字符串。  
  
```javascript  
var string = "this is a test";  
var strStr = string.toString();  
document.write(strStr);  
  
// Output: this is a test  
  
```
