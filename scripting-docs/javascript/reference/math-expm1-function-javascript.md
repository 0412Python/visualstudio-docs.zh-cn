# Math.expm1 函数 (JavaScript)
返回 e（自然对数的底）的乘幂数减去 1 的结果。  
  
## 语法  
  
```  
  
Math.expm1(number)   
```  
  
## 备注  
 所需 `number` 参数是一个数值表达式，表示 e 的幂。  
  
 返回值是一个数字。  常量 e 是欧拉数，约等于 2.71828，`number` 是提供的参数。  
  
 **适用于**：[Math 对象](../../javascript/reference/math-object-javascript.md)
