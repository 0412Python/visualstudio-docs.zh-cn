# Math.sinh 函数 (JavaScript)
返回数字的反双曲正弦。  
  
## 语法  
  
```  
  
Math.sinh(x)   
```  
  
## 备注  
 返回值与 `(exp(x) - exp(-x))/2` 相同。  如果 x 为 `NaN`，则结果为 `NaN`。
