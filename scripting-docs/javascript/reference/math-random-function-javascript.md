# Math.random 函数 (JavaScript)
返回介于 0 和 1 之间的伪随机数。  
  
## 语法  
  
```  
  
Math.random( )  
  
```  
  
## 备注  
 生成的伪随机数介于 0（含）和 1（不含）之间，即返回值可以是 0，但它永远小于 1。  在第一次加载 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 时，自动给随机数生成器提供种子。
