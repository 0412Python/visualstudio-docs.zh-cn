# <a name="constructor-property-weakmap"></a>constructor 属性 (WeakMap)
指定创建的函数`WeakMap`对象。  
  
## <a name="syntax"></a>语法  
  
```JavaScript  
weakmap.constructor  
```  
  
## <a name="remarks"></a>备注  
 所需`weakmap`是的名称`WeakMap`对象。  
  
 `constructor` 属性是具有原型的每个对象的原型的成员。 它包括所有内部 JavaScript 对象，`Global` 和 `Math` 对象除外。 `constructor` 属性包含对构造特定对象实例的函数的引用。
