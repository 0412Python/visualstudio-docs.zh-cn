# Math.trunc 函数 (JavaScript)
返回数字的整数部分，删除任何小数数字。  
  
## 语法  
  
```vb  
Math.trunc(number)  
```  
  
## 备注  
 所需 `number` 参数是一个数值表达式，它需要被截断的整数。  
  
 如果 `number` 为 `NaN`，则返回 `NaN`。
