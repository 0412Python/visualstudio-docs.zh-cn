# <a name="length-property-string-javascript"></a>length 属性 (String) (JavaScript)
返回 `String` 对象的长度。  
  
> [!WARNING]
>  JavaScript 字符串是不可变，，因此不能修改字符串的长度。  
  
## <a name="syntax"></a>语法  
  
```  
  
      strVariable.length  
"String Literal".length   
```  
  
## <a name="remarks"></a>备注  
 `length`属性包含一个整数，指示中的字符数`String`对象。 中的最后一个字符`String`对象的索引为我`length`-1。
