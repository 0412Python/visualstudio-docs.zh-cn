# <a name="mathasin-function-javascript"></a>Math.asin 函数 (JavaScript)
返回数字的反正弦值。  
  
## <a name="syntax"></a>语法  
  
```  
  
Math.asin(number)   
```  
  
## <a name="remarks"></a>备注  
 所需`number`自变量是数值表达式，它需要反正弦值。  
  
 返回值的反正弦`number`自变量，以弧度为单位。  
  
 **适用于**: [Math 对象](../../javascript/reference/math-object-javascript.md)
