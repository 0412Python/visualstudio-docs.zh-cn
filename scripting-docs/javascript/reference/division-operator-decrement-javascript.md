# <a name="division-operator---javascript"></a>除法运算符 （-） (JavaScript)
将两个表达式的值。  
  
## <a name="syntax"></a>语法  
  
```  
  
result = number1 / number2   
```  
  
## <a name="parameters"></a>参数  
 `result`  
 任何数值变量。  
  
 `number1`  
 任何数值表达式。  
  
 `number2`  
 任何数值表达式。
