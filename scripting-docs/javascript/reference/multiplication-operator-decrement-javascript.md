# 乘法运算符 (*) (JavaScript)
将两个表达式的值相乘。  
  
## 语法  
  
```  
  
result = number1*number2  
```  
  
## 参数  
 *result*  
 任何变量。  
  
 `number1`  
 任何表达式。  
  
 `number2`  
 任何表达式。
