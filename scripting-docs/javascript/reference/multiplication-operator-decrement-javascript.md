# <a name="multiplication-operator--javascript"></a>乘法运算符 (*) (JavaScript)
将值与两个表达式相乘。  
  
## <a name="syntax"></a>语法  
  
```  
  
result = number1*number2  
```  
  
## <a name="parameters"></a>参数  
 *结果*  
 任何变量。  
  
 `number1`  
 任何表达式。  
  
 `number2`  
 任何表达式。
