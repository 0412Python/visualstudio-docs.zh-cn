# <a name="mathatan2-function-javascript"></a>Math.atan2 函数 (JavaScript)
从 X 轴的角度 （以弧度为单位） 返回到点 (*y*，*x*)。  
  
## <a name="syntax"></a>语法  
  
```  
  
Math.atan2(y, x)  
```  
  
## <a name="parameters"></a>参数  
 *x*  
 必需。 一个数值表达式，表示笛卡尔坐标的 x 坐标。  
  
 *y*  
 必需。 一个数值表达式，表示笛卡尔坐标的 y 坐标。  
  
## <a name="remarks"></a>备注  
 返回值是之间 *-pi*和*pi*。 它表示所提供的角度 (*y*，*x*) 点，以弧度为单位。
