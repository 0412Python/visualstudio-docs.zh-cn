# Math.atan2 函数 (JavaScript)
返回从 X 轴到 \(*y*,*x*\) 点的角度（以弧度为单位）。  
  
## 语法  
  
```  
  
Math.atan2(y, x)  
```  
  
## 参数  
 *x*  
 必需。  一个数值表达式，表示笛卡尔 x 坐标。  
  
 *y*  
 必需。  一个数值表达式，表示笛卡尔 y 坐标。  
  
## 备注  
 返回值介于 *\-pi* 和 *pi* 之间。  它表示提供的 \(*y*,*x*\) 点的角度（以弧度为单位）。
