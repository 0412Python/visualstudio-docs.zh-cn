# JavaScript 保留字
[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 具有无法用作标识符的许多保留字。  对 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 语言来说，保留字具有特定含义，因为它们是该语言句法的一部分。  在加载脚本时，使用保留字会导致编译错误。  
  
 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 还包含未来保留字的列表。  这些字当前不是 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 语言的一部分，但可将其保留以供将来使用。  有关 JavaScript 中的未来保留关键字的详细信息，请参见 [JavaScript 未来保留字](../../javascript/reference/javascript-future-reserved-words.md)。  
  
 在选择标识符时，避免选择已经是内部 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 对象或函数的名称（比如 `String` 或 `parseInt`）也是非常重要的。  
  
## 保留的关键字  
  
||||||  
|-|-|-|-|-|  
|break|默认|function|return|var|  
|case|删除|if|switch|void|  
|catch|do|in|this|while|  
|const|else|instanceof|throw|with|  
|continue|finally|let|try||  
|debugger|for|new|typeof||
