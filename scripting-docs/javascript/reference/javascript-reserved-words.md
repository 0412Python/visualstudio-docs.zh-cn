# <a name="javascript-reserved-words"></a>JavaScript 保留字
[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]具有大量你不能用作标识符的保留字。 保留的字具有特定含义到[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]语言，因为它们是语言语法的一部分。 加载你的脚本时将使用保留的字导致编译错误。  
  
 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]此外具有未来保留字列表。 这些单词不是当前的一部分[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]语言中，尽管它们是保留供将来使用。 有关在 JavaScript 中的未来保留关键字的详细信息，请参阅[JavaScript 未来保留字](../../javascript/reference/javascript-future-reserved-words.md)。  
  
 选择标识符时也很重要以避免任何已内部函数的名称的字词[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]对象或函数，如`String`或`parseInt`。

|catch|do|in|this|while|

|continue|finally|let|try||
