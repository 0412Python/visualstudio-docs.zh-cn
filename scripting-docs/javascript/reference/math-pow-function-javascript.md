# <a name="mathpow-function-javascript"></a>Math.pow 函数 (JavaScript)
返回基表达式转到的指定次幂的值。  
  
## <a name="syntax"></a>语法  
  
```  
  
Math.pow(  
base, exponent)   
```  
  
## <a name="parameters"></a>参数  
 *base*  
 必需。 基表达式的值。  
  
 *指数*  
 必需。 表达式的指数值。  
  
## <a name="example"></a>示例  
 在下面的示例中，等于 baseexponent 数值表达式返回 1000年。  
  
```JavaScript  
Math.pow(10,3);  
```
