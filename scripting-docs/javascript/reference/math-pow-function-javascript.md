# Math.pow 函数 (JavaScript)
返回基数表达式的指定次幂的值。  
  
## 语法  
  
```  
  
Math.pow(  
base, exponent)   
```  
  
## 参数  
 *base*  
 必需。  表达式的基础值。  
  
 *exponent*  
 必需。  表达式的指数值。  
  
## 示例  
 在下面的示例中，与 baseexponent 相等的数值表达式返回 1000。  
  
```javascript  
Math.pow(10,3);  
```
