# <a name="mathcosh-function-javascript"></a>Math.cosh 函数 (JavaScript)
返回数字的双曲余弦。  
  
## <a name="syntax"></a>语法  
  
```  
  
Math.cosh(x)   
```  
  
## <a name="remarks"></a>备注  
 返回值与 `(exp(x) + exp(-x))/2` 相同。 如果 `x` 为 `NaN`，则结果为 `NaN`。  
  
 **适用于**: [Math 对象](../../javascript/reference/math-object-javascript.md)
