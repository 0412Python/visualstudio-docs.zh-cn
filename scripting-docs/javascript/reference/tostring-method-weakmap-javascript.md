# toString 方法 (WeakMap) (JavaScript)
返回 `WeakMap` 对象的字符串表示形式。  
  
## 语法  
  
```javascript  
weakmapObj.toString()  
```  
  
#### 参数  
 `weakmapObj`  
 必需。  `WeakMap` 对象。  
  
## 属性值\/返回值  
 `WeakMap` 的字符串表示形式。
