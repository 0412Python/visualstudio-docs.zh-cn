# 除法赋值运算符 (/=) (JavaScript)
变量值除以表达式值，并将结果赋给该变量。  
  
## 语法  
  
```  
  
result /= expression   
```  
  
## 参数  
 `result`  
 任何数值变量。  
  
 `expression`  
 任何数值表达式。  
  
## 备注  
 使用 `/=` 运算符与指定 `result = result / expression` 完全相同。
