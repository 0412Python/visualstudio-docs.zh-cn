# <a name="tostring-method-map-javascript"></a>toString 方法 (Map) (JavaScript)
返回的字符串表示形式映射。  
  
## <a name="syntax"></a>语法  
  
```JavaScript  
mapObj.toString()  
```  
  
#### <a name="parameters"></a>参数  
 `mapObj`  
 必需。 一个 `Map` 对象。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 字符串表示形式映射中。
