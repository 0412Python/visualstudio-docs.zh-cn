# toString 方法 (Map) (JavaScript)
返回映射的字符串表示形式。  
  
## 语法  
  
```javascript  
mapObj.toString()  
```  
  
#### 参数  
 `mapObj`  
 必需。  `Map` 对象。  
  
## 属性值\/返回值  
 映射的字符串表示形式。
