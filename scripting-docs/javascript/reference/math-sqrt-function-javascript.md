# Math.sqrt 函数 (JavaScript)
返回一个数的平方根。  
  
## 语法  
  
```  
  
Math.sqrt(  
number  
)   
```  
  
## 备注  
 必需的 `number` 参数是一个数值表达式。  
  
 如果 `number` 为负，则返回值为 `NaN`。
