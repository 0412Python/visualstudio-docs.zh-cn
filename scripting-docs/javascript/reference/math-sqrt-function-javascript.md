# <a name="mathsqrt-function-javascript"></a>Math.sqrt 函数 (JavaScript)
返回数字的平方根。  
  
## <a name="syntax"></a>语法  
  
```  
  
Math.sqrt(  
number  
)   
```  
  
## <a name="remarks"></a>备注  
 所需的 `number` 参数是数值表达式。  
  
 如果`number`为负，则返回值是`NaN`。
