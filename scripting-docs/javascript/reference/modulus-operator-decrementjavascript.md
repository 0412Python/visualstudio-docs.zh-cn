# <a name="modulus-operator--javascript"></a>取模运算符 (JavaScript)
将划分的另一个数值表达式，值的数值表达式的值，并生成其余部分。  
  
## <a name="syntax"></a>语法  
  
```  
result = expression1 % expression2  
```  
  
## <a name="arguments"></a>参数  
 `result`  
 任何变量。  
  
 `expression1`  
 任何数值表达式。  
  
 `expression2`  
 任何数值表达式。
