# <a name="mathtanh-function-javascript"></a>Math.tanh 函数 (JavaScript)
返回数字的双曲正切。  
  
## <a name="syntax"></a>语法  
  
```  
  
Math.tanh(  
x  
)  
  
```  
  
## <a name="remarks"></a>备注  
 返回值与 `(exp(x) - exp(-x))/(exp(x) + exp(-x))` 相同。 如果 `x` 为 `NaN`，则结果为 `NaN`。
