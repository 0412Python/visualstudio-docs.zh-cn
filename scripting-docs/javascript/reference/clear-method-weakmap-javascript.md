# clear 方法 (WeakMap) (JavaScript)
从 `WeakMap` 对象中移除所有元素。  
  
## 语法  
  
```javascript  
weakmapObj.clear()  
```  
  
#### 参数  
 `weakmapObj`  
 必需。  要清除的 `WeakMap` 对象。
