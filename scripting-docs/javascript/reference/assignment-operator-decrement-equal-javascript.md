# 赋值运算符 (=) (JavaScript)
给变量赋值。  
  
## 语法  
  
```  
  
result = expression  
```  
  
## 参数  
 `result`  
 任何变量。  
  
 `expression`  
 任何数值表达式。  
  
## 备注  
 \= 运算符的行为与其他运算符的行为类似，因此包含此运算符的表达式具有值。  这就意味着可以用下面的方式将赋值运算连起来写：`j = k = l = 0`。  在此示例中，`j`、`k` 和 `l` 等于零。
