# <a name="assignment-operator--javascript"></a>赋值运算符 (=) (JavaScript)
对变量赋值。  
  
## <a name="syntax"></a>语法  
  
```  
  
result = expression  
```  
  
## <a name="parameters"></a>参数  
 `result`  
 任何变量。  
  
 `expression`  
 任何数值表达式。  
  
## <a name="remarks"></a>备注  
 = 运算符在行为上类似于其他运算符，因此包含它的表达式具有一个值。 这意味着你可以按以下方式链接赋值运算符： `j = k = l = 0`。 在这种情况下`j`， `k`，和`l`等于零。
