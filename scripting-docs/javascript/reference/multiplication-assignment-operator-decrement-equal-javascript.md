# 乘法赋值运算符 (*=) (JavaScript)
变量值乘以表达式值，并将结果赋给该变量。  
  
## 语法  
  
```  
  
result *= expression  
```  
  
## 参数  
 *result*  
 任何变量。  
  
 *expression*  
 任何表达式。  
  
## 备注  
 使用 **\*\=** 运算符与指定以下内容完全相同：  
  
```javascript  
result = result * expression  
```
