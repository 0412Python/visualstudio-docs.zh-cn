# <a name="multiplication-assignment-operator--javascript"></a>乘法赋值运算符 (*=) (JavaScript)
将变量值乘以表达式值，并将结果赋给变量。  
  
## <a name="syntax"></a>语法  
  
```  
  
result *= expression  
```  
  
## <a name="parameters"></a>参数  
 *结果*  
 任何变量。  
  
 *表达式*  
 任何表达式。  
  
## <a name="remarks"></a>备注  
 使用 **\* =** 运算符正是与指定相同：  
  
```JavaScript  
result = result * expression  
```
