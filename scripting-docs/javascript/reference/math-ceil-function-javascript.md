# Math.ceil 函数 (JavaScript)
返回大于或等于其数字参数的最小整数。  
  
## 语法  
  
```  
  
Math.ceil(number)   
```  
  
## 备注  
 必需的 `number` 参数是一个数值表达式。  
  
 返回值为大于或等于其数字参数的最小整数值。  
  
 **适用于**：[Math 对象](../../javascript/reference/math-object-javascript.md)
