# <a name="mathceil-function-javascript"></a>Math.ceil 函数 (JavaScript)
大于或等于其数值自变量，则返回的最小整数。  
  
## <a name="syntax"></a>语法  
  
```  
  
Math.ceil(number)   
```  
  
## <a name="remarks"></a>备注  
 所需的 `number` 参数是数值表达式。  
  
 返回值为整数值等于大于或等于其数值自变量的最小整数。  
  
 **适用于**: [Math 对象](../../javascript/reference/math-object-javascript.md)
