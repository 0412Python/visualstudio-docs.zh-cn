# <a name="resolvedoptions-method-intlnumberformat"></a>resolvedOptions 方法 (Intl.NumberFormat)
返回一个包含属性和值的对象`NumberFormat`格式化程序对象。  
  
## <a name="syntax"></a>语法  
  
```  
numberFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>参数  
 `numberFormatObj`  
 `NumberFormat`要从中检索信息对象。  
  
## <a name="remarks"></a>备注  
 返回的对象的属性对应的计算属性`NumberFormat`对象。
