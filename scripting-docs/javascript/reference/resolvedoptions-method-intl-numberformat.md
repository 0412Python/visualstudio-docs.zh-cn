# resolvedOptions 方法 (Intl.NumberFormat)
返回包含 `NumberFormat` 格式化程序对象的属性和值的对象。  
  
## 语法  
  
```  
numberFormatObj.resolvedOptions()  
```  
  
#### 参数  
 `numberFormatObj`  
 要从其检索信息的 `NumberFormat` 对象。  
  
## 备注  
 所返回对象的属性对应于 `NumberFormat` 的计算属性。
