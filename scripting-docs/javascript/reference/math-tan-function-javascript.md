# Math.tan 函数 (JavaScript)
返回一个数的正切。  
  
## 语法  
  
```  
  
Math.tan(  
radians  
)  
  
```  
  
## 备注  
 必需的 `radians` 参数是包含测量的角度（以弧度为单位）的数值表达式。  
  
 返回值为 `radians` 的数值参数的正切值。
