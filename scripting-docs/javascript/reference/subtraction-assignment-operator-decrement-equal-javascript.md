# <a name="subtraction-assignment-operator---javascript"></a>减法赋值运算符 (-=) (JavaScript)
从变量值中减去表达式值，并将结果赋给该变量。  
  
## <a name="syntax"></a>语法  
  
```  
  
result -= expression  
```  
  
## <a name="parameters"></a>参数  
 *结果*  
 任何数值变量。  
  
 *表达式*  
 任何数值表达式。  
  
## <a name="remarks"></a>备注  
 使用`-=`运算符是完全相同的方式执行以下操作：  
  
```JavaScript  
result = result - expression  
```
