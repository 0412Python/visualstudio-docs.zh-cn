# 减法赋值运算符 (-=) (JavaScript)
从变量值中减去表达式值，并将结果赋给该变量。  
  
## 语法  
  
```  
  
result -= expression  
```  
  
## 参数  
 *result*  
 任何数值变量。  
  
 *expression*  
 任何数值表达式。  
  
## 备注  
 使用 `-=` 运算符与执行以下操作完全相同：  
  
```javascript  
result = result – expression  
```
