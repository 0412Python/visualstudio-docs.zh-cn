# <a name="unterminated-comment"></a>未终止的注释
开始一个多行注释块，但未正确终止它。 多行注释开头"/ *"组合，并以反序结束"\*/"组合。 下面是一个示例：  
  
```JavaScript  
/* This is a comment  
This is another part of the same comment.*/  
```  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   请务必终止多行注释以"* /"。
