# <a name="expected-identifier-string-or-number"></a>应为标识符、字符串或数字
不正确的文本语法用于声明对象文字。 对象文字的属性必须是标识符、 字符串或数字。 对象文字 （也称为"对象初始值设定项"） 包含以逗号分隔的属性： 值对，所有包含在括号内的列表。 例如：  
  
```JavaScript  
var point = {x:1.2, y:-3.4};  
```  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保使用正确的文字语法。
