# 正则表达式中有语法错误 (JavaScript)
您的搜索字符串的结构违反 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 正则表达式的一个或多个语法规则。  
  
### 更正此错误  
  
-   确保您的正则表达式搜索字符串的结构遵循 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 正则表达式语法。
