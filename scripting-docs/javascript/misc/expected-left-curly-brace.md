# 应有“{”
未加入标记函数体开头的左大括号。  组成函数体的代码即使只有一行，也必须在其两端加上左右大括号。  注意，在函数体中使用大括号比在循环中使用更严格。  
  
### 更正此错误  
  
-   添加标记函数体开头的左大括号。
