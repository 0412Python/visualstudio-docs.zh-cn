# <a name="expected-3939"></a>预期 &#39;{&#39;
不包含标记的函数体开始的左大括号。 代码，即使它是单行，后者构成函数的正文必须包含在向左和右大括号。 请注意，函数体中的大括号的使用是比在循环中更严格。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   添加标记的函数体开始的左大括号。
