# <a name="label-redefined"></a>标签被重定义
创建新的标签，但赋予了它的现有标签的名称。 标签可用于标记代码块，但在指定的范围内，它们必须是唯一的。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   确保在程序中使用的所有标签在其各自的范围内唯一。
