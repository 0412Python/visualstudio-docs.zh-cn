# <a name="label-redefined"></a>标签被重定义
创建新标签，但为它提供了现有标签的名称。 可以使用标签来标记代码块，但在指定的范围内，它们必须唯一。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   确保在程序中使用的所有标签都都在其各自的范围内唯一。
