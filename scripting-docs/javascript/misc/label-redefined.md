# 标签被重定义
您创建了一个新标签，但为其给定的是现有标签的名称。  标签可用于标记代码块，但在指定的范围内，标记必须是唯一的。  
  
### 更正此错误  
  
-   确保程序中使用的所有标签在其各自的范围内都是唯一的。
