# 应有“=”(JavaScript)
尝试创建要用于条件编译语句的变量，但不在变量和要赋给变量的值之间放置等号。  
  
### 更正此错误  
  
-   添加等号。  例如：  
  
    ```javascript  
    @set @myvar1 = 1  
    ```
