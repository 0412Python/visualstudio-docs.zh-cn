# 应有 Array 或 Arguments 对象
未提供作为参数的数组。  此错误仅适用于 **Function.prototype.apply** 方法。  如果指定，则该函数的第二个参数必须为 `Array` 对象或 **Arguments** 对象。  
  
### 更正此错误  
  
-   指定 `Array` 或 **Arguments** 对象作为第二个参数。
