# <a name="expected-39-39"></a>预期 &#39;-&#39;
你试图创建正则表达式文本，但不是包括一个斜杠 （/）。 就像作为一对引号内的字符写入字符串文本，正则表达式文本表示为一对大斜杠 （/） 字符之间的字符。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   插入终止正斜杠标记的正则表达式的末尾。
