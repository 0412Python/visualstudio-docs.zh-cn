# 应有“/”
尝试创建一个正则表达式文本，但不包括斜杠 \(\/\)。  就像将字符串文本编写为一对引号中的字符一样，正则表达式文本将表示为一对斜杠 \(\/\) 字符中的字符。  
  
### 更正此错误  
  
-   插入用于标记正则表达式的结尾的终止正斜杠。
