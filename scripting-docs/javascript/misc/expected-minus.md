# <a name="expected-39-39"></a>预期&#39;-&#39;
尝试创建正则表达式文本，但不是包含斜杠 （/） 之一。 就像为一对引号引起来的字符写入字符串文本，正则表达式文本表示为一对大斜杠 （/） 字符的字符。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   插入标记的正则表达式的末尾终止以正斜杠。
