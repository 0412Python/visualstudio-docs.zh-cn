# <a name="expected-"></a>应有“]”
进行对数组元素的引用，但并未包含右方括号。 引用的数组元素的任何表达式必须包含左中括号和右括号。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   将右方括号添加到引用的数组元素的表达式。
