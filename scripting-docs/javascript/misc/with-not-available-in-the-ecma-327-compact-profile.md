# “with”在 ECMA 327 Compact Profile 中不可用
ECMA 327 Compact Profile 不支持“with”\(ErrWithNotInCP\)  
  
 无法在 Compact Profile 中使用 with 关键字。  
  
 有关 with 关键字的更多信息，请参见 [with 语句](../../javascript/reference/with-statement-javascript.md)。
