# 应为标识符 (JavaScript)
您在需要标识符的上下文中使用了标识符之外的内容。  标识符可以为：  
  
-   变量，  
  
-   属性，  
  
-   数组，  
  
-   或函数名称。  
  
### 更正此错误  
  
-   更改表达式，使一个标识符出现在等号的左侧。
