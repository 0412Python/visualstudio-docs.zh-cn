# 使用关键字作为标识符是无效的
1048：无法使用标识符的关键字  
  
 不能将关键字用作变量、函数或其他用户定义的标识符的名称。  
  
 有关关键字的更多信息，请参见 [JavaScript 保留字](../../javascript/reference/javascript-reserved-words.md)和 [JavaScript 未来保留字](../../javascript/reference/javascript-future-reserved-words.md)。
