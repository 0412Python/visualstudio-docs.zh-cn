# 使用未来保留字作为标识符是无效的
1049：无法使用标识符的未来保留字  
  
 不能将未来保留字用作变量、函数或其他用户定义的标识符的名称。  
  
 有关未来保留字的详细信息，请参见 [JavaScript 未来保留字](../../javascript/reference/javascript-future-reserved-words.md)。
