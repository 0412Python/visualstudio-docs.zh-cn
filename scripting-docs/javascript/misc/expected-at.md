# <a name="expected-3939"></a>预期 &#39; @&#39;
尝试创建一个变量以与使用的条件编译语句一起使用`@set`语句，但是没有按照 at 符号"**@**"之前的变量的名称。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   添加 at 符号"**@**"紧接在变量名称之前。 例如:   
  
    ```JavaScript  
    @set @myvar = 1  
    ```
