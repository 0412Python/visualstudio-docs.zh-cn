# <a name="expected-3939"></a>预期 &#39;: &#39;
你试图创建使用三元的条件运算符的表达式，但不是包括第二个和第三个操作数之间的冒号。 三元 （三个操作数） 条件运算符之间需要一个冒号的第二个 (true) 和第三 (false) 操作数。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   之间插入一个冒号的第二个和第三个操作数。
