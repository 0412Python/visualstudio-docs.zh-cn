# 应有“:”
尝试使用三元条件运算符创建表达式，但在第二个和第三个操作数之间不包括冒号。  三元（三个操作数）条件运算符需要在第二个 \(true\) 和第三个 \(false\) 操作数之间使用冒号。  
  
### 更正此错误  
  
-   在第二个和第三个操作数之间插入冒号。
