# <a name="expected-"></a>应有“:”
尝试创建表达式中使用三元条件运算符，但不是包括第二个和第三个操作数之间的冒号。 三元 （三个操作数） 条件运算符之间需要一个冒号 (true) 第二个和第三个 (false) 操作数。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   插入第二个和第三个操作数之间的一个冒号。
