# <a name="expected-constant"></a>应为常量
你尝试在条件编译测试语句中使用 （非条件编译） 变量。 测试条件编译语句的计算结果必须是常数。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   将文本替换为该变量。  
  
-   将条件编译变量替换为该变量。
