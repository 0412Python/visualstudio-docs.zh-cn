# 应为 JavaScript 对象
尝试为需要 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 对象的内置函数传递非 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 对象。  不同的内置函数需要 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 中定义的对象（与宿主或外部组件（如控件）定义的对象相反）。  
  
### 更正此错误  
  
-   确保作为参数传入的对象的类型正确。
