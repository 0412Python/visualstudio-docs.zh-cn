# <a name="expected-39catch39"></a>预期 &#39; catch &#39;
使用异常处理**重**阻止，但没有编写关联**捕获**语句。 异常处理机制要求可能会失败，以及如果发生异常，不应执行的代码的代码被包装到**重**块。 从引发异常**重**阻止使用**引发**语句，并捕获外部**重**块与一个或多**捕获**语句。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   添加关联**捕获**块。  
  
-   请尝试使用**最后**阻止而不是**捕获**块。
