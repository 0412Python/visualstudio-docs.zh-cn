# 应有“}”
不包含标记函数体、循环、代码块或对象初始值设定项的结尾的右大括号。  此错误的一个示例是，仅带标记循环体的左大括号的 **for** 循环。  
  
### 更正此错误  
  
-   添加标记函数、循环、代码块或对象初始值设定项的结尾的右大括号。
