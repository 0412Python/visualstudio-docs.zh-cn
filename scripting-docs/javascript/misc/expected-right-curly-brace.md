# <a name="expected-"></a>应有“}”
未包含标记的函数体、 循环、 代码块末尾的右大括号或对象初始值设定项。 此错误的示例将**为**仅标记的循环主体的左大括号的循环。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   添加标记的结束函数、 循环、 块或对象初始值设定项的右大括号。
