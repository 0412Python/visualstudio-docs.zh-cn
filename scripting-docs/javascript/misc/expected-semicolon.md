# <a name="expected-3939"></a>预期 &#39;&#39;
你试图将多个语句放在一行中，但不得未用分号分隔它们。 使用分号终止语句。 虽然可以将多个语句放在一行上，必须从以分号下一步语句分隔每个。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   将标记每个以分号的语句的末尾。  
  
-   请确保函数调用正确地使用括号。  
  
-   请确保你未忘记的标头中分号**为**循环。
