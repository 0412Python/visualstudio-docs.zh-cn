# 应用“;”
尝试在一个行中放置多个语句，但不用分号将它们分隔开。  分号用于终止语句。  虽然可以在一个行上放置多个语句，但必须用分号将每个语句与下一个语句分隔开。  
  
### 更正此错误  
  
-   用分号标记每个语句的结尾。  
  
-   确保您的函数调用正确使用括号。  
  
-   确保记得在 **for** 循环的标头中置入分号。
