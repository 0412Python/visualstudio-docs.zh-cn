# <a name="expected--javascript"></a>应有“(”(JavaScript)
您试图将括在括号内，一组表达式，但不是包括左括号。 某些表达式必须括在一组左、 右括号。 请注意，使用下面的示例中的括号。  
  
```JavaScript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   将左的括号添加到的求值表达式。
