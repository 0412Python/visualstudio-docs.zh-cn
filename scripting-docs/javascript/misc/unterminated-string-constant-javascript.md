# <a name="unterminated-string-constant-javascript"></a>未终止的字符串常量 (JavaScript)
未结束右引号你字符串常量。 字符串常量必须括在成对的引号内。  
  
> [!NOTE]
>  你可以使用配对的单引号或双引号引号引起来。 可以用单个引号括起来的字符串中包含两个双引号和单引号可以包含在由双引号括起来的字符串。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   将在右引号添加到字符串的末尾。
