# <a name="unterminated-string-constant-javascript"></a>未终止的字符串常量 (JavaScript)
未结束你在字符串常量前面为右引号。 字符串常量必须用一对引号引起来。  
  
> [!NOTE]
>  可以使用配对的单引号或双引号括起来。 中字符串括在单引号中，可能包含两个双引号和单引号可以包含字符串括在双引号内。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   添加到字符串的末尾的右引号。
