# <a name="cant-have-continue-outside-of-loop"></a>“continue”不能位于循环外
你尝试使用**继续**位于循环外的语句。 **继续**答： 在正文中仅可以使用语句  
  
-   `do-while` 循环中，  
  
-   `while` 循环中，  
  
-   **有关**循环中，  
  
-   **/ 中**循环。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保**继续**语句出现在正文答：  
  
    -   `do-while` 循环中，  
  
    -   `while` 循环中，  
  
    -   **有关**循环中，  
  
    -   **/ 中**循环。
