# <a name="can39t-have-39continue39-outside-of-loop"></a>可以 &#39; 具有 &#39; 继续 （& a) #39;位于循环外
你尝试使用**继续**位于循环外的语句。 **继续**答： 主体中仅可以使用语句  
  
-   `do-while`循环中，  
  
-   `while`循环中，  
  
-   **有关**循环，  
  
-   **/ 中**循环。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保**继续**语句出现在正文答：  
  
    -   `do-while`循环中，  
  
    -   `while`循环中，  
  
    -   **有关**循环，  
  
    -   **/ 中**循环。
