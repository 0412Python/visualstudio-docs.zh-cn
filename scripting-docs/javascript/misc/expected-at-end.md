# <a name="expected-39end39"></a>预期 &#39;@end&#39;
试图创建有条件编译的块的代码，但不是包括 **@end** 末尾的语句。 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]语句可以有条件地编译的封闭内为这些 **@if/ @end** 块。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   添加相应 **@end** 语句。
