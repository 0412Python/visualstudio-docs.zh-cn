# <a name="expected-3939-javascript"></a>预期&#39;)&#39; (JavaScript)
您试图将括在括号内，一组表达式，但不是包括右括号。 某些表达式必须括在一组左、 右括号。 请注意，使用下面的示例中的括号。  
  
```JavaScript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
-   添加为计算表达式的右括号。
