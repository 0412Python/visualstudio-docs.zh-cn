# 应有“)”(JavaScript)
尝试将表达式放入一组括号内，但不包括该右括号。  某个表达式必须用一组左括号和右括号引起来。  请注意以下示例中括号的用法。  
  
```javascript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### 更正此错误  
  
-   将右括号添加到计算表达式。
