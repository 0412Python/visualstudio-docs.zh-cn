# IActiveScriptGarbageCollector 接口
提供了一种方法来启动垃圾回收。  应由若要清除它们的资源的操作脚本引擎实现此接口。  
  
## 方法  
  
|方法|说明|  
|--------|--------|  
|[IActiveScriptGarbageCollector::CollectGarbage](../../winscript/reference/iactivescriptgarbagecollector-collectgarbage.md)|启动垃圾回收。|
