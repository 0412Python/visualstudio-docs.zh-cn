# IJsDebugBreakPoint::Delete 方法
删除断点。  
  
## 语法  
  
```  
HRESULT Delete(void);  
```  
  
## 返回值  
  
## 备注  
 如果对已删除的断点调用，则返回 S\_FALSE。

 **标头：**jscript9diag.h
