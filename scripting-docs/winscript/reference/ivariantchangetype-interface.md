# IVariantChangeType 接口
允许调用方从一种类型转换的值不同到另一个。  
  
## 方法  
 除了从 `IUnknown` 继承的方法之外，`IVariantChangeType` 接口还公开下面的方法。  
  
|方法|说明|  
|--------|--------|  
|[IVariantChangeType::ChangeType](../../winscript/reference/ivariantchangetype-changetype.md)|采用不同的值并用指定的类型创建新的变量。|
