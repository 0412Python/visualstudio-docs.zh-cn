# <a name="ivariantchangetype-interface"></a>IVariantChangeType 接口
允许调用方将从一种类型的变体值转换为另一个。  
  
## <a name="methods"></a>方法  
 除了从继承的方法`IUnknown`、`IVariantChangeType`接口公开以下方法。  
  
|方法|描述|  
|------------|-----------------|  
|[IVariantChangeType::ChangeType](../../winscript/reference/ivariantchangetype-changetype.md)|可将变量值并创建具有指定类型的新变量。|
