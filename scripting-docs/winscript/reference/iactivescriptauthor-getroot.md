# <a name="iactivescriptauthorgetroot"></a>IActiveScriptAuthor::GetRoot
返回`IScriptNode`作者脚本树的根。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetRoot(  
   IScriptNode        **ppsp  
);  
```  
  
#### <a name="parameters"></a>参数  
 `ppsp`  
 [out]接收指向的变量的地址`IScriptNode`的根节点的接口。  
  
## <a name="return-value"></a>返回值  
 一个 `HRESULT`。 可能的值包括（但并不限于）下表中的项。  
  
|值|描述|  
|-----------|-----------------|  
|`S_OK`|方法成功。|
