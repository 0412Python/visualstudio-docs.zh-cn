# IEnumDebugCodeContexts 接口
枚举对应于文档上下文中的代码上下文。  
  
 除了从 `IUnknown` 继承的方法之外，`IEnumDebugCodeContexts` 接口还公开下面的方法。  
  
## Vtable 顺序中的方法  
  
|方法|说明|  
|--------|--------|  
|[IEnumDebugCodeContexts::Next](../../winscript/reference/ienumdebugcodecontexts-next.md)|检索枚举序列中指定数目的段。|  
|[IEnumDebugCodeContexts::Skip](../../winscript/reference/ienumdebugcodecontexts-skip.md)|跳过枚举序列中指定数目的段。|  
|[IEnumDebugCodeContexts::Reset](../../winscript/reference/ienumdebugcodecontexts-reset.md)|将枚举序列重置到开始处。|  
|[IEnumDebugCodeContexts::Clone](../../winscript/reference/ienumdebugcodecontexts-clone.md)|创建与当前枚举器包含相同状态的一个枚举器。|
