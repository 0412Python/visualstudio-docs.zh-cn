# PROFILER_HEAP_OBJECT_ID 类型
堆对象的ID。  使用在 [PROFILER\_HEAP\_OBJECT 结构](../../winscript/reference/profiler-heap-object-structure.md)[PROFILER\_HEAP\_OBJECT\_SCOPE\_LIST 结构](../../winscript/reference/profiler-heap-object-scope-list-structure.md)，[PROFILER\_HEAP\_OBJECT\_OPTIONAL\_INFO 结构](../../winscript/reference/profiler-heap-object-optional-info-structure.md)和 [PROFILER\_HEAP\_OBJECT\_RELATIONSHIP 结构](../../winscript/reference/profiler-heap-object-relationship-structure.md)。  
  
## 语法  
  
```  
typedef DWORD_PTR PROFILER_HEAP_OBJECT_ID;  
```
