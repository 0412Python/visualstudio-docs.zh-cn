# PROFILER_EXTERNAL_OBJECT_ADDRESS 类型
对象的外部对象地址，如c. C\+\+分配对象，该对象是在JavaScript堆外。  使用在 [PROFILER\_HEAP\_OBJECT 结构](../../winscript/reference/profiler-heap-object-structure.md) 和 [PROFILER\_HEAP\_OBJECT\_RELATIONSHIP 结构](../../winscript/reference/profiler-heap-object-relationship-structure.md)。  
  
## 语法  
  
```  
typedef void* PROFILER_EXTERNAL_OBJECT_ADDRESS;  
```
