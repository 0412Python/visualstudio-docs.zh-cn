# IRemoteDebugApplicationThreadEx 接口
表示执行线程在特定应用程序中。  
  
 除了从 `IUnknown` 继承的方法之外，`IRemoteDebugApplicationThreadEx` 接口还公开下面的方法。  
  
## Vtable 顺序中的方法  
  
|方法|说明|  
|--------|--------|  
|[IRemoteDebugApplicationThreadEx:EnumGlobalContexts](../../winscript/reference/iremotedebugapplicationthreadex-enumglobalcontexts.md)|枚举运行在此线程的所有语言的全局表达式上下文。|
