# <a name="iremotedebugapplicationthreadex-interface"></a>IRemoteDebugApplicationThreadEx 接口
表示特定的应用程序中的执行线程。  
  
 除了从继承的方法`IUnknown`、`IRemoteDebugApplicationThreadEx`接口公开以下方法。  
  
## <a name="methods-in-vtable-order"></a>Vtable 顺序中的方法  
  
|方法|描述|  
|------------|-----------------|  
|[IRemoteDebugApplicationThreadEx:EnumGlobalContexts](../../winscript/reference/iremotedebugapplicationthreadex-enumglobalcontexts.md)|枚举在此线程中运行的所有语言的全局表达式上下文。|
