# <a name="idebugpropertygetparent"></a>IDebugProperty::GetParent
获取属性的父属性。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetParent (  
   IDebugProperty** ppParent  
);  
```  
  
#### <a name="parameters"></a>参数  
 `ppParent`  
 [out]返回`IDebugProperty`表示属性的父级的接口。  
  
## <a name="return-value"></a>返回值  
 返回一个有效`HRESULT`，通常`S_OK`。
