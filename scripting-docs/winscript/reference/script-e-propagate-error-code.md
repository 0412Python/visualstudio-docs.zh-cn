# <a name="scriptepropagate-error-code"></a>SCRIPT_E_PROPAGATE 错误代码
脚本错误被传播到调用方，这可能是在不同线程中。 主机应将错误代码传递到调用方。  
  
## <a name="syntax"></a>语法  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
