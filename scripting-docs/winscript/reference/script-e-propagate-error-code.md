# SCRIPT_E_PROPAGATE 错误代码
脚本错误传播给调用方，可能是在不同的线程。  宿主应通过错误代码给调用方。  
  
## 语法  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
