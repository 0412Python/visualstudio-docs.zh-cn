# IDebugExpressionCallBack 接口
提供状态事件与 `IDebugExpression` 计算的进度相关。  
  
## 方法  
 除了从 `IUnknown` 继承的方法之外，`IDebugExpressionCallBack` 接口还公开下面的方法。  
  
|方法|说明|  
|--------|--------|  
|[IDebugExpressionCallBack::onComplete](../../winscript/reference/idebugexpressioncallback-oncomplete.md)|指示表达式计算完成。|
