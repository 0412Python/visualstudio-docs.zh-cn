# <a name="idebugdocument-interface"></a>IDebugDocument 接口
用于调试的所有文档的基接口。  
  
## <a name="remarks"></a>备注  
 `IDebugDocument`接口公开继承自方法[IDebugDocumentInfo 接口](../../winscript/reference/idebugdocumentinfo-interface.md)。 它不公开任何其他方法。
