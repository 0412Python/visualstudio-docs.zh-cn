# <a name="ienumdebugpropertyinforeset"></a>IEnumDebugPropertyInfo::Reset
将枚举序列重置到开头。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Reset (  
void  
);  
```  
  
## <a name="return-value"></a>返回值  
 返回一个有效`HRESULT`，通常`S_OK`。
