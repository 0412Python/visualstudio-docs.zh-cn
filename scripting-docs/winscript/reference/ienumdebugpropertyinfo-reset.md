# IEnumDebugPropertyInfo::Reset
将枚举序列重置到开始处。  
  
## 语法  
  
```  
HRESULT Reset (  
void  
);  
```  
  
## 返回值  
 返回有效的 `HRESULT`，通常 `S_OK`。
