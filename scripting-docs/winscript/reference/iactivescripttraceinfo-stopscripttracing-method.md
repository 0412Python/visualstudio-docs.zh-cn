# IActiveScriptTraceInfo::StopScriptTracing 方法
停止脚本跟踪。  
  
## 语法  
  
```  
HRESULT StopScriptTracing(void);   
}  
  
```  
  
#### 参数  
 此方法没有参数。  
  
## 返回值  
 可以返回此方法的值如下所示：  
  
1.  S\_OK:成功。  
  
2.  E\_NOTIMPL:未实现。
