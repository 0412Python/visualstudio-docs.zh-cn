# <a name="idebugasyncoperationqueryiscomplete"></a>IDebugAsyncOperation::QueryIsComplete
确定是否已完成调试操作。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT QueryIsComplete();  
```  
  
#### <a name="parameters"></a>参数  
 此方法采用任何参数。  
  
## <a name="return-value"></a>返回值  
 该方法返回 `HRESULT`。 可能的值包括（但并不限于）下表中的项。  
  
|值|描述|  
|-----------|-----------------|  
|`S_OK`|操作已完成。|  
|`S_FALSE`|该操作不完整。|  
  
## <a name="remarks"></a>备注  
 此方法可确定是否已完成调试操作。
