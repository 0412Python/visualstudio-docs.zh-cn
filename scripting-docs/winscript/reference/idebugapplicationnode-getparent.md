# <a name="idebugapplicationnodegetparent"></a>IDebugApplicationNode::GetParent
返回此应用程序节点的父节点。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetParent(  
   IDebugApplicationNode**  pprddp  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pprddp`  
 [out]此应用程序节点的父应用程序节点。  
  
## <a name="return-value"></a>返回值  
 该方法返回 `HRESULT`。 可能的值包括（但并不限于）下表中的项。  
  
|值|描述|  
|-----------|-----------------|  
|`S_OK`|方法成功。|  
  
## <a name="remarks"></a>备注  
 此方法返回此应用程序节点的父节点。
