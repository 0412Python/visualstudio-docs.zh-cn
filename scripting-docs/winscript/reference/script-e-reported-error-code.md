# SCRIPT_E_REPORTED 错误代码
脚本引擎未经处理的异常移到宿主定义报表。[IActiveScriptSite::OnScriptError](../../winscript/reference/iactivescriptsite-onscripterror.md)。  宿主可以忽略此错误。  
  
## 语法  
  
```cpp  
SCRIPT_E_REPORTED   0x80020101L  
  
```
