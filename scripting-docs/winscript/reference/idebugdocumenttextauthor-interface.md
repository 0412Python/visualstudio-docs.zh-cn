# IDebugDocumentTextAuthor 接口
提供文本文档进行编辑。  
  
 除了从 `IDebugDocumentText` 继承的方法之外，`IDebugDocumentTextAuthor` 接口还公开下面的方法。  
  
## Vtable 顺序中的方法  
  
|方法|说明|  
|--------|--------|  
|[IDebugDocumentTextAuthor::InsertText](../../winscript/reference/idebugdocumenttextauthor-inserttext.md)|插入文本到文档中。|  
|[IDebugDocumentTextAuthor::RemoveText](../../winscript/reference/idebugdocumenttextauthor-removetext.md)|从文档中移除文本。|  
|[IDebugDocumentTextAuthor::ReplaceText](../../winscript/reference/idebugdocumenttextauthor-replacetext.md)|替换文档中的文本。|
