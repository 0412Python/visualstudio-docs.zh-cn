# <a name="idebugcodecontext-interface"></a>IDebugCodeContext 接口
表示可执行代码中的位置的抽象。  
  
 除了从继承的方法`IUnknown`、`IDebugCodeContext`接口公开以下方法。  
  
## <a name="methods-in-vtable-order"></a>Vtable 顺序中的方法  
  
|方法|描述|  
|------------|-----------------|  
|[IDebugCodeContext::GetDocumentContext](../../winscript/reference/idebugcodecontext-getdocumentcontext.md)|返回与此代码上下文关联的文档上下文。|  
|[IDebugCodeContext::SetBreakPoint](../../winscript/reference/idebugcodecontext-setbreakpoint.md)|设置或清除在此代码上下文断点。|
