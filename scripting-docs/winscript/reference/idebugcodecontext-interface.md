# IDebugCodeContext 接口
表示在可执行代码中的位置的抽象。  
  
 除了从 `IUnknown` 继承的方法之外，`IDebugCodeContext` 接口还公开下面的方法。  
  
## Vtable 顺序中的方法  
  
|方法|说明|  
|--------|--------|  
|[IDebugCodeContext::GetDocumentContext](../../winscript/reference/idebugcodecontext-getdocumentcontext.md)|返回文档上下文与此代码上下文。|  
|[IDebugCodeContext::SetBreakPoint](../../winscript/reference/idebugcodecontext-setbreakpoint.md)|此代码上下文设置或清除断点。|
