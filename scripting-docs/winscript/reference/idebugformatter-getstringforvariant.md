# <a name="idebugformattergetstringforvariant"></a>IDebugFormatter::GetStringForVariant
返回表示给定的变量值的字符串。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetStringForVariant(  
   VARIANT*  pvar,  
   ULONG     nRadix,  
   BSTR*     pbstrValue  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pvar`  
 [in]若要将其表示为一个字符串的变量。  
  
 `nRadix`  
 [in]要用于数字值的基数。  
  
 `pbstrValue`  
 [out]字符串表示`pvar`。  
  
## <a name="return-value"></a>返回值  
 该方法返回 `HRESULT`。 可能的值包括（但并不限于）下表中的项。  
  
|值|描述|  
|-----------|-----------------|  
|`S_OK`|方法成功。|  
  
## <a name="remarks"></a>备注  
 此方法返回表示给定的变量值的字符串。
