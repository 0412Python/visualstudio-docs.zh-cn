# <a name="iactivescriptsiteinterruptpoll-interface"></a>IActiveScriptSiteInterruptPoll 接口
`IActiveScriptSiteInterruptPoll`接口允许主机可以指定脚本应终止。  
  
## <a name="methods"></a>方法  
 除了从继承的方法`IUnknown`、`IActiveScriptSiteInterruptPoll`接口公开以下方法。  
  
|方法|描述|  
|------------|-----------------|  
|[IActiveScriptSiteInterruptPoll::QueryContinue](../../winscript/reference/iactivescriptsiteinterruptpoll-querycontinue.md)|允许主机可以指定脚本应终止。|
