# IActiveScriptSiteInterruptPoll 接口
`IActiveScriptSiteInterruptPoll` 接口允许宿主指定脚本应停止。  
  
## 方法  
 除了从 `IUnknown` 继承的方法之外，`IActiveScriptSiteInterruptPoll` 接口还公开以下方法。  
  
|方法|说明|  
|--------|--------|  
|[IActiveScriptSiteInterruptPoll::QueryContinue](../../winscript/reference/iactivescriptsiteinterruptpoll-querycontinue.md)|允许宿主指定脚本应停止。|
