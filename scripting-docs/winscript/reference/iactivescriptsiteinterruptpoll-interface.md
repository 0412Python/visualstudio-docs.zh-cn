# <a name="iactivescriptsiteinterruptpoll-interface"></a>IActiveScriptSiteInterruptPoll 接口
`IActiveScriptSiteInterruptPoll`接口使宿主可以指定应终止脚本。  
  
## <a name="methods"></a>方法  
 除了继承的方法之外`IUnknown`，则`IActiveScriptSiteInterruptPoll`接口公开以下方法。  
  
|方法|描述|  
|------------|-----------------|  
|[IActiveScriptSiteInterruptPoll::QueryContinue](../../winscript/reference/iactivescriptsiteinterruptpoll-querycontinue.md)|使宿主可以指定应终止脚本。|
