# <a name="ijsenumdebugpropertygetcount-method"></a>IJsEnumDebugProperty::GetCount 方法
获取该对象的属性数目。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetCount(  
   ULONG *pCount  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pCount`  
 [out]属性的数目。  
  
## <a name="return-value"></a>返回值  
  
## <a name="requirements"></a>要求  
 **标头：** jscript9diag.h
