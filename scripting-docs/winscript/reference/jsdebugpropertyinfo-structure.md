# JsDebugPropertyInfo 结构
指示有关属性的信息。  
  
## 语法  
  
```  
typedef struct tagJsDebugPropertyInfo  
{  
   BSTR name;  
   BSTR type;  
   BSTR value;  
   BSTR fullName;  
   JS_PROPERTY_ATTRIBUTES attr;  
} JsDebugPropertyInfo;  
```  
  
## 成员  
 `name`  
 属性的名称。  
  
 `type`  
 属性的类型。  
  
 `value`  
 该属性的值。  
  
 `fullName`  
 属性的完整名称。  
  
 `attr`  
 表示属性的枚举。  
  
## 要求  
 **标头：**jscript9diag.h
