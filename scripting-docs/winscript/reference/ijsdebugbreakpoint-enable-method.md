# IJsDebugBreakPoint::Enable 方法
启用断点。  
  
## 语法  
  
```  
HRESULT Enable(void);  
```  
  
## 返回值  
  
## 备注  
 如果对已删除的断点调用，则返回 E\_UNEXPECTED。  
  
## 要求  
 **标头：**jscript9diag.h
