# <a name="idisperrorgethresult"></a>IDispError::GetHresult
检索中的错误代码`IDispError`对象。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetHresult(  
   HRESULT*  phr  
);  
```  
  
#### <a name="parameters"></a>参数  
 `phr`  
 [out]指定的错误代码。  
  
## <a name="return-value"></a>返回值  
 该方法返回 `HRESULT`。 可能的值包括（但并不限于）下表中的项。  
  
|值|描述|  
|-----------|-----------------|  
|`S_OK`|方法成功。|  
  
## <a name="remarks"></a>备注  
 此方法检索中的错误代码`IDispError`对象。  
  
> [!NOTE]
>  未实现此方法。
