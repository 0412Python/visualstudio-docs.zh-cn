# IBindEventHandler 接口
使调用方提供对象处理一个指定事件处理程序。  
  
## 方法  
 除了从 `IUnknown` 继承的方法之外，`IBindEventHandler` 接口还公开下面的方法。  
  
|方法|说明|  
|--------|--------|  
|[IBindEventHandler::BindHandler](../../winscript/reference/ibindeventhandler-bindhandler.md)|将某个事件对象。|
