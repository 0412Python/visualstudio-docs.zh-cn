# SCRIPT_E_RECORDED 错误代码
错误已通过在脚本引擎和宿主之间。  宿主需要通过错误代码给调用方。  
  
## 语法  
  
```cpp  
SCRIPT_E_RECORDED   0x86664004L  
```
