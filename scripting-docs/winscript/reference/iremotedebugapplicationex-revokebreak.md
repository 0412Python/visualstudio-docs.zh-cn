# IRemoteDebugApplicationEx:RevokeBreak
取消中断命令。  
  
## 语法  
  
```  
HRESULT RevokeBreak( );  
```  
  
#### 参数  
 无。  
  
## 返回值  
 该方法返回 `HRESULT`。  可能的值包括，但是，并不限于，这些下表中。  
  
|值|说明|  
|-------|--------|  
|`S_OK`|方法成功。|
