# IDebugAsyncOperationCallBack 接口
提供状态事件与 `IDebugAsyncOperation` 接口计算的进度相关。  
  
## 方法  
 除了从 `IUnknown` 继承的方法之外，`IDebugAsyncOperationCallBack` 接口还公开下面的方法。  
  
|方法|说明|  
|--------|--------|  
|[IDebugAsyncOperationCallBack::onComplete](../../winscript/reference/idebugasyncoperationcallback-oncomplete.md)|信号结果从异步获得调试操作。|
