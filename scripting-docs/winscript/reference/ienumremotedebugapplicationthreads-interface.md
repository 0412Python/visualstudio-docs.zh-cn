# IEnumRemoteDebugApplicationThreads 接口
枚举在应用程序中运行的线程。  
  
 除了从 `IUnknown` 继承的方法之外，`IEnumRemoteDebugApplicationThreads` 接口还公开下面的方法。  
  
## Vtable 顺序中的方法  
  
|方法|说明|  
|--------|--------|  
|[IEnumRemoteDebugApplicationThreads::Next](../../winscript/reference/ienumremotedebugapplicationthreads-next.md)|检索段指定数目的枚举序列的。|  
|[IEnumRemoteDebugApplicationThreads::Skip](../../winscript/reference/ienumremotedebugapplicationthreads-skip.md)|跳过枚举序列中指定数目的段。|  
|[IEnumRemoteDebugApplicationThreads::Reset](../../winscript/reference/ienumremotedebugapplicationthreads-reset.md)|将枚举序列重置到开始处。|  
|[IEnumRemoteDebugApplicationThreads::Clone](../../winscript/reference/ienumremotedebugapplicationthreads-clone.md)|创建与当前枚举器包含相同状态的一个枚举器。|
