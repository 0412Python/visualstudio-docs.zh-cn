# IEnumDebugExtendedPropertyInfo 接口
枚举 `ExtendedDebugPropertyInfo` 结构。  
  
## Vtable 顺序中的方法  
 下表显示了 `IEnumDebugExtendedPropertyInfo` 的方法。  
  
|方法|说明|  
|--------|--------|  
|[IEnumDebugExtendedPropertyInfo::Clone](../../winscript/reference/ienumdebugextendedpropertyinfo-clone.md)|创建与当前枚举器包含相同枚举状态的一个枚举器。|  
|[IEnumDebugExtendedPropertyInfo::GetCount](../../winscript/reference/ienumdebugextendedpropertyinfo-getcount.md)|获取 `ExtendedDebugPropertyInfo` 结构数在枚举数。|  
|[IEnumDebugExtendedPropertyInfo::Next](../../winscript/reference/ienumdebugextendedpropertyinfo-next.md)|检索 `ExtendedDebugPropertyInfo` 结构指定数目的枚举序列的。|  
|[IEnumDebugExtendedPropertyInfo::Skip](../../winscript/reference/ienumdebugextendedpropertyinfo-skip.md)|跳过 `ExtendedDebugPropertyInfo` 结构指定数目的枚举序列的。|  
|[IEnumDebugExtendedPropertyInfo::Reset](../../winscript/reference/ienumdebugextendedpropertyinfo-reset.md)|将枚举序列重置到开始处。|  
  
## 要求  
 标头:dbgprop.h
