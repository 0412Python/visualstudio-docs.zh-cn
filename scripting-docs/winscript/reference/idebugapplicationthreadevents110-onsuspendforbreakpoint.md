# IDebugApplicationThreadEvents110::OnSuspendForBreakPoint
确定线程是否为断点完全挂起的和非继续正常执行。  
  
> [!IMPORTANT]
>  [IDebugApplicationThreadEvents110 接口](../../winscript/reference/idebugapplicationthreadevents110-interface.md) 由PDM v11.0实现和更大。  找到在activdbg100.h。  
  
## 语法  
  
```cpp  
HRESULT OnSuspendForBreakPoint( void );  
```  
  
#### 参数  
 此方法没有参数。
