# <a name="idebugdocumentcontextgetdocument"></a>IDebugDocumentContext::GetDocument
返回包含此上下文的文档。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetDocument(  
   IDebugDocument**  ppsd  
);  
```  
  
#### <a name="parameters"></a>参数  
 `ppsd`  
 [out]包含此上下文的文档。  
  
## <a name="return-value"></a>返回值  
 该方法返回 `HRESULT`。 可能的值包括（但并不限于）下表中的项。  
  
|值|描述|  
|-----------|-----------------|  
|`S_OK`|方法成功。|  
  
## <a name="remarks"></a>备注  
 `GetDocument`方法返回包含此上下文的文档。
