# <a name="ibindeventhandlerbindhandler"></a>IBindEventHandler::BindHandler
将事件绑定到对象。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT BindHandler(  
   LPCOLESTR   pstrEvent,  
   IDispatch*  pdisp  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pstrEvent`  
 [in]指定要处理的事件。  
  
 `pdisp`  
 [in]指定要处理该事件的对象。  
  
## <a name="return-value"></a>返回值  
 该方法返回 `HRESULT`。 可能的值包括（但并不限于）下表中的项。  
  
|值|描述|  
|-----------|-----------------|  
|`S_OK`|方法成功。|  
  
## <a name="remarks"></a>备注  
 此方法将事件绑定到对象。
