# <a name="idebugapplicationnodeclose"></a>IDebugApplicationNode::Close
使此应用程序释放所有引用并进入非活动状态。  
  
## <a name="syntax"></a>语法  
  
```cpp
HRESULT Close();  
```  
  
#### <a name="parameters"></a>参数  
 此方法需要任何参数。  
  
## <a name="return-value"></a>返回值  
 该方法返回 `HRESULT`。 可能的值包括（但并不限于）下表中的项。  
  
|“值”|描述|  
|-----------|-----------------|  
|`S_OK`|方法成功。|  
  
## <a name="remarks"></a>备注  
 通常情况下，应用程序所有者应用程序退出时调用此方法。
