# IDebugCookie 接口
允许调试cookie设置，用于 `IMachineDebugManagerCookie` 接口的使用。  有关更多信息，请参见 [IMachineDebugManagerCookie 接口](../../winscript/reference/imachinedebugmanagercookie-interface.md)。  此接口由处理实现调试管理器\(PDM\)和使用由脚本调试器。  
  
## 方法  
 除了从 `IUnknown` 继承的方法之外，`IDebugCookie` 接口还公开下面的方法。  
  
|方法|说明|  
|--------|--------|  
|[IDebugCookie::SetDebugCookie](../../winscript/reference/idebugcookie-setdebugcookie.md)|设置调试应用程序cookie。|
