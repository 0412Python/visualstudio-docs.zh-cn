# <a name="ijsdebugframegetname-method"></a>IJsDebugFrame::GetName 方法
获取堆栈帧的用户友好名称。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetName(  
   BSTR *pName  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pName`  
 [out]堆栈帧的名称。  
  
## <a name="return-value"></a>返回值  
  
## <a name="requirements"></a>要求  
 **标头：** jscript9diag.h
