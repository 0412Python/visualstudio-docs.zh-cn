# IEnumDebugExpressionContexts 接口
枚举 `IDebugExpressionContexts` 对象的集合。  
  
 除了从 `IUnknown` 继承的方法之外，`IEnumDebugExpressionContexts` 接口还公开下面的方法。  
  
## Vtable 顺序中的方法  
  
|方法|说明|  
|--------|--------|  
|[IEnumDebugExpressionContexts::Next](../../winscript/reference/ienumdebugexpressioncontexts-next.md)|检索枚举序列中指定数目的段。|  
|[IEnumDebugExpressionContexts::Skip](../../winscript/reference/ienumdebugexpressioncontexts-skip.md)|跳过枚举序列中指定数目的段。|  
|[IEnumDebugExpressionContexts::Reset](../../winscript/reference/ienumdebugexpressioncontexts-reset.md)|将枚举序列重置到开始处。|  
|[IEnumDebugExpressionContexts::Clone](../../winscript/reference/ienumdebugexpressioncontexts-clone.md)|创建与当前枚举器包含相同状态的一个枚举器。|
