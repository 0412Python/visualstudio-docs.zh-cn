# <a name="how-to-report-a-problem-in-visual-studio-for-mac"></a>如何报告 Visual Studio for Mac 的问题

如果无法使用 Visual Studio for Mac，请告诉我们你遇到的问题。 下面介绍如何将问题报告给[开发人员社区](https://developercommunity.visualstudio.com/)，以便我们诊断并修复该问题。

## <a name="how-to-report-a-problem"></a>如何报告问题

若要报告 Visual Studio for Mac 的问题，请从菜单栏中选择“帮助”>“报告问题”：

![“报告问题”链接](media/report-problem-image1.png)

也可以使用欢迎页上的“报告问题”链接。

下图展示了“报告问题”网页。 可以使用该页面来查看所有现有问题，对类似问题投票，以及为其他用户报告的问题提交解决方案：

![“报告问题”网页](media/report-problem-image2.png)

选择网页上的“报告问题”按钮。

如果尚未登录“开发者社区”应用程序，请按照以下页面的提示登录。 选择“接受”按钮来授权“开发者社区”应用程序使用你的 Microsoft 帐户：

![“报告问题”网页](media/report-problem-image3.png)

为问题添加描述性标题和其他与之相关的任何可能有帮助的信息。 请尽可能提供重现步骤，以便我们尽快了解并修复你的问题。

![“报告问题”网页](media/report-problem-image4.png)

## <a name="provide-a-suggestion"></a>提供建议

请使用欢迎页面上的“提供建议”链接定向到 [Visual Studio for Mac 的建议功能页面](https://developercommunity.visualstudio.com/content/idea/post.html?space=41)。 可在此处添加新请求或对现有请求投票。
