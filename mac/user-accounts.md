# <a name="user-accounts"></a>用户帐户

下载并安装 Visual Studio for Mac 后，必须登录以激活许可证，并获取对 Xamarin Studio Community、Xamarin Studio Professional 或 Xamarin Studio Enterprise 的访问权限。

登录后，可以访问与用户帐户以及链接到该帐户的所有帐户关联的所有服务。 无论你在何处使用 Visual Studio，你的帐户设置都随你一起漫游。 你可以为单个用户配置对特定 Visual Studio 实例的权限级别。
