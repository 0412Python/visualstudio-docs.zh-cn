# <a name="block-selection"></a>块选择
通过 Visual Studio for Mac，可以轻松地选择、剪切、复制和粘贴选定部分的代码。 可以使用代码块上的“选项 + 拖动”命令突出显示和选择矩形代码块。 粘贴块时会保留此块选择部分的格式。

也可以编辑块选择部分，同时修改多行文本。 可以使用零宽度的块选择将脱字号同时放置到多行的行首，从而可以在多行中键入相同文本：
