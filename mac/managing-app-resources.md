# <a name="managing-app-resources"></a>管理应用资源

应用资源文件（如图像、文本文件和音频文件）是应用程序的整型元素且不使用应用程序编译。 每个 Visual Studio for Mac 支持的平台处理这些文件类型的方法都不同。 有关处理 Visual Studio for Mac 中的应用资源的详细信息，请查看以下指南。

## <a name="xamarinforms"></a>Xamarin.Forms

Xamarin.Forms 代码在多个平台上运行 - 每个平台都有自己的文件系统。 这意味着可通过在每个平台上使用本机文件 API 或通过添加文件作为嵌入资源来读取和写入文件。

* [使用图像](https://developer.xamarin.com/guides/xamarin-forms/user-interface/images/)
* [使用文件]( https://developer.xamarin.com/guides/xamarin-forms/application-fundamentals/files/)
