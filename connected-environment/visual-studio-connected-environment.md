# <a name="vs-connected-environment-for-azure-container-service-aks-preview"></a>适用于 Azure 容器服务的 VS 通连环境 (AKS)（预览版）
VS 通连环境有助于在 Kubernetes 上快速开发。 你将创建基于 Kubernetes 的环境，该环境完全托管在 Azure 中并针对开发进行了优化，然后使用 VS Code、Visual Studio 或命令行等熟悉的工具在云中迭代开发容器。

这种方法具有以下几个好处：

* 获得具有生产代表性和对云资源的完全访问权限的无基础结构开发环境。
* 使用 VS Code 或 Visual Studio 直接在 Kubernetes 中调试 Node.js 和 .NET Core 容器。 可使用命令行界面开发所有其他语言。
* 在开发团队中共享 Kubernetes 实例以节省成本，并最大程度减少新团队成员的本地计算机设置。
* 独立地开发代码并对其他组件进行端到端测试，无需复制或模拟依赖项。
