## <a name="ensure-the-development-environment-was-successfully-created"></a>确保已成功创建开发环境
在上一节中，我们使用 `vsce env create` 命令在 Azure 中创建了开发环境。 完成此过程大约需要 7 分钟，因此请再次检查是否已成功创建。 可随时使用以下命令检查有权访问的环境及其状态：`vsce env list`。
